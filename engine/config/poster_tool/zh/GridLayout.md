## GRID COLLAGE / 网格拼贴风格

### 适用场景
校园招聘、设计工作坊、创意大赛、品牌发布会、艺术展览、设计师招募、创意学院开放日、设计软件推广

### 配色方案
- **主色调:** 亮蓝色、纯白色、纯黑色
- **背景:** 白色或浅灰色网格图案
- **辅助色:** 灰色层次（浅灰到深灰）、彩色点缀
- **饱和度:** 蓝色高饱和度，灰色系低饱和度
- **色彩行为:** 干净明亮的基底上，蓝色和黑色创造视觉层次

#### 常用配色组合
- 设计软件: 白色网格背景 + 亮蓝色 + 黑色文字块
- 校招清新: 浅蓝色背景 + 白色 + 深灰色元素
- 创意拼贴: 白色基底 + 多层次灰色 + 蓝色强调

### 视觉元素
- 方格/网格背景图案
- 拼贴风格的矩形文字块
- 选框/裁剪框UI元素（带角标记）
- 像素化渐变色块（灰度过渡）
- 蓝色标签和按钮元素
- 表情图标或吉祥物形象
- 层叠的矩形和正方形
- 设计软件界面元素暗示

### 效果处理
- 干净利落的硬边图形
- 无立体效果的平面设计感
- 网格的精确对齐
- 色块的纯色填充
- 选框的虚线或实线边框
- 层叠元素的遮挡关系
- 像素渐变的阶梯过渡
- 整体的界面设计感

### 构图策略
- 网格作为背景秩序基础
- 文字块以拼贴方式错落排布
- 选框元素创造"正在编辑"的感觉
- 像素渐变块在角落或底部
- 吉祥物或图标元素点缀
- 蓝色元素作为视觉引导
- 信息分层清晰（主标题、副标题、细节）

### 文字设置
- **颜色选择:** 高对比度的纯色组合
  - 推荐主色: 纯黑色 (#000000)、纯白色 (#FFFFFF)
  - 辅助色: 亮蓝色 (#0099FF)、中灰色 (#666666)
- **填充方式:** 纯色填充，单一颜色，无渐变
- **禁止效果:** 
  - 无渐变填充
  - 无阴影/投影
  - 无外发光/内发光
  - 无描边
  - 无模糊效果
  - 无任何文字特效
- **字体风格:** 几何无衬线体或黑体，字重多样 (Light到Bold混合使用)
- **整体原则:** 文字保持干净、锐利、高可读性，通过字体选择和高对比度纯色与网格背景形成清晰的视觉层次

### 情绪氛围
创意活力、秩序美感、设计思维。整体能量年轻专业，传达对设计工具和创作过程的热爱，唤起设计软件使用时的创造快感，带有包豪斯设计和现代UI设计的双重影响。
