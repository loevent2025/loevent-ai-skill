# AI Demo Day – Education Event Knowledge Base

## 📌 Module Activation Criteria

Apply the rules in this module when the event theme involves the following keywords:
- AI bootcamp graduation showcase, Bootcamp Demo Day
- Student AI project showcase, university AI project presentations
- AI skill certification showcase, student portfolio showcase day

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Student project showcase + mentor critique + certificate ceremony + group photo |
| Duration | 3–4 hours |
| Participants | Students, mentors, training institutions, enterprise HR |
| Core Objective | Demonstrate learning outcomes, skill certification, recruitment networking, community belonging |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Interests |
|----------|---------------|---------------|
| P0 | Presenting Students | Showcasing their work, skill certification, career opportunities |
| P1 | Enterprise HR / Recruiters | Talent discovery, skill assessment, direct hiring |
| P2 | Mentors / Training Institutions | Teaching outcomes, brand visibility, student development |
| P3 | Prospective Students | Understanding course effectiveness, learning outcome references |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Bootcamp Graduation Showcase | Energetic & Engaging (B) | Celebratory atmosphere, recognizing growth |
| University Project Presentation | Formal & Professional (A) | Academic value, expert evaluation |
| Recruitment-Focused Showcase | Benefits-oriented (D) | Hiring opportunities, skill validation |
| Skill Certification Showcase | Formal & Professional (A) | Skill verification, authoritative certification |
