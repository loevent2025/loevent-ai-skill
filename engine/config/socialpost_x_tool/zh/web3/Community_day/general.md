# 咖啡馆共建 / Cafe Coworking × 通用类

## 适用场景
行业咖啡馆共建日、Builder Cafe、Code & Coffee、创业者工作日、产品用户聚会、远程工作者共建、自由职业者 Meetup

## 场景特征
- **形式**：咖啡馆包场或固定区域、带电脑来工作、自由交流、可能有简短分享
- **时长**：3-8 小时，通常分时段（上午/下午/全天 drop in）
- **核心价值**：专注工作的氛围、自然发生的交流、同类人陪伴感

## 受众定位
- **Primary Audience:** 需要工作环境的从业者——想找个地方专注工作又能认识人
- **Secondary Audience:** 自由职业者/远程工作者——平时一个人工作太闷
- **Tertiary Audience:** 好奇者——想来坐坐、认识人、看看氛围
- **Audience Behavior:** 带电脑工作、偶尔交流、自然 networking、可能寻求帮助

## 内容元素
- 场地说明（咖啡馆氛围、环境）
- 时段选择（上午/下午/全天/随时 drop in）
- 提供什么（咖啡、WiFi、插座、小食）
- 会发生什么（工作、交流、可能的简短分享）
- Intro Circle（可选：开场自我介绍环节）
- 参与方式（带电脑来就行）

## 语言处理
- ✅ 轻松、低压力、inclusive
- ✅ "带电脑来就行"、"想工作就工作，想聊天就聊天"
- ✅ 强调氛围和陪伴感
- ✅ All skill levels welcome
- ❌ 不要太正式
- ❌ 避免"干货"、"学习机会"

## 结构策略
- 场景开场块占比 20%：咖啡馆氛围
- 时段/形式块占比 20%：怎么参与
- 会发生什么块占比 25%：工作、交流、可能的分享
- 提供什么块占比 15%：咖啡、环境
- 人群说明块占比 10%
- 行动号召块占比 10%
- Emoji 密度控制：总量 8-12 个，轻松简洁

## 情绪氛围
轻松专注、陪伴感、自然交流。"找个地方工作，顺便认识人"的舒适感。

## 风格变体

### 变体A：开放共建型 / Open Cowork
- **适用：** 面向所有人的咖啡馆共建
- **开场格式：** ☕ Cafe Cowork | 带电脑来，一起工作几小时
- **语气：** 开放、轻松、inclusive

### 变体B：产品/工具用户型 / Product Cafe
- **适用：** 特定产品用户的共建（如 Cursor、Figma、Notion）
- **开场格式：** ☕💻 Cafe [产品名] | [产品] 用户一起 build
- **语气：** 产品社区、用户连接、可能有团队在场

### 变体C：定期共建型 / Regular Cowork
- **适用：** 固定频率的咖啡馆共建（每周/每两周）
- **开场格式：** ☕ [城市] Code & Coffee | 每两周一次的咖啡馆工作日
- **语气：** 社群感、持续、归属

## 时段设计模板

### 全天多时段型
```
⏰ 选个适合你的时段：
- 9:00-12:00 上午场（早起党）
- 12:00-15:00 下午场 A
- 15:00-18:00 下午场 B
- 或者：随时 drop in 坐一会儿
```

### 半天型
```
⏰ 14:00-18:00
随时来随时走，待多久都行
```

### Intro Circle 模板（可选）
```
🗣️ 开场 Intro Circle（14:30）
简单说一下：
- 你是谁
- 今天在做什么
- 需要什么帮助 / 能帮别人什么
之后自由活动，想工作就工作，想聊天就聊天
```

## 注意事项

### 咖啡馆共建要点

1. **时段设计要灵活**
   - 提供多个时段选择
   - 允许 drop in（随时来走）
   - 说清楚每个时段的容量

2. **场地信息要清楚**
   - WiFi 稳定性
   - 插座数量
   - 是否有安静区域
   - 咖啡/饮品安排

3. **Intro Circle 是可选的**
   - 如果有，说清楚时间和形式
   - 强调可以 pass
   - 之后 self-organized

4. **强调低压力**
   - 想工作就工作，想聊天就聊天
   - 不是会议，不是课程
   - 带电脑来就行

5. **产品用户型特别注意**
   - 团队在场是卖点
   - 周边/credits 是福利
   - Workshop 是可选的，不是主体
