# Web3 Hackathon - Investment Event Knowledge Base

## 📌 Module Activation Conditions

Apply this module when the event theme involves the following keywords:
- Startup hackathons, startup challenge competitions, founder marathons
- Investor-judged competitions, pitch hackathons, fundraising matching competitions
- Accelerator selection rounds, cohort admission competitions

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Project development + pitch deck refinement + investor judging + demo presentations |
| Duration | 48-72 hours intensive, or 2-4 week competition cycle |
| Participants | Founding teams, investment firms, accelerators, FAs |
| Core Objective | Project screening, fundraising matching, accelerator admission, investment discovery |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Early-Stage Founders | Fundraising opportunities, accelerator resources, investor exposure |
| P1 | Technical Founders | MVP validation, team building, path to commercialization |
| P2 | Investment Firms | Early-stage deal flow, talent discovery, ecosystem strategy |
| P3 | Serial Entrepreneurs | New direction validation, resource integration, rapid launch |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Emphasis |
|------------|-------------------|---------------|
| Accelerator Selection Round | Formal & Professional (A) | Accelerator resources, professional judging |
| Founder Marathon | Energetic & Engaging (B) | Startup energy, team building |
| Fundraising Challenge | Benefits-Driven (D) | Investor matching, fundraising opportunities |
| Business Model Competition | Suspense-Driven (C) | Business validation, model innovation |

## ❓ FAQ Templates

| Question | Answer |
|----------|--------|
| Do I need a complete team? | You can form a team on-site — we'll facilitate a team-matching session |
| Can I enter with an existing project? | Yes, but you must demonstrate new progress during the competition |
| Is the investment guaranteed? | Investment interest must be independently negotiated and confirmed post-event |
| What should I prepare? | We recommend having a project idea and a brief deck ready in advance |

## ⚠️ Prohibited Actions

- ❌ Promising specific investment amounts or terms
- ❌ Exaggerating accelerator acceptance rates
- ❌ Implying token issuance plans
- ❌ Falsely advertising investment firm participation
