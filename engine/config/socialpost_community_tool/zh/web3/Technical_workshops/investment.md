# Web3 技术工作坊 - 投资类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- 投资分析工具实操、链上数据分析培训
- 估值模型工作坊、尽调方法论培训
- DeFi 协议分析、Token 经济学研究

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 方法论讲解 + 工具实操 + 案例分析 + 实战练习 |
| 时长 | 3-6 小时，理论结合实操 |
| 参与方 | 投资分析师、研究员、投资人、项目评估者 |
| 核心目标 | 分析能力提升、工具掌握、方法论建立 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 投资分析师 | 分析框架、工具效率、数据解读 |
| P1 | 研究员 | 研究方法、数据源、行业洞察 |
| P2 | 投资人 | 尽调能力、风险识别、决策支持 |
| P3 | 项目方 | 了解投资视角、优化呈现 |

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 专业分析培训 | 正式专业型 (A) | 方法论、专业深度 |
