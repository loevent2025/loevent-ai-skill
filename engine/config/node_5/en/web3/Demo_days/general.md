# Web3 Industry — Demo Day Partner Reference Guide

> **Applicable to:** Project pitch events, investor matchmaking sessions, accelerator Demo Days, ecosystem showcase days, and similar formats

---

## 1. Event Profile

| Attribute | Description |
|-----------|-------------|
| Scale | 50–200 attendees (primarily invite-only) |
| Duration | Half-day to full day |
| Format | Project pitches + investor Q&A + 1-on-1 meetings + networking reception |
| Core Value | Fundraising connections, project exposure, investor networking, industry insights |
| Budget Level | Mid-range; emphasis on venue, speakers, and targeted invitations |

## 6. Demo Day Format Recommendations

### 6.1 Pitch Session Structure

| Segment | Recommended Duration | Notes |
|---------|----------------------|-------|
| Project Pitch | 5–8 min / project | Standardized pitch template |
| Investor Commentary | 3–5 min / project | 2–3 judges rotating feedback |
| Q&A | 3–5 min / project | Audience questions |
| Transition | 2 min | Setup time between teams |

### 6.2 Recommended Project Volume

| Event Duration | Suggested Number of Projects | Notes |
|----------------|-------------------------------|-------|
| Half-day (3–4 hrs) | 8–12 | Compact schedule with breaks |
| Full day (6–8 hrs) | 15–25 | Includes lunch, breaks, and networking |

### 6.3 Supporting Sessions

- **Investor Insights Keynotes:** 1–2 thematic presentations (15–20 min each)
- **1-on-1 Meetings:** Appointment-based, 10–15 min per session
- **Networking Breaks / Lunch:** Informal engagement time
- **After Party:** Evening session for deeper networking

## 10. Execution Guidelines

### 10.1 Priority Sequence

1. **Co-organizers** — Confirm during event planning phase (2–3 months prior)
2. **Investor Invitations** — Launch outreach 1.5–2 months prior
3. **Project Selection** — Complete screening 1–1.5 months prior
4. **Strategic Partners** — Confirm 1 month prior
5. **Service Partners** — Finalize 2–3 weeks prior

### 10.2 Key Considerations

- Align project screening criteria closely with co-organizers
- Curate investor invitations carefully to avoid audience overlap
- Sequence pitches to ensure sector diversity across the program
- Collect mutual interest signals from both sides before scheduling 1-on-1 meetings
- Establish confidentiality protocols for project materials in advance
- Set up an investment intent tracking mechanism before the event
- Confirm media coverage scope with all participating project teams
- Treat the After Party as a high-value networking opportunity and plan it accordingly
