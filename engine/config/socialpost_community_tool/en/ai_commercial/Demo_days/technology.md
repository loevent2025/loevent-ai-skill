# AI Demo Day — Technical Events Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- AI Hackathon Demo Day, Agent Development Competition Showcase
- LLM Application Showcase Day, AI Technical Project Pitch
- Grant Project Review, AI Technical Incubation Showcase

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Project pitch + Technical judging + Live demo + Q&A interaction |
| Duration | 3–6 hours |
| Participants | Competing / incubated project teams, technical judges, developer audience |
| Core Objective | Technical outcome showcase, expert review and feedback, project visibility, follow-on incubation |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Presenting Project Teams | Technical validation, expert feedback, visibility opportunities |
| P1 | Technical Judges | Technical innovation, completeness, feasibility |
| P2 | Developer Audience | Technical inspiration, project ideas, collaboration opportunities |
| P3 | Model Vendors / Ecosystem Partners | Ecosystem project discovery, technical talent |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Hackathon Demo Day | Formal & Professional (A) | Technical judging, outcome review |
| Technical Incubation Showcase | Formal & Professional (A) | Milestone review, professional evaluation |
| Open Technical Showcase | Energetic & Engaging (B) | Community showcase, interactive experience |
| Grant Project Review | Formal & Professional (A) | Milestone acceptance, official review |
