# Web3 Large-Scale Expo / Conference – Education Event Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- Web3 education conference, developer training week
- Industry certification, talent development, skill training
- University partnerships, career development, employer matchmaking

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Main forum + Track-specific courses + Hands-on workshops + Live project exercises |
| Content Depth | From beginner to advanced — building a structured knowledge framework |
| Participants | Education institutions, project team mentors, universities, job seekers |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Concerns |
|----------|---------------|--------------|
| P0 | Web3 Learners | Learning path, knowledge framework, entry barrier |
| P1 | Career-Switching Developers | Technical differences, transition roadmap, hands-on projects |
| P2 | University Faculty & Students | Academic research, internship opportunities, course resources |
| P3 | Corporate Training Leaders | Team training, talent recruitment, skill certification |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Comprehensive Education Conference | Energetic & Engaging (B) | Learning atmosphere, growth path |
| Technical Training Week | Suspense-Driven (C) | Learning pain points, solutions |
| Career Development / Job Fair | Benefits-Oriented (D) | Job opportunities, career pathways |
| University Partnership / Academic Event | Formal & Professional (A) | Academic value, industry-academia alignment |

## ⚠️ Prohibited Actions

- ❌ Promising 100% employment or citing fabricated employment rates
- ❌ Exaggerating course outcomes or learning results
- ❌ Using misleading language such as "guaranteed employment" or "high-salary assurance"
- ❌ Fabricating mentor backgrounds or credentials
