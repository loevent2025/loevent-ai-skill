# Web3 黑客松 - 投资类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- 创业黑客松、Startup 挑战赛、创始人马拉松
- 投资人评审赛、项目路演黑客松、融资对接赛
- 孵化器选拔赛、加速营入选赛

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 项目开发 + BP 打磨 + 投资人评审 + 路演展示 |
| 时长 | 48-72 小时集中，或 2-4 周赛程 |
| 参与方 | 创业团队、投资机构、孵化器、FA |
| 核心目标 | 项目筛选、融资对接、孵化录取、投资发现 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 早期创业者 | 融资机会、孵化资源、投资人曝光 |
| P1 | 技术创始人 | MVP 验证、团队组建、商业化路径 |
| P2 | 投资机构 | 早期项目源、人才发现、生态布局 |
| P3 | 连续创业者 | 新方向验证、资源整合、快速启动 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 孵化器选拔赛 | 正式专业型 (A) | 孵化资源、专业评审 |
| 创业者马拉松 | 热情活力型 (B) | 创业氛围、团队组建 |
| 融资挑战赛 | 福利导向型 (D) | 投资对接、融资机会 |
| 商业模式赛 | 悬念引导型 (C) | 商业验证、模式创新 |

---

## 📝 内容必含元素

### 1. 融资价值（至少2处）
- "[数量]+ 家投资机构评审/观摩"
- "优胜项目获得 [金额] 种子轮投资机会"

### 2. 孵化资源（至少1处）
- "直通 [孵化器] 加速计划"
- "获得 [时长] 免费办公空间"

### 3. 导师支持（至少1处）
- "[数量]+ 位创业导师一对一辅导"
- "投资人 Office Hour 答疑"

### 4. 合规提示（必须包含）
- "⚠️ 融资机会需双方独立评估，不构成投资承诺"

## ❓ 常见问答模板

| 问题 | 回答 |
|------|------|
| 必须有完整团队吗？ | 可以现场组队，我们会安排组队环节 |
| 已有项目可以参赛吗？ | 可以，但需在赛事期间有新进展 |
| 投资是确定的吗？ | 投资意向需赛后双方独立洽谈确认 |
| 需要准备什么？ | 建议提前准备项目 Idea 和简易 BP |

## ⚠️ 禁止事项

- ❌ 承诺确定的投资金额或条款
- ❌ 夸大孵化器录取率
- ❌ 暗示 Token 发行计划
- ❌ 虚假宣传投资机构参与
