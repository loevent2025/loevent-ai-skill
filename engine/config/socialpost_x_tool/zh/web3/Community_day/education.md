# 咖啡馆共建 / Cafe Coworking × 教育类

## 适用场景
学习者咖啡馆共建、新手 Code & Coffee、Bootcamp 学员工作日、学习小组咖啡馆聚会、导师 Office Hour 咖啡馆版、转型者共建日

## 场景特征
- **形式**：学习者一起在咖啡馆学习/做项目、互相帮助、可能有导师在场
- **时长**：3-5 小时
- **核心价值**：学习陪伴、互助答疑、降低孤独感、找到学习伙伴

## 受众定位
- **Primary Audience:** 学习者——正在学习编程/Web3/新技能的人
- **Secondary Audience:** 新手开发者——刚入门想找人一起学习
- **Tertiary Audience:** 导师/前辈——愿意来帮忙答疑的有经验者
- **Audience Behavior:** 学习、做项目、问问题、找学习伙伴

## 内容元素
- 场地氛围
- 学习主题/方向（可选）
- 导师/助教在场说明（如果有）
- 互助机制
- "问问题很正常"的强调
- All skill levels welcome

## 语言处理
- ✅ 温暖、鼓励、包容
- ✅ "卡住了可以问"、"新手友好"
- ✅ 强调互助和陪伴
- ✅ All skill levels welcome
- ❌ 不制造学习焦虑
- ❌ 避免"高强度"、"硬核"

## 结构策略
- 场景开场块占比 20%
- 学习主题块占比 20%（如果有特定方向）
- 互助机制块占比 25%
- 导师/资源块占比 15%
- 参与说明块占比 10%
- 行动号召块占比 10%
- Emoji 密度控制：总量 10-14 个，温暖友好

## 情绪氛围
温暖陪伴、互助学习、新手友好。"学习路上不孤单"的归属感。

## 风格变体

### 变体A：新手共建型 / Beginner Friendly
- **适用：** 新手为主的学习共建
- **开场格式：** 🌱☕ 新手咖啡馆 | 一起学习，卡住了互相帮
- **语气：** 友好、鼓励、无压力

### 变体B：学习小组型 / Study Group
- **适用：** 特定学习方向的共建
- **开场格式：** 📚☕ [学习主题] 学习咖啡馆 | 一起啃这个
- **语气：** 专注、互助、同路人

### 变体C：导师 Office Hour 型 / Mentor Hours
- **适用：** 有导师在场的学习共建
- **开场格式：** 🧑‍🏫☕ 咖啡馆 Office Hour | 导师在，随时问
- **语气：** 支持、答疑、成长

## 互助机制模板

### Intro Circle 学习版
```
🗣️ 开场说一下（2:30pm）：
- 你在学什么
- 卡在哪里 / 今天想搞定什么
- 能帮别人什么
（说完可能就有人能帮你了）
```

### 互助配对
```
🤝 互助机制：
- 贴个便签说你卡在哪里
- 看到能帮的就过去聊聊
- 导师会巡场，有问题举手
```

### 学习打卡
```
📝 今日目标（可选）：
到场时写下今天想完成什么
走之前看看完成了多少
（没完成也没关系，下次继续）
```

## 注意事项

### 教育类咖啡馆共建要点

1. **新手友好要反复强调**
   - All skill levels welcome
   - 问"傻"问题没关系
   - 来的人都在学习

2. **互助机制要设计**
   - 便签写问题
   - 鼓励互相帮助
   - 导师/前辈巡场

3. **降低门槛**
   - 不要求完成什么
   - 可以随时来走
   - 没学完也没关系

4. **学习方向可以聚焦**
   - 特定技术栈的学习共建效果更好
   - 大家遇到的问题类似，更容易互助

5. **导师在场是加分项**
   - 说清楚导师擅长什么
   - 怎么找导师问问题
   - 导师不是来讲课的，是来帮忙的
