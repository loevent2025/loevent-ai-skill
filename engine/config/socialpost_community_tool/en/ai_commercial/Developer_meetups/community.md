# AI Developer Meetup – Community Events Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- AI Builder Meetup, Community Hangout, Builder Night
- AI Developer After Party, Monthly Community Meetup
- AI Enthusiast Gathering, Builder Social

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Lightning talks + Open socializing + Social games + Party atmosphere |
| Duration | 2–4 hours |
| Participants | Community Builders, AI enthusiasts, community newcomers |
| Core Objective | Community bonding, Builder socializing, casual conversation, sense of belonging |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Community Builders | Friends' get-together, casual conversations, community belonging |
| P1 | AI enthusiasts | Making friends, community integration, staying in the loop |
| P2 | Community newcomers | Icebreaking, meeting Builders, finding belonging |
| P3 | Potential collaborators | Finding partners, project discussions, making connections |

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
