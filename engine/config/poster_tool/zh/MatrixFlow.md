# MATRIX / 矩阵风格（黑客帝国）

## 适用场景
会议主视觉、黑客马拉松、NFT 发售、加密艺术节、网络安全活动、区块链峰会、黑客文化聚会、科技颠覆大会

## 配色方案
- **主色调:** 亮霓虹绿、剧毒绿、酸橙绿
- **背景:** 纯黑或极深灰色，营造最大对比度
- **辅助色:** 深绿、祖母绿、白色高光
- **饱和度:** 霓虹绿高饱和度，搭配深阴影
- **色彩行为:** 黑色背景上的发光绿色，高对比度，磷光屏效果

### 常用配色组合
- 经典矩阵: 纯黑背景上亮霓虹绿的坠落代码
- 深度系统: 深绿色 + 剧毒绿渐变，以黑色为基底
- 终端黑客: 酸橙绿 + 白色高光，在深黑色上发光

## 视觉元素
- 瀑布式字符和代码雨（片假名、数字、符号）
- 终端窗口和命令行界面
- 二进制代码流（0和1）
- 系统状态指示器和加载条
- 线框网格和数字建筑
- 发光光标元素
- 绿色电路板图案
- 滚动文本和数据流
- 磷光屏扫描线
- 数字读数和系统监控器

## 效果处理
- 绿色元素的磷光发光效果
- 坠落字符的动态模糊
- 代码流的拖尾渐隐
- 屏幕扫描线叠加
- 亮元素柔和的外发光
- 背景层深度的 Gaussian 模糊
- 亮度变化在代码雨中创造深度
- 最亮字符的光晕效果
- CRT 显示器纹理的细微颗粒
- 垂直运动轨迹

## 构图策略
- 垂直瀑布式列填充画面
- 具有前景和背景代码流的分层深度
- 集中亮度区域吸引视线焦点
- 战略性定位终端窗口以形成层次
- 代码密度的不对称分布
- 用于文本放置区域的深色负空间
- 创造运动感的有节奏的垂直流动
- 暗示数字空间的透视网格
- 焦点区域元素较亮，边缘较暗
- varying code intensity 的全幅覆盖

## 文字设置
- **颜色选择:** 选择与纯黑背景形成高对比度的纯色
  - 推荐主色: 亮霓虹绿 (#00FF41) 或剧毒绿 (#39FF14)
  - 备选色: 纯白色 (#FFFFFF)、酸橙绿 (#BFFF00)
- **填充方式:** 纯色填充，单一颜色，无渐变
- **禁止效果:** 
  - 无磷光发光效果
  - 无外发光/内发光
  - 无光晕效果
  - 无描边
  - 无阴影/投影
  - 无扫描线叠加
  - 无渐变填充
  - 无任何文字特效
- **整体原则:** 文字保持干净、锐利、高可读性，通过字体选择和高对比度建立矩阵感，而非依赖发光特效

## 情绪氛围
黑客神秘感、数字觉醒、系统渗透美学。整体能量神秘、技术化、叛逆——唤起地下黑客文化和闯入数字系统的刺激感，将代码视为工具和艺术形式，并带有赛博朋克哲学底蕴。
