#梦幻月夜风格 / DREAMY MOONLIT NIGHT

## 适用场景
中秋节活动推广、传统节日派对、夜间音乐节、月光主题活动、东方美学展览、浪漫主题派对、夜店特别活动、品牌节日营销、文化融合活动、星空主题展览、冥想/疗愈活动、婚礼/求婚活动

## 配色方案
- **主色调：** 深邃夜空色系 - 午夜蓝 (#1A2744)、星空靛 (#2C3E6E)、月光白 (#F5F5F5)、霓虹紫粉 (#D896FF)
- **背景色：** 夜空渐变，从顶部深邃午夜蓝 (#0F1729) 过渡到中部星空蓝 (#1E3A5F) 再到底部雾蓝 (#4A6FA5)
- **月亮色：** 中心纯白 (#FFFFFF) 向边缘过渡到月光米白 (#FFF8E7)，周围有柔和的光晕
- **云朵色：** 白色 (#FFFFFF) 到浅灰蓝 (#C8D4E3) 的体积渐变，受月光照亮
- **点缀色：** 霓虹粉紫 (#D896FF)、电光紫 (#7B4FBF) 用于强调文字
- **饱和度：** 背景低饱和 (20-40%)，营造静谧氛围；强调元素高饱和 (70-90%)
- **色彩行为：** 整体冷色调为主，月亮暖白色作为视觉焦点形成对比，霓虹色文字增加现代活力

### 常用配色组合
- 静谧月夜：午夜蓝 (#0F1729) → 星空蓝 (#1E3A5F) → 雾蓝 (#4A6FA5) + 月光白 (#FFFFFF)
- 霓虹中秋：深紫蓝 (#1A1A3A) → 靛蓝 (#2E4A7D) + 霓虹粉 (#D896FF) + 电光紫 (#7B4FBF)
- 梦幻星海：深空蓝 (#0A1628) → 钴蓝 (#1E4D8C) → 浅蓝 (#6B9AC4) + 星芒白 (#FFFAFA)

## 视觉元素

### 核心：巨型发光月亮
- **满月作为视觉中心**——大尺寸圆形月亮，直径占画面宽度40-60%
- **月亮发光效果**——中心明亮纯白，边缘柔和渐变，周围有大范围光晕弥散 (50-100px)
- **月亮位置**——通常位于画面中央或中上部，作为绝对视觉焦点
- **月亮质感**——可以是纯净的发光圆盘，或带有轻微的月球表面纹理

### 夜空背景
- **渐变夜空**——从深到浅的蓝色渐变，营造空间深度
- **星星点缀**——散布的小星星/十字光芒，大小不一，增加夜空细节
- **星芒效果**——部分星星有4角或6角光芒，颜色为白色或淡蓝
- **可选：淡淡的星云/银河纹理**——增加梦幻感

### 云层元素
- **体积感云朵**——位于画面底部或中下部，有明显的光影层次
- **云朵受月光照亮**——面向月亮的一侧较亮，背面较暗
- **云层层叠**——前后多层云朵增加空间深度
- **云朵边缘柔和**——自然的羽化效果，与夜空融合

### 装饰细节
- 可选的东方元素剪影（如远山、建筑轮廓）
- 光点/萤火虫效果点缀
- 简洁的几何线条或框架元素

## 效果处理
- 月亮使用多层光晕叠加：内层强光 + 中层柔光 + 外层弥散光晕
- 背景渐变平滑自然，可叠加轻微噪点 (3-5%) 增加质感
- 云朵使用柔和的明暗渐变，边缘羽化 (20-50px)
- 星星使用"滤色"混合模式，部分添加十字光芒
- 整体画面可叠加轻微的蓝色色调统一
- 避免过度锐利的边缘，保持梦幻柔和感
- 月亮周围可添加轻微的镜头光晕效果

## 构图策略
- **中心焦点构图**——月亮位于画面中心或黄金分割点
- **垂直层次分明**——天空（上）→ 月亮（中）→ 云层（下）
- **对称或近似对称**——营造平衡稳定的视觉感受
- **文字区域规划**——月亮上方/内部放置主标题，下方放置副标题
- **底部信息区**——活动详情位于云层下方或画面最底部
- **留出呼吸空间**——月亮周围不要有过多元素干扰
- **四角可放置辅助信息**——品牌Logo、日期等

## 文字设置
- **颜色选择：**
  - 深色夜空区域使用白色 (#FFFFFF) 或浅色文字
  - 月亮区域内可使用深色文字形成对比：深紫 (#3D2B56)、深蓝 (#1A2744)
  - 强调词使用霓虹粉紫 (#D896FF)、电光紫 (#9B6DFF)
- **填充方式：** 纯色填充，单一颜色，无渐变
- **禁止效果：**
  - 无渐变填充
  - 无阴影/投影
  - 无外发光/内发光
  - 无描边
  - 无模糊效果
  - 无任何文字特效
- **字体风格：**
  - 主标题：可使用3D透视立体效果的粗体无衬线，深紫色，带有倾斜角度
  - 强调词：霓虹风格手写/草书体，粉紫色，增加浪漫感
  - 信息文字：简洁现代的无衬线体，白色，字重Light到Regular
- **整体原则：** 文字层次分明，主次清晰，与梦幻背景形成清晰对比，中英文混排时注意字体协调

## 情绪氛围
梦幻、浪漫、静谧、神秘、团圆。整体传递一种"月光下的浪漫"气质，巨大的满月营造震撼而温柔的视觉焦点，夜空与云层增加空间深度和梦幻感。适合追求浪漫美学、东方意境与现代潮流结合的品牌活动。唤起对月夜的诗意想象、对团圆美好的向往、以及对梦幻时刻的沉醉。
