---

## 场景特征

| 特征项 | 说明 |
|-------|------|
| 规模 | 50-300人 |
| 时长 | 半天至2天 |
| 形式 | 双边对话+跨界圆桌+联合工作坊+社交配对 |
| 核心价值 | 生态破圈、人才引入、创新催化、合作撮合 |

### 嘉宾配对原则

| 原则 | 说明 |
|------|------|
| 层级对等 | 双边嘉宾职级、影响力相当 |
| 议题互补 | 一方提出问题，一方提供方案视角 |
| 开放心态 | 筛选愿意对话、非封闭心态的嘉宾 |

## 权益设计参考

### 双边联合主办权益

**品牌权益：**
- 活动品牌并列（双边Logo对等）
- 所有物料双边露出
- 官方联合发布

**内容权益：**
- 议程设计参与
- 本方Keynote/圆桌时段
- 嘉宾邀请主导权

**数据权益：**
- 双边参与者信息
- 跨界合作意向数据
- 后续触达权限

**对接权益：**
- 优先配对对方资源
- 定向商务撮合
- 后续合作优先权

### 跨界合作伙伴权益

**品牌权益：**
- 合作伙伴品牌露出
- 社媒联合传播

**内容权益：**
- 圆桌/Panel参与
- 案例分享机会

**对接权益：**
- 定向配对名额
- 双边社群触达

## 执行要点

### 优先级排序
1. **跨界主题确定** - 活动前2-3个月
2. **双边联合主办方** - 主题确定后立即对接（需同步推进）
3. **双边核心嘉宾** - 活动前1-2个月
4. **传播矩阵搭建** - 活动前3-4周
5. **配对机制设计** - 活动前2周

### 跨界活动特殊注意事项

| 要点 | 说明 |
|------|------|
| 双边平衡 | 合作方数量、嘉宾质量、发言时间需双边对等 |
| 术语桥梁 | 准备双边术语对照，安排懂双边的主持人 |
| 预期管理 | 跨界合作需时间沉淀，避免承诺即时成果 |
| 后续通道 | 建立跨界社群/对接机制，延续碰撞价值 |
| 内容审核 | 双边合规要求可能不同，内容需双边确认 |

### 常见问题预案

| 问题 | 预案 |
|-----|------|
| 双边参与比例失衡 | 动态调整邀请、分组时强制混合 |
| 对话无法深入 | 主持人引导、提前准备结构化问题 |
| 一方嘉宾取消 | 备选嘉宾需双边都准备、保持层级对等 |
| 术语沟通障碍 | 安排"翻译官"角色、提前发放背景材料 |
