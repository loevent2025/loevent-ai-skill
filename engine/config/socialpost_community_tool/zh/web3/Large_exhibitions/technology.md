# Web3 大型展会/博览会 - 技术类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- 区块链技术峰会、开发者大会、协议技术发布
- 基础设施展示、Layer1/Layer2 技术展
- ZK 技术峰会、模块化架构大会、共识机制研讨

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 主论坛 + 技术分论坛 + Demo 展区 + Workshop |
| 内容深度 | 底层架构、源码级分析、技术路线图发布 |
| 参与方 | 头部协议核心团队、知名开发者、技术研究员 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 协议核心开发者 | 底层架构、共识机制、性能优化 |
| P1 | 基础设施建设者 | SDK、开发框架、DevOps |
| P2 | 技术架构师 | 系统设计、技术选型、安全审计 |
| P3 | 安全研究员 | 漏洞分析、安全模型、形式化验证 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 协议技术发布会 | 正式专业型 (A) | 技术深度、权威发布 |
| 开发者技术大会 | 正式专业型 (A) | 技术分享、实战经验 |
| 技术研讨/Workshop | 悬念引导型 (C) | 技术难题、解决方案 |
| 黑客松配套技术展 | 福利导向型 (D) | 奖金、资源、孵化 |

## 📝 内容必含元素

### 1. 技术权威性（至少2处）
- "汇聚全球 [数量]+ 顶尖协议核心贡献者"
- "深度解析 [协议/技术] 底层架构演进"
- "首发披露 [协议/项目] 技术路线图"

### 2. 前沿议题（至少2处）
- "聚焦 [ZK/模块化/跨链] 技术最新突破"
- "探讨 [扩容/隐私/互操作性] 未来方向"

### 3. 实战价值（至少1处）
- "现场 Demo 演示最新开发工具"
- "获取第一手技术文档与代码示例"

### 4. 合规提示（必须包含）
- "⚠️ 本活动为技术交流，不涉及任何投资建议（NFA）"
