# Web3 Large-Scale Expo / Conference - Investment Event Knowledge Base

## 📌 Module Activation Conditions

Apply this module when the event theme involves the following keywords:
- Crypto investment summits, institutional investor conferences
- Project pitches, Pitch Days, fundraising matching
- Market trend forums, VC closed-door sessions, asset allocation

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Emphasis |
|------------|-------------------|---------------|
| Institutional Investor Summit | Formal & Professional (A) | Authoritative insights, in-depth analysis |
| Project Pitch / Demo Day | Formal & Professional (A) | Project showcase, investor matching |
| Market Trend Forum | Suspense-Driven (C) | Cycle analysis, trend forecasting |
| Investor Closed-Door Session | Benefits-Driven (D) | Scarce resources, exclusive access |

## 📝 Required Content Elements

### 1. Authoritative Endorsement (at least 2 instances)
- "Featuring in-depth insights from partners at [firm name]"
- "Bringing together [number]+ leading investment firms"

### 2. Market Insights (at least 2 instances)
- "Decoding [year] market trends and allocation strategies"
- "In-depth analysis of [sector] investment thesis and valuation frameworks"

### 3. Matching Value (at least 1 instance)
- "1-on-1 closed-door meetings between project teams and investors"
- "[Number] curated projects pitching live on-site"

### 4. Compliance Disclaimer (mandatory)
- "⚠️ All content is for informational purposes only and does not constitute investment advice (NFA)"
