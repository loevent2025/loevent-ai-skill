# Web3 Hackathon — Community Event Knowledge Base

## 📌 Module Activation Criteria

Apply the rules in this module when the event theme involves the following keywords:
- Community hackathon, ecosystem Hackathon, Builder Jam
- DAO hackathon, community co-building competition, open-source contribution challenge
- Themed creative competition, Meme / NFT creation marathon

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Online / offline development + community voting + showcase exchange + celebration party |
| Duration | 24–72 hours, relaxed atmosphere |
| Participants | Community members, Builders, creators, ecosystem projects |
| Core Objective | Community engagement, ecosystem development, Builder connections, culture building |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Focus Areas |
|----------|---------------|-------------|
| P0 | Community Builders | Hands-on project experience, community recognition, collaboration opportunities |
| P1 | Ecosystem Developers | Ecosystem contribution, official incentives, exposure opportunities |
| P2 | Creative Enthusiasts | Creative expression, community interaction, reward perks |
| P3 | Community Newcomers | Participation experience, joining the community, learning opportunities |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Ecosystem Co-Building Competition | Energetic & Engaging (B) | Community atmosphere, co-building spirit |
| DAO Hackathon | Energetic & Engaging (B) | Governance innovation, community participation |
| Creative / Meme Competition | Benefits-Oriented (D) | Fun participation, reward incentives |
| Open-Source Contribution Challenge | Curiosity-Driven (C) | Technical contribution, ecosystem value |

---

## 📝 Required Content Elements

### 1. Community Atmosphere (at least 2 instances)
- "Build alongside [number]+ Builders"
- "[Community name]'s largest co-building event of the year"

### 2. Participation Incentives (at least 1 instance)
- "Community votes to select the most popular project"
- "All participants receive an exclusive POAP / NFT"

### 3. Ecosystem Value (at least 1 instance)
- "Top projects receive official ecosystem support"
- "Eligible for community grant / treasury funding"

### 4. Community Spirit (recommended)
- "Let's BUIDL together!" / "WAGMI"

## ❓ FAQ Templates

| Question | Answer |
|----------|--------|
| Can I join without a technical background? | Yes! Creative, design, and operations skills are all needed |
| Do I have to use a specific tech stack? | Building on the ecosystem's tech stack is recommended but not required |
| How do I participate in community voting? | Hold the community NFT / Token to be eligible to vote |
| How do I claim my POAP? | Automatically sent to your wallet after you submit your project |

---

## ⚠️ Prohibited Actions

- ❌ Over-commercializing a community event
- ❌ Promising specific token airdrops
- ❌ Undermining the fairness of community voting
- ❌ Over-emphasizing prize money at the expense of the co-building spirit
