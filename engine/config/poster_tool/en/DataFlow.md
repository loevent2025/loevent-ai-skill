# NEON FLOW Style Guide (Complete Revised Edition v2)

---

## Applicable Scenarios

Tech summits, AI entrepreneur meetups, developer meetups, SaaS product launches, tech community presentations, startup pitch events, tech media features, data science exhibitions, AI forums, digital transformation conferences, hackathons, blockchain conferences, metaverse-themed events

---

## Color Palette

**Primary Colors:** Cyan, Magenta, Electric Blue, Neon Pink, Violet

**Arc Gradient Colors:** Magenta → Purple → Cyan → Blue (smooth multi-color gradient)

**Background:** Pure Black, Deep Charcoal Black (must be unified throughout — no zoning)

**Background Text Colors:**
- Standard text: Light gray (#666666 – #888888)
- Highlighted text: Magenta (#FF0080) or Cyan (#00FFFF)
- Approx. 30% highlighted + 70% gray

**Accent Colors:** Bright Pink, Cyan, White

**Saturation:** High-saturation neon effect in arc areas; low-saturation in background text

**Color Rendering:** Smooth multi-color gradients in arc areas; strict solid fills for all text elements

---

## Common Color Combinations

- **Classic Neon Flow:** Pure Black (#000000) + Magenta (#FF0080) + Cyan (#00FFFF) + Violet (#8A2BE2)

- **Tech Pink-Cyan:** Deep Charcoal (#0A0A0A) + Neon Pink (#FF1493) + Electric Blue (#00D4FF) + White (#FFFFFF)

- **Aurora Gradient:** Pure Black (#000000) + Rose Red (#FF007F) + Blue-Violet (#6A0DAD) + Ice Blue (#7DF9FF)

- **Warm-Cool Convergence:** Deep Black (#050505) + Coral Pink (#FF6B6B) + Teal (#00CED1) + Light Gray (#E8E8E8)

- **Pure Cyan Tech:** Pure Black (#000000) + Cyan (#00FFFF) + Electric Blue (#00D4FF) + Mint Green (#00FFA3)

## Visual Hierarchy (Back to Front)

1. **Base Layer: Pure Black Background** — Unified pure black canvas
2. **Second Layer: Digital Code Layer (required)** — Numeric and code elements covering the entire canvas
3. **Third Layer: Neon Light Arcs** — Floating, glowing streamline ribbons
4. **Top Layer: Text Information** — Clean white titles and event details

## Visual Element Library (Select 1–2 per design)

### Flowing Light Arc Types
- **Multi-layer Streamline Arcs** — Multiple parallel flowing curves, flowing naturally like silk ribbons
- **S-Curve Light Flow** — Elegant S-shaped ribbon with multi-color gradient transitions
- **Wave Light Arcs** — Gently undulating wave-shaped ribbons with natural flow
- **Converging Light Flow** — Streamlines converging from canvas edges toward the center
- **Radiating Light Arcs** — Arc lines radiating outward from a single focal point
- **Spiral Light Arcs** — Upward-spiraling or coiling ribbon forms

### Digital & Code Atmosphere Elements (Required — background must include)
- **Code Character Layer** — Ultra-low-opacity code text as textural background
- **Data Labels** — Scattered technical labels such as "AI_SYS", "DATA_STREAM", etc.
- **Variable Symbols** — Programming variables and function names as decorative text
- **Binary Data Stream** — Scattered "0"s and "1"s, some arranged vertically to suggest falling data
- **Floating Values** — Randomly distributed technical values such as "0.9847", "256", "1024", "3.14159", "0xFF00"
- **Coordinate Data** — Coordinate-style markers such as "[42.7, 89.3]", "x=1280"
- **Percentage Indicators** — Progress values such as "99.7%", "87%"
- **Hexadecimal Values** — Hex codes such as "#00FFFF", "0x7F"
- **Character Fog** — Ultra-low-opacity characters forming a hazy background layer

### Geometric Accent Elements
- **Solid Color Squares** — Small solid-color square accents in corner positions
- **Circular Logo Frame** — Circular container for brand identity
- **Thin-line Borders** — Optional thin-line frames for section delineation

## Element Selection Principles

- **Digital Foundation First:** Numeric and code elements are the core of the background — must be clearly visible, never omitted

- **Floating Arcs:** Neon light arcs "float above" the digital background — they do not "occupy" a zone

- **Natural Fade-out:** All arc edges must fade to transparent in all 360 directions, dissolving into the background

- **Unified Background:** The entire canvas shares one pure black background — no zoning

- **Text Legibility:** Primary text is placed in clean background areas to ensure maximum readability

- **Restrained Accents:** Geometric decorative elements must not exceed 3% of the canvas — keep the composition clean

---

## Effect Treatment

### Background Digital Layer (Most Critical — Establish This First)
- **Must be clearly legible:** Digital elements are the defining visual characteristic of this style — they must be immediately visible at a glance (not "barely visible")
- **Opacity settings:** Standard text: 30–50% opacity; highlighted text: 60–80% opacity — ensure readability
- **Color layering:**
  - Standard text: Light gray (#666666 – #888888), clearly legible
  - Highlighted text: Magenta (#FF0080) or Cyan (#00FFFF), more prominent
  - Approximately 30% of text uses highlight colors; 70% uses gray
- **Full canvas coverage:** Text elements extend from top to bottom, covering the entire background — including arc and text areas
- **Density distribution:** Medium-to-high density with appropriate spacing between elements, creating a dense data-stream visual
- **Monospace font:** All numeric elements use monospace fonts to reinforce the code aesthetic
- **Size settings:** Medium-to-large text size for readability; some labels may use bold
- **Layout:** Primarily horizontal rows with multiple labels and values per row, creating a "code wall" effect
- **Required element types:**
  - Technical labels (key elements, some bold and highlighted): "AI_ORG", "TOOL_STREAM", "DATA_FLOW", "NEURAL_NET", "CONNECT", "DIGITAL_INTEL", "ORG_WISDOM", "FUTURE_TECH", "INNOVATE", "STREAM", "LOOPI"
  - Technical values: "0.9847", "256", "1024", "3.14159", "0xFF00", "127.0.0.1"
  - Binary digits: "0" and "1" scattered throughout
  - Coordinate data: "[42.7, 89.3]", "x=1280", "y=720"
  - Percentages: "99.7%", "87%", "loading: 42%"
  - Hexadecimal values: "#00FFFF", "#FF0080", "0x7F"
- **No hard-edge elements:** Do not use circuit board, chip, or wiring patterns

### Background Color Treatment
- **Unified composition:** The entire canvas background is a uniform pure black from top to bottom
- **No zoning:** The background is one continuous black canvas — not divided into "arc area" and "text area"
- **No dividing lines:** Absolutely no horizontal dividing lines, seam artifacts, or zone boundaries

### Light Arc Treatment (Core Visual)
- **Floating glow:** Arcs resemble aurora ribbons drifting through a dark night sky — floating above the digital background
- **Multi-color gradient:** Magenta → Purple → Cyan → Blue; color transitions are smooth with no breaks
- **Streamline form:** Multiple parallel curves flowing like silk ribbons — elegant and fluid
- **Neon glow:** Each arc has a soft outer glow, simulating the halo of a real neon tube
- **Variable stroke width:** Subtle variation in arc thickness adds a sense of dynamic motion
- **Non-blocking:** Arcs are semi-transparent; digital elements behind them remain visible

### Edge Fade Treatment (Eliminate Hard Edges — Critical)
- **Omnidirectional fade:** All edges of every arc (top, bottom, left, right) must fade to transparent
- **No visible boundaries:** Absolutely no sharp edges or cutoff lines anywhere
- **Natural attenuation:** Glow gradually diminishes outward — opacity decays from 100% to 0%
- **Sufficient transition distance:** The fade transition must be long enough (50–80% of arc height)
- **Physical realism:** Resembles the actual physical scattering of light in darkness
- **Seamless integration:** The relationship between arc and background is "light glowing in darkness" — not a compositing seam

### Text Information Area
- **Shared background:** Text sits directly on the unified pure black background, sharing the same visual space as the arcs
- **Solid color fill:** All text uses a single flat color — no effects of any kind
- **Clear hierarchy:** Primary title, subtitle, and supporting text are clearly differentiated in scale
- **Coexists with digital layer:** The background beneath text areas also contains digital elements, but text takes visual priority

## Composition Strategy

- **Unified space:** The entire poster is one cohesive visual space — not a segmented layout

- **Digital foundation:** Numeric and code elements serve as the base texture covering the entire canvas

- **Floating arcs:** Neon arcs float naturally in the upper portion of the canvas, above the digital layer

- **Natural fade:** Arc light naturally dissipates outward, dissolving into the dark background

- **Anchored text:** Clear white text information is anchored at the bottom, occupying the top layer

- **Continuous flow:** The visual experience from top to bottom is continuous and uninterrupted

- **Visual guidance:** The arc form naturally guides the eye downward toward the text area

- **Flat front-facing view:** Maintain a direct front-facing perspective — no perspective distortion

---

## Typography Settings

**Fill Method:** Solid color fill, single color, no gradients

**Prohibited Effects:**
- No outer glow / inner glow
- No halo or aura effects
- No stroke / outline
- No drop shadow / shadow
- No gradient fill
- No contour lighting
- No text effects of any kind

**Primary Title:** Large bold typeface, white, bilingual Chinese-English mix supported

**Subtitle / Info Text:** Medium-to-small typeface, white or light gray

**Brand Identity:** Corner placement, visually consistent with overall style

**Overall Principle:** All text remains clean, legible, and highly readable — placed in clean background areas to create a dynamic contrast with the neon arc visuals. Typography and layout establish the professional tech aesthetic

## Smooth Transition Key Points

| Common Issue | Solution |
|---|---|
| Hard line / dividing line in center | Emphasize that the background is a unified whole — arcs "float above" the background, not "within" a zone |
| Segmented / collage appearance | Specify that arc edges must fade to transparent in all 360 directions |
| Unnatural transition | Require sufficient fade distance (50–80% of arc height) |
| Split-layer appearance | Reinforce that the entire canvas is a unified space — "light glowing in darkness" |
| Non-uniform background | Specify that the canvas is one pure black from top to bottom |
| Missing digital elements | Digital elements must be clearly visible (opacity 30–80%), covering the entire canvas |
| Digital layer too faint | Increase opacity: standard text 30–50%, highlighted text 60–80%; use light gray (#666–#888) and magenta/cyan highlights |

## Sample Prompt (AI Entrepreneur Meetup — Smooth Transition Optimized)

Render as a flat image, surface graphic design, pattern design. Canvas fills to the edges — no outer border, no canvas/paper texture, no model. Tech community event flat poster, direct front-facing perspective.

**[Layer 1: Pure Black Background]**
The entire canvas background is uniform pure black (#000000), consistent from top to bottom. No zones, no layering — one continuous, unified black canvas.

**[Layer 2: Digital Code Background — Core Visual Element, Must Be Clearly Legible]**
On the pure black background, dense numeric and code elements fill the canvas, forming a code-wall effect. This is the defining visual characteristic of the style — it must be immediately visible, not barely perceptible.

Color and opacity:
- Standard text: Light gray (#666666 – #888888), opacity 30–50%, clearly legible
- Highlighted text: Magenta (#FF0080) or Cyan (#00FFFF), opacity 60–80%, more prominent
- Approximately 30% highlighted + 70% gray

Layout: Horizontal rows with multiple labels and values per row, covering the full canvas (top to bottom, left to right).

Required elements (all must appear):
- Technical labels (key elements, some bold, some highlighted): "AI_ORG", "TOOL_STREAM", "DATA_FLOW", "NEURAL_NET", "CONNECT", "DIGITAL_INTEL", "ORG_WISDOM", "FUTURE_TECH", "INNOVATE", "STREAM", "LOOPI", "AI_CIKEEL", "NUL_NET", "SCIENTI"
- Technical values: "0.9847", "256", "1024", "3.14159", "0xFF00", "127.0.0.1"
- Binary digits: "0" and "1" scattered throughout
- Coordinate data: "[42.7, 89.3]", "x=1280", "y=720"
- Percentage indicators: "99.7%", "87%", "loading: 42%"
- Hexadecimal values: "#00FFFF", "#FF0080", "0x7F"

All elements use monospace fonts at a medium-to-large size for readability. Highlighted labels (e.g., TOOL_STREAM, DATA_FLOW, CONNECT) use magenta or cyan; standard labels use light gray. No circuit board, chip, or wiring patterns.

**[Layer 3: Neon Light Arcs — Floating Above the Digital Background]**
In the upper portion of the digital background, 5–7 flowing neon arc curves float above the canvas. The arcs resemble aurora ribbons drifting through a dark night sky — flowing naturally from left to right in elegant S-curves or wave forms. Arc colors begin in Magenta (#FF0080), transitioning smoothly to Violet (#8A2BE2), then Cyan (#00FFFF), and finally Electric Blue (#00D4FF) — a seamless multi-color gradient with no breaks. Each arc has a soft neon glow that radiates naturally outward like a real neon tube. Stroke width varies subtly along the arc to convey dynamic motion. Arcs are semi-transparent; digital elements behind them remain visible.

**[Arc Edge Fade Treatment — Eliminate Hard Edges]**
All edges of every arc (top, bottom, left, right) must fade to transparent — absolutely no visible boundary lines or cutoffs. Arc glow gradually diminishes outward; opacity decays from 100% at the core to 0% at the edge. This omnidirectional fade is a natural attenuation, like the true physical scattering of light in darkness. The glow beneath the arc must be handled with particular softness — the transition from core to transparent must be long enough (50–80% of arc height) to ensure no visible boundary exists between the arc zone and the background.

**[Brand & Decorative Elements]**
Two small solid magenta (#FF0080) squares in the upper-right corner as accent decorations — sharp edges, no shadow.

**[Layer 4: Text Information — Top Layer]**
Text information occupies approximately the lower 40% of the canvas. Text sits directly on the pure black background, sharing the same unified visual space as the arcs and digital background. Primary title: "AI Entrepreneur Meetup:" on the first line; "Organizing Tool Presentation" on the second line — white (#FFFFFF) bold large sans-serif typeface, left-aligned. Below: event date "2026.02.26 10:51" and location "Hong Kong" in white small typeface. Lower-right corner: brand name "loevent" in white medium typeface.

**[Typography Restrictions]**
All text uses strict solid color fills — single color only. No outer glow, no inner glow, no halo, no stroke, no shadow, no gradient, no text effects of any kind. Maintain clean, high-readability presentation.

**[Overall Visual Requirements — Most Critical]**
The entire poster is one cohesive, unified visual space comprising four layers: the pure black base, clearly visible digital code elements across the full canvas, floating neon arcs above, and white text on top. Digital elements must be clearly legible — not too faint to read. Arcs and background form a single unified space — absolutely no horizontal dividing lines, seam artifacts, or zone boundaries. This aesthetic is the visual language of the tech community: flowing neon arcs symbolize the movement and connection of innovative ideas; clearly visible digital elements reinforce the data-driven tech sensibility; the composition is professional, energetic, and inviting.
