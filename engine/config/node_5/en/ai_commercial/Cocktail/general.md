> Applicable to: AI industry networking receptions, product launch celebrations, conference after-parties, VIP appreciation dinners, community parties, and similar events

## Event Characteristics

| Category | Description |
|----------|-------------|
| Scale | 30–150 attendees |
| Duration | 2–4 hours (evening to night) |
| Format | Open networking + light sharing sessions + immersive atmosphere |

## Execution Essentials

### Priority Timeline
1. **Confirm venue** — 3–4 weeks before the event (book earlier for popular venues)
2. **Secure co-host** — 2–3 weeks before the event
3. **Finalize F&B** — 1–2 weeks before the event
4. **Launch invitations** — 1–2 weeks before the event

### Key Considerations for Networking Events
- **Brand alignment**: Partner brand identity must match the target audience profile
- **Natural integration**: Brand placements should blend into the atmosphere; avoid overt commercialization
- **Experience first**: Partner benefits must not compromise the attendee experience
- **Photo-ready**: All partner collateral should be designed for social media sharing
- **Headcount control**: Partner guest allocations must be agreed upon in advance

### Contingency Plans

| Issue | Contingency |
|-------|-------------|
| Beverage shortage | Confirm per-capita consumption in advance; secure backup procurement channels |
| Last-minute venue change | Maintain a backup venue; include clear terms in the contract |
| Excessive partner marketing | Set boundaries in advance; manage on-site execution |
| Higher-than-expected turnout | Confirm venue flexibility; control invitation distribution |
