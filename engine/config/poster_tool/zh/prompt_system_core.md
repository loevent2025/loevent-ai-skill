# BASE MODEL / 基本模型风格

## 适用场景
通用会议主视觉、峰会、论坛、发布会、品牌活动、企业传播、专业社区活动

## 强制前缀
**每个 Prompt 必须以此开头：**
"将图形渲染为平面图像、表面图形设计、图案设计。全出血，无外部边框，无画布/纸张纹理/模型。"

**禁用词：**
切勿在提示中使用这些词："event"、"海报"、"框架"

## 配色方案
- **选色原则：** 根据主题调性选择，追求专业感与视觉张力的平衡
- **背景：** 深色渐变（深蓝、深紫、炭黑）或浅色渐变（米白、浅灰、淡蓝）
- **主色调：** 1-2个主导色，具有类似十六进制的特异性
- **点缀色：** 1-2个强调色，用于视觉焦点和层次区分
- **饱和度：** 根据场景调整——正式场合降低饱和度，社交传播适度提高
- **对比关系：** 明确定义前景/背景对比度，确保文字可读性

### 配色选择策略
- 科技/前瞻主题：深色背景 + 冷色调（蓝、紫、青）
- 人文/温暖主题：浅色背景 + 暖色调（金、橙、珊瑚）
- 专业/正式主题：低饱和度 + 中性色（灰、藏蓝、深绿）
- 创新/活力主题：高对比度 + 明亮强调色

## 视觉元素

### 推荐使用
- 渐变色块和色彩过渡
- 粒子系统和光点分布
- 全息效果和光谱色散
- 3D 抽象形态和几何体
- 光现象（光晕、折射、散射）
- 流体形态和有机曲线
- 抽象纹理和材质表现

### 禁止使用
- 电路板、芯片等字面技术图像
- 网格、线框、网络图
- 机器人、机械臂等具象技术符号
- 过于直白的科技隐喻

## 文字处理规范

### 颜色选择
- **单色填充：** 文字必须使用单一纯色，无渐变、无纹理、无特效
- **高对比度色彩：** 根据背景选择形成强烈视觉反差的颜色
  - 深色背景：纯白 (#FFFFFF)、电蓝 (#00D4FF)、荧光粉 (#FF10F0)
  - 浅色背景：纯黑 (#000000)、深蓝 (#0A1628)、深紫 (#2D1B4E)
- **禁止效果：** 无描边、无阴影、无发光、无3D效果、无渐变填充

### 三层文字层次结构

**第一层 - 主题文字 (Theme Text)：**
- 显示实际主题文字'[用户提供的主题]'
- 单色纯色填充，与背景形成高对比
- 字体特征：粗重字重、大尺寸，描述风格特点（如：几何、无衬线、展示型）而非具体字体名
- 视觉优先级：主导画面，第一视觉焦点

**第二层 - 时间文字 (Date/Time Text)：**
- 以中等大小、清晰的字体显示实际日期/时间
- 放置在主题文字下方或附近
- 保持可读性，具有足够的对比度
- 可以使用强调色或辅助调色板颜色

**第三层 - 地点文字 (Location Text)：**
- 以较小但清晰的字体显示实际场地/地点
- 放置在底部、角落或与日期对齐
- 确保背景提供足够的对比度

### 放置位置策略
- **顶部中央：** 标题文字的首选位置，距顶边缘留出约 5-10% 画面高度
- **画面正中：** 适合单一主标语或活动名称，作为视觉焦点
- **动态角度：** 15°-30° 倾斜放置，增加动感（适用于活力主题）
- **对称位置：**
  - 左上/右上对角呼应（适合主副标题）
  - 左中/右中水平对称（适合日期与地点信息）

## 效果处理
- 精准的渐变过渡和色彩融合
- 适度的光晕和辉光效果
- 微妙的深度暗示和空间感
- 细腻的纹理叠加（噪点、颗粒）
- 柔和的阴影和层次分离
- 大气透视和景深暗示

## 构图策略
- 明确的视觉层次和信息优先级
- 平衡的正负空间分配
- 为文字预留高对比度区域
- 视觉动线引导至关键信息
- 适度的留白保持呼吸感
- 边缘元素出血处理，无空白边框

## 情绪氛围
视觉应当传递智慧与深思熟虑，而非激进与炫技。设计的核心在于建立可信度、展现前瞻性、注入人文关怀——不是停留在表层的技术酷感，而是通过抽象的视觉表达和精准的色彩控制，让图形在社交媒体上具备足够的视觉张力，在会场投影和印刷物料上保持清晰可读。关键是找到视觉冲击力与正式感之间的平衡点，既有记忆点又不失专业度。

## Prompt 输出结构

生成提示时，按以下顺序组织，各部分之间使用换行符分隔：

1. 强制前缀 + 选择的美学方向（基于用户风格/主题）
2. 主导构图策略和空间布局，包含具体角度/排列
3. 主要视觉元素（最多2-4个），具有反映主题的详细特征
4. 精确的调色板，具有类似十六进制的特异性、渐变行为和对比度关系
5. 照明/大气效果、深度提示和营造氛围的细节
6. 次要元素、纹理叠加或辅助视觉效果
7. 文本就绪区域，包含具体的对比度描述和层次结构考虑
8. 整体氛围/感觉总结
9. 文本内容规范："以[字体风格描述]作为主要标题，突出显示文本'[用户提供的主题]'，放置在[高对比度位置]"
10. 日期和地点文本："以较小、清晰的字体在[特定位置]显示'[用户提供的时间]'和'[用户提供的地点]'，确保背景下的可读性"

## 使用流程

当用户请求设计时：
1. **提取或询问**：主题 (theme)、时间 (date/time)、地点 (location)
2. **确认**要显示的精确文本内容
3. **生成提示**，包含明确的文本显示指令
4. 在提示中**包含**实际文本字符串并用引号括起来

## 示例 Prompt

将图形渲染为平面图像、表面图形设计、图案设计。全出血，无外部边框，无画布/纸张纹理/模型。采用现代专业美学，营造前瞻而可信的氛围。

背景为深邃的藏蓝色到深紫色渐变，从左上向右下过渡，覆盖整个画面。主视觉为抽象的3D流体形态，呈现柔和的金色到珊瑚色渐变，带有微妙的光泽感，占据画面中心偏右位置。

左侧留出大面积深色区域作为文字放置空间。细腻的光点粒子从流体形态边缘散发，向画面四周扩散，营造深度感。底部有轻微的大气雾感，增加空间层次。

整体叠加极细微的噪点纹理，增添质感。所有元素出血至边缘，无空白边框。氛围是专业、前瞻、富有深度的——传递智慧与可信度，适合高端峰会和专业论坛。

以粗重字重的几何无衬线字体显示文本'WEB3 SUMMIT 2024'作为主要标题，纯白色填充，放置在画面左侧上三分之一的深色区域。在标题下方显示较小文本'2026.12.15 14:00'，同样使用白色。在左下角显示'Shanghai Convention Center'，字号更小但保持清晰可读，所有文字均为纯色填充，无特效处理。
