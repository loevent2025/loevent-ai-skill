# Web3 Hackathon – Education Event Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- Beginner hackathon, introductory coding marathon, learning-focused Hackathon
- University hackathon, student developer challenge, bootcamp capstone competition
- Skill certification challenge, hands-on assessment competition, graduation project showcase

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Pre-event training + Mentor-led teams + Hands-on development + Project showcase |
| Duration | 24–48 hours, typically paired with a training curriculum |
| Participants | Learners, students, mentors, training institutions |
| Core Objective | Skill application, learning outcome validation, portfolio building |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Concerns |
|----------|---------------|--------------|
| P0 | Web3 Learners | Hands-on opportunity, skill validation, portfolio building |
| P1 | University Students | Resume boost, internship opportunities, industry entry point |
| P2 | Career-Switching Developers | Tech transition validation, project experience, job search preparation |
| P3 | Bootcamp Cohort Members | Curriculum application, capstone assessment, skill certification |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| University Hackathon | Energetic & Engaging (B) | Learning atmosphere, growth experience |
| Beginner Entry-Level Competition | Energetic & Engaging (B) | Low barrier, mentor guidance |
| Bootcamp Capstone Competition | Benefits-Oriented (D) | Certification, career opportunities |
| Skill Challenge | Suspense-Driven (C) | Skill validation, competency proof |

## ❓ FAQ Templates

| Question | Answer |
|----------|--------|
| Can I join with absolutely zero experience? | Yes! We offer pre-event training and mentor guidance throughout |
| Do I need to form my own team? | You can form your own team or be assigned to a group on-site |
| What if I don't finish my project? | Completion isn't the only measure — the learning process matters most |
| What's the certificate good for? | It serves as proof of learning and is recognized by select employers |

---

## ⚠️ Prohibited Actions

- ❌ Over-emphasizing competition and rankings
- ❌ Creating excessive pressure on beginners
- ❌ Exaggerating the employment value of certificates
- ❌ Promising guaranteed internship or job placements
