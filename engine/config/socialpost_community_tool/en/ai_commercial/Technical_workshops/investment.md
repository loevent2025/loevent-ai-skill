# AI Technical Workshop — Investment Event Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- AI technical due diligence workshop, investor technical assessment training

- AI startup technical planning, technical DD methodology

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Technical walkthrough + Case study teardown + Hands-on experience + Discussion |
| Duration | 3–4 hours |
| Participants | AI investors, FAs, founders, technical advisors |
| Core Objective | Technical understanding, due diligence skills, investment judgment, commercialization assessment |

| Priority | Audience Type | Key Concerns |
|----------|---------------|--------------|
| P0 | AI Investors | Technical assessment skills, due diligence methodology, investment judgment |

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|

| AI Technology Primer | Energetic & Engaging (B) | Accessible, introductory, practical |

## 📝 Required Content Elements

### 1. Assessment Value (minimum 2 instances)
- "Master the core framework for evaluating AI projects technically"
- "Learn to assess [technical moats / commercial viability]"

### 2. Hands-On Experience (minimum 1 instance)
- "Get hands-on experience with the AI product development process"
- "Live teardown of real-world AI project case studies"

### 3. Investor Perspective (minimum 1 instance)
- "Understand AI technical value through an investor's lens"
- "The essential question checklist for technical due diligence"

### 4. Compliance Disclaimer (required)
- "💡 This event is for technical learning purposes and does not constitute investment advice."
