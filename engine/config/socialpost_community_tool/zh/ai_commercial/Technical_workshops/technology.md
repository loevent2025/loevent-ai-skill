# AI 技术工作坊 - 技术类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- AI Agent 开发工作坊、Agent 架构实战课
- LLM 应用开发 Workshop、RAG 系统搭建实操
- Prompt Engineering 工作坊、Fine-tuning 实战课
- AI Infra 技术工作坊、模型部署实操

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 讲师授课 + 现场编码 + 导师辅导 + 作品展示 |
| 时长 | 3-6 小时（半天至全天） |
| 参与方 | AI 开发者、技术导师、模型厂商技术团队 |
| 核心目标 | 技能习得、动手实操、问题解决、最佳实践 |
