# 5. 柔光极简风格 / SOFT LIGHT MINIMALISM（高饱和浅色渐变版）

## 适用场景
高端美妆品牌、奢侈品营销、冥想/健康App、极简主义品牌、摄影画廊、精品酒店、婚礼/活动请柬、文学出版、艺术书籍封面、环境/自然品牌

## 配色方案
- **主色调：** 高饱和的明亮粉彩色系 - 樱花粉 (#FFB3D9)、薰衣草紫 (#C9A0FF)、天际蓝 (#A0D4FF)、蜜桃橙 (#FFCBA0)
- **背景色：** 高饱和浅色多彩渐变，从鲜粉 (#FFB8D0) 到亮紫 (#D4A0FF) 再到明蓝 (#A0E0FF) 的流动过渡
- **光晕色：** 暖粉高光 (#FFE0F0)、冷紫高光 (#F0E0FF)，透明度高，边缘柔和模糊
- **图像叠加：** 自然影像（山脉、海洋、天空）使用偏色处理，融入高饱和色调滤镜
- **饱和度：** 中高 (50-75%)，追求鲜艳明快但不刺眼的"糖果感"高级质感
- **色彩行为：** 颜色鲜明可辨，层次清晰，渐变过渡流畅自然

### 常用配色组合
- 糖果黎明：樱花粉 (#FFB3D9) → 薰衣草紫 (#C9A0FF) → 天际蓝 (#A0D4FF)
- 蜜桃晚霞：蜜桃橙 (#FFCBA0) → 玫瑰粉 (#FF9EC4) → 丁香紫 (#D9A0FF)
- 梦幻极光：薄荷青 (#A0FFE0) → 天蓝 (#A0D4FF) → 薰衣草 (#C4A0FF)

## 视觉元素
- 自然风景摄影（山脉、海洋、沙漠、天空），经过高饱和色调滤镜处理
- 极简的几何形状——通常是圆形、弧线或简单的字母形态
- 图像与渐变背景的柔和融合，边界不分明
- 彩色镜头光晕效果，带有粉/紫/蓝色调
- 双重曝光或叠加效果，色彩更加鲜明
- 大面积留白（渐变色留白）
- 隐约可见的辅助元素（半透明的几何或文字）
- 图像可以被裁切或只显示局部

## 效果处理
- 图像整体保持适度对比度，饱和度增强并偏向粉紫蓝色调
- 图像边缘使用大范围羽化与渐变背景融合 (80-150px)
- 添加与背景色调统一的高饱和色彩叠加层
- 模拟彩色镜头光晕的圆形高光，带有粉/紫/橙的微妙色彩
- 可选的轻微高斯模糊 (3-8px) 增加梦幻感
- 使用"滤色"或"柔光"混合模式叠加效果
- 整体色调鲜明——暖调（偏粉橙）或冷调（偏蓝紫）或全彩渐变
- 细微的颗粒/噪点可选 (3-8%不透明度)

## 构图策略
- 极度克制，元素数量最少化
- 渐变留白占60-75%，是构图的核心
- 图像通常位于画面一侧或角落，不居中
- 文字与图像保持"呼吸"距离
- 利用图像的自然线条引导视线
- 可以使用大面积单一元素（如一个大字母）作为框架
- 对称或极端不对称，避免中庸构图
- 所有元素都服务于"优雅鲜明"的整体氛围

## 文字设置
- **颜色选择：** 
  - 浅色渐变背景上使用深色文字形成对比
  - 推荐：深紫 (#4A2060)、深玫红 (#6A2050)、深蓝紫 (#3A3060)
  - 特殊情况可用：纯白 (#FFFFFF) 叠加在图像或渐变较深区域
- **填充方式：** 纯色填充，单一颜色，无渐变
- **禁止效果：** 
  - 无渐变填充
  - 无阴影/投影
  - 无外发光/内发光
  - 无描边
  - 无模糊效果
  - 无任何文字特效
- **字体风格：** 优雅的衬线体或极细的无衬线体，字重Ultra Light至Light，强调精致感
- **整体原则：** 文字保持干净、锐利、高可读性，通过字体选择和高对比度纯色与鲜艳渐变背景形成清晰的视觉层次

## 情绪氛围
清新、活力、甜美、梦幻、精致。整体传递一种"明媚而优雅"的气质，鲜艳的色彩带来愉悦感，极简的构图保持高级感。适合追求年轻活力、甜美浪漫和现代艺术气质的品牌。唤起欢快、轻盈和对美好生活的向往。
