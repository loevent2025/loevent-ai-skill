# Web3 Demo Day — General Event Knowledge Base

## 📌 Module Activation Conditions

Apply this module when the event is a comprehensive Demo Day covering multiple types:
- Integrated project showcase days, Multi-track demo exhibitions
- Annual/quarterly project reviews, Ecosystem panoramic showcases
- Multi-host joint Demo Days

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Multi-track showcase + Expert judging + Community interaction + Awards ceremony |
| Duration | Half-day to full-day |
| Participants | Project teams, Investors, Technical experts, Community members |
| Core Objectives | Achievement showcase, Multi-party matchmaking, Brand exposure, Ecosystem building |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Showcase project teams | Exposure opportunities, Multi-dimensional feedback, Resource access |
| P1 | Investors | Project discovery, Investment opportunities, Industry insights |
| P2 | Technical / Industry audience | Trend awareness, Learning and benchmarking, Network building |
| P3 | Media / KOLs | Coverage material, Content creation, Industry developments |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Large-scale comprehensive Demo Day | Formal Professional (A) | Scale impact, Diverse value |
| Ecosystem project showcase | Energetic & Engaging (B) | Ecosystem atmosphere, Broad participation |
| Annual achievement review | Formal Professional (A) | Achievement retrospective, Milestones |
| Open showcase day | Benefits-Driven (D) | Broad participation, Interactive experience |

## ⚠️ Prohibited Actions

- ❌ Framing project showcases as investment recommendations
- ❌ Overstating a project's development stage
- ❌ Leaking undisclosed project information
- ❌ Promising future project performance
