# MATRIX — Style Guide

## Applicable Use Cases

Conference key visuals, hackathons, NFT drops, crypto art festivals, cybersecurity events, blockchain summits, hacker culture meetups, tech disruption conferences

---

## Color Palette

- **Primary tones:** Bright neon green, toxic green, acid lime green
- **Backgrounds:** Pure black or near-black dark gray for maximum contrast
- **Supporting colors:** Deep green, emerald, white highlights
- **Saturation:** High saturation neon green paired with deep shadows
- **Color behavior:** Glowing green on black, high contrast, phosphorescent screen effect

---

## Standard Color Combinations

| Theme | Colors |
|---|---|
| **Classic Matrix** | Pure black background + bright neon green falling code |
| **Deep System** | Deep green + toxic green gradient on a black base |
| **Terminal Hacker** | Acid lime green + white highlights glowing on deep black |

---

## Visual Elements

- Cascading character streams and code rain (katakana, numerals, symbols)
- Terminal windows and command-line interfaces
- Binary code streams (0s and 1s)
- System status indicators and loading bars
- Wireframe grids and digital architecture
- Glowing cursor elements
- Green circuit board patterns
- Scrolling text and data streams
- Phosphorescent screen scanlines
- Digital readouts and system monitors

---

## Treatment & Effects

- Phosphorescent glow on green elements
- Motion blur on falling characters
- Trailing fade on code streams
- Screen scanline overlay
- Soft outer glow on bright elements
- Gaussian blur for background layer depth
- Brightness variation creating depth within the code rain
- Halo effect on the brightest characters
- Subtle grain simulating CRT monitor texture
- Vertical motion trails

---

## Composition Strategies

- Vertical cascading columns filling the full frame
- Layered depth with foreground and background code streams
- Concentrated bright zones drawing focal attention
- Strategically positioned terminal windows to establish hierarchy
- Asymmetric distribution of code density
- Dark negative space reserved for text placement
- Rhythmic vertical flow creating a sense of motion
- Perspective grid implying digital space
- Brighter elements at focal points, darker toward the edges
- Full-frame coverage with varying code intensity

---

## Typography Guidelines

- **Color selection:** Choose solid colors with high contrast against a pure black background
  - Recommended primary: Bright Neon Green `#00FF41` or Toxic Green `#39FF14`
  - Alternatives: Pure White `#FFFFFF`, Acid Lime Green `#BFFF00`
- **Fill method:** Solid color fill only — single color, no gradients
- **Prohibited effects:**
  - No phosphorescent glow
  - No outer glow / inner glow
  - No halo effects
  - No strokes
  - No shadows / drop shadows
  - No scanline overlay
  - No gradient fills
  - No text effects of any kind
- **Overall principle:** Typography must remain clean, sharp, and highly legible — the Matrix aesthetic is achieved through typeface selection and high contrast, not glow effects

## Mood & Atmosphere

Hacker mystique, digital awakening, system infiltration aesthetics. The overall energy is enigmatic, technical, and subversive — evoking underground hacker culture and the thrill of breaking into digital systems, treating code as both tool and art form, underpinned by a cyberpunk philosophical sensibility.

## Example Prompt

Render the graphic as a flat image, surface graphic design, pattern design. Full bleed, no external borders, no canvas/paper texture/mockups. Content fills the entire canvas to the edges with absolutely no margins or white space at corners. No visible text, letters, numbers, or color codes in the image. Matrix aesthetic with a digital code rain atmosphere. The background fills the entire frame in pure black, establishing the foundation for maximum contrast. Cascading streams of bright neon green characters fall vertically from top to bottom, covering the full width — a mix of abstract symbols and alphanumeric forms, but no legible text. Code streams vary in brightness: some columns glow intensely in foreground tones while others fade into the background, creating layered depth. The brightest characters emit a phosphorescent glow with a soft outer halo. Motion blur trails follow the falling characters, implying downward movement. Horizontal scanlines sweep subtly across the frame at low opacity, adding CRT monitor authenticity. Several vertical code columns glow more intensely in the left-center region, forming a focal point, while peripheral streams are darker. Terminal window frames in deep green wireframe are suggested in the upper right, with transparent fills revealing the code behind them. A deep green wireframe grid pattern emerges subtly in the lower portion, implying digital architecture. Small cursor elements blink in toxic green with a bright glow. Strategic dark zones preserve space for text overlay, particularly in the right third of the frame. Brightness variation throughout creates rhythm and depth — some characters nearly white-hot, others fading to deep emerald. A subtle grain texture overlay adds analog screen quality. All elements extend to the canvas edges; continuous code rain fills the full frame from top to bottom. The overall energy is enigmatic and deeply technical — pure Matrix aesthetics celebrating hacker culture and digital infiltration, ideal for hackathons and crypto events that embrace cyberpunk philosophy. **Text areas use bright neon green (`#00FF41`) or pure white (`#FFFFFF`) solid fill only — single color, no glow, no halo, no stroke, no shadow, no scanline overlay, no gradient, no text effects of any kind. Maintain clean, sharp, high-legibility typography with maximum contrast against the pure black background.** Important: Pure visual graphics only — no text or alphanumeric characters in the generated image.
