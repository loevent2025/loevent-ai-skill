# Web3大型展会场景知识库 - 社区类

## 场景概述
社区类Web3大型展会聚焦Builder和爱好者的交流互动，强调去中心化精神、平等对话和社群归属感。这类活动规模灵活、形式多样，是Web3社群建设、链上经验分享和全球Builder连接的重要载体，也是Web3文化、Meme精神和社区自治的重要体现。

---

## 场景特征
- **规模**：数十至数百人参与
- **时长**：通常2-4小时，大型社区日可全天
- **形式**：Meetup分享 + 圆桌讨论 + 自由交流 + 主题沙龙 + IRL社交
- **参与者**：Builder、开发者、DAO成员、创业者、爱好者、学生
- **特点**：氛围轻松、互动性强、去中心化组织、Web3原生社交

## 社区运营建议

### 活动筹备
- 提前2周开始宣传（Twitter + TG群 + Farcaster）
- 选择交通便利、氛围轻松的场地
- 准备签到和POAP领取流程
- 安排破冰和引导环节
- 准备茶歇和小食（咖啡是标配）

### 氛围营造
- 组织者热情GM欢迎新Builder
- 设计简单的破冰（如：你在Build什么？）
- 准备话题引导卡片
- 鼓励大家互相介绍项目和方向
- 保持轻松友好的Web3原生氛围

### 社群维护
- 活动后发总结Thread和照片
- 建立持续活跃的TG/Discord交流群
- 定期组织线上/线下IRL活动
- 鼓励成员自发组织分享和Meetup
- 积累社区的内容、故事和链上记忆
