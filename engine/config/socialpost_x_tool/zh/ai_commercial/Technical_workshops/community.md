# 技术工作坊 × 社区类（AI/Model 商业化版）

## 适用场景
开源项目贡献者工作坊、LLM 框架使用培训、社区共学工作坊、AI 工具使用教学、生态项目实操培训、社区 Onboarding 工作坊

## 场景特征
- **形式**：社区教学、共学互助、贡献者培训、实操练习
- **核心价值**：社区技能共享、贡献者培养、生态熟悉、社区连接

## 受众定位
- **主要受众**：社区成员——想学习社区项目的开发者
- **次要受众**：潜在贡献者——想开始为项目贡献的人
- **第三受众**：新成员——刚加入社区想了解的人
- **延伸受众**：生态开发者——在生态内开发的团队
- **受众行为**：学习社区项目、准备贡献、了解生态、建立连接

### 常用受众组合
- **贡献者培训**：潜在贡献者为主，适合贡献入门工作坊
- **用户学习**：社区用户为主，适合使用技巧工作坊
- **共学互助**：各层次成员，适合共学小组形式

## 内容元素
- 社区身份召唤
- 工作坊主题（社区/生态导向）

- 贡献引导（如何参与）
- 社群入口

```
🛠️💙 LangChain 贡献者入门工作坊 | 开始你的开源贡献

⏰ 9月30日 14:00-17:00
📍 线上 Zoom + Discord

---

**想给 LangChain 贡献代码但不知道从哪开始？这个工作坊专门为你准备。**

🎯 学习目标：
- 了解 LangChain 的代码结构
- 掌握贡献流程（从 Issue 到 PR）
- 完成你的第一个贡献（可以是文档、测试或小 Feature）
- 知道遇到问题找谁问

- 在导师帮助下完成修改
- 提交你的第一个 PR

- 怎么参与 Review
- 怎么参与讨论

- LangChain 核心贡献者
- 活跃社区成员

📋 前置要求：
- 用过 LangChain

- 有 Git 基础

🎁 参与者获得：
- 你的第一个 PR（如果顺利的话）
- 贡献者指南文档
- 贡献者 Discord 频道入口
- 持续的社区支持
```

## 注意事项

### 社区类工作坊要点
1. **社区归属感是核心**：强调"我们""社区的"
2. **贡献引导**：不只是学习，还有贡献路径
3. **社区成员做导师**：让社区教社区
4. **长期连接**：工作坊后的社群入口
5. **共学氛围**：互相帮助，不是单向教学
