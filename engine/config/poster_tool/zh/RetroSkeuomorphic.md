RETRO TECH 复古科技风格
适用场景
怀旧科技品牌、复古设计项目、科技历史展览、创意工作室视觉、数字艺术活动、极简主义品牌、设计师社区、科技博物馆海报
配色方案
主色调： 米白色、暖灰色、复古奶油色、柔和象牙白、淡黄色、浅棕色
背景： 纯黑色、深炭黑
饱和度： 低饱和度，强调复古质感
色彩表现： 平面色块，无渐变，高对比度
常用配色组合
- 经典复古： 纯黑色 (#000000) + 米白色 (#F5F5DC) + 暖灰色 (#D3D3D3) + 淡黄色 (#FFF8DC)
- 极简怀旧： 深炭黑 (#1A1A1A) + 奶油色 (#FFFDD0) + 浅灰色 (#E8E8E8) + 浅棕色 (#D2B48C)
- 科技记忆： 黑色 (#000000) + 象牙白 (#FFFFF0) + 中灰色 (#BEBEBE) + 米色 (#F5F5DC)
- 设计师款： 纯黑 (#000000) + 纸白色 (#FAF9F6) + 银灰色 (#C0C0C0) + 杏色 (#FFEBCD)

元素选择原则
- 单一主元素： 1个元素占据画面60-80%，作为绝对视觉焦点
- 一主一辅： 主元素占50-65%，辅助元素占15-25%，明确主次关系
- 元素自由组合： 任意两个元素均可组合，不限于传统搭配（如软盘+键盘，鼠标+显示器等）
效果处理
- 真实设备还原： 保留设备的真实比例、细节和结构特征
- 平面摄影感： 如同产品摄影的俯视平面图，而非插画或图标
- 精确几何呈现： 保持设备真实的边角、接缝、按键、插槽等细节
- 纯色填充： 无渐变，单色色块，但保留设备本身的结构分区
- 高对比度： 黑色背景与浅色设备的强烈对比
- 无阴影投影： 完全去除任何阴影效果
- 无光影处理： 不表现光源和反射
- 设备级细节： 螺丝孔、通风口、接口、标识等真实存在的细节要保留
- 材质统一化： 所有表面统一为平面哑光质感，无塑料光泽
构图策略
- 单一主元素构图： 居中对称，占据画面主要视觉空间
- 一主一辅构图： 主元素中心或略偏，辅助元素侧边或角落，尺寸明显小于主元素
- 第一视角俯视： 正上方垂直俯视角度
- 黑色背景衬托： 纯黑背景突出主体设备
- 留白用于信息： 周围空间容纳文字内容
- 视觉重心平衡： 注意元素位置的重量感
文字设置
填充方式： 纯色填充，单一颜色，无渐变
禁止效果：
- 无外发光/内发光
- 无光晕效果
- 无描边
- 无阴影/投影
- 无渐变填充
- 无轮廓光
- 无任何文字特效
整体原则： 文字保持干净、清晰、高可读性，采用经典衬线或无衬线字体，与复古平面风格的简洁美学保持一致，通过字体选择和排版建立怀旧专业感
情绪氛围
怀旧温暖、经典永恒、设计纯粹、极简优雅。整体氛围是致敬科技历史的、人文关怀的、设计驱动的——通过标志性的复古科技元素唤起人们对个人计算机黄金时代的集体记忆。
