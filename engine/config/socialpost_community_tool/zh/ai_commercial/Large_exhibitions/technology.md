# AI 大型展会/博览会 - 技术类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- AI Agent 技术峰会、智能体开发者大会
- LLM 技术论坛、大模型架构峰会
- AI Infra 展会、MLOps 技术大会
- RAG/Fine-tuning 技术专场、Prompt Engineering 峰会

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 主论坛 + 技术分论坛 + Demo 展区 + Workshop |
| 内容深度 | Agent 架构、模型原理、系统设计、性能优化 |
| 参与方 | 头部 AI 公司技术团队、模型厂商、AI 基础设施提供商 |
| 核心目标 | 技术前沿分享、架构方案交流、开发者生态建设 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | AI 架构师/算法工程师 | Agent 架构、模型选型、性能优化 |
| P1 | AI 应用开发者 | 框架使用、API 调用、最佳实践 |
| P2 | AI Infra 工程师 | 推理优化、部署方案、成本控制 |
| P3 | 技术管理者/CTO | 技术趋势、团队建设、技术决策 |

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|

## 📝 内容必含元素

### 1. 技术权威性（至少2处）
- "汇聚 [数量]+ 头部 AI 公司核心技术团队"
- "深度解析 [Agent/模型] 底层架构演进"
- "首发披露 [公司/产品] 技术路线图"

### 2. 前沿议题（至少2处）
- "聚焦 [多Agent协作/RAG优化/推理加速] 最新突破"
- "探讨 [Agent稳定性/幻觉控制/成本优化] 解决方案"

### 3. 实战价值（至少1处）
- "现场 Demo 演示最新 Agent 开发工具"
- "获取一手技术文档与代码示例"

### 4. 务实提示（必须包含）
- "💡 本活动聚焦技术实践，内容基于当前模型能力，AI 技术持续演进中"

## ⚠️ 禁止事项

- ❌ 夸大 AI 能力，如"替代人类""100%准确"
- ❌ 承诺具体效果提升数字（除非有真实案例支撑）
- ❌ 使用"AI 万能""颠覆一切"等过度营销话术
- ❌ 杜撰模型能力或 API 功能
