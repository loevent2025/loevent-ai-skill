# RETROWAVE / VAPORWAVE 复古浪潮风格

## 适用场景
大会主视觉、黑客马拉松活动、NFT空投活动、加密艺术节、区块链社区聚会、文化展示

## 标志性色系
- **主色:** 深紫色 (#1A0F3D), 霓虹粉 (#FF1B8D), 霓虹青色 (#00D4FF), 活力洋红色 (#8B4A9E)
- **渐变:** 紫色到粉色再到青色的多层渐变
- **背景:** 带有星粒的深空紫
- **霓虹色:** 高饱和度的粉色、青色、紫色，带有发光效果
- **阴影:** 深紫黑色 (#0A0520)
- **特点:** 复古霓虹、蓝粉渐变、怀旧调色、软光辉光、80-90年代美学

## 视觉元素

### 插画风格
- **线条:** 手绘漫画/卡通风格，带有粗黑描边 (3-5px)
- **对象:** 干净的元素（加密货币硬币、NFT框架、钱包图标、代币徽章、DAO徽章）
- **透视:** 对角俯视，动态倾斜构图
- **风格:** 带有维度阴影的扁平插画
- **细节:** 手写字体、科技标签、区块链排版
- **纹理:** 轻微颗粒或噪点以获得模拟感（可选，5-10%）

### 元素
- **加密符号:** 比特币、以太坊和山寨币标志，带有霓虹处理
- **NFT 工艺品:** 数字艺术框架、收藏卡、像素艺术显示
- **简单图标:** 数字钱包、DAO徽章、代币徽章
- **几何形状:** 抽象药丸、胶囊、几何装饰
- **元宇宙对象:** 3D头像、虚拟方块、数字穿戴设备
- **代币表示:** 硬币堆、代币徽章、简单标志

### 背景与装饰
- **太空背景:** 带有散落星粒的深紫色星空
- **网格线:** 可选的复古网格透视（蒸汽波风格）
- **浮动元素:** 抽象几何形状、光粒、闪光、简单的药丸/胶囊
- **大气效果:** 紫色薄雾、柔和雾气、霓虹迷雾

## 效果处理

### 霓虹光效
- **外发光:** 所有明亮元素上都有强烈的霓虹发光 (20-40px)
- **边缘照明:** 物体上明亮的粉色/青色边缘光
- **柔和泛光:** 高斯模糊光晕效果 (30-50px半径)
- **光线:** 穿过构图的对角渐变光束
- **反射:** 物体表面微妙的霓虹反射

### 色彩渐变
- **多点渐变:** 从紫色 → 粉色 → 青色的4-6个颜色点
- **渐变叠加:** 整个构图的色彩洗涤
- **色差:** 边缘轻微的RGB颜色分离（可选，2-3px）
- **模拟色彩分级:** 电影般的色彩偏移，怀旧的温暖
- **饱和度:** 霓虹色高饱和度，暗部去饱和

### 质感处理
- **线条质量:** 手绘的不完美线条，有机感
- **阴影:** 物体上的赛璐珞阴影或渐变阴影
- **高光:** 反光表面上的明亮镜面高光
- **颗粒:** 可选的胶片颗粒或VHS噪点（5-8%不透明度）
- **表面:** 哑光和光泽表面的混合

## 构图策略

### 透视与布局
- **视角:** 30-45°俯视对角透视
- **构图:** 物体对角线或放射状排列，间距清晰
- **焦点:** 中央英雄对象（最大、最亮）
- **平衡:** 元素分离良好，不对称动态平衡
- **深度:** 重叠对象创建空间层次，同时保持清晰

### 元素排列
- **英雄对象:** 中心焦点（占画面30-40%），最高对比度 - 主加密硬币或NFT艺术品
- **辅助对象:** 3-5个次要干净元素（每个占15-20%）- 代币、钱包图标、NFT框架、几何形状
- **小细节:** 简单的散落粒子、星星、小型几何装饰（5-10%）
- **负空间:** 25-30%的呼吸空间，主要在背景中
- **动态流:** 对角线引导视线移动
- **清晰分离:** 每个元素间距明显，没有杂乱重叠

### 视觉层次
- **前景:** 最亮的霓虹对象，发光最强
- **中景:** 亮度适中的辅助元素
- **背景:** 黑暗空间，带有星星、大气效果
- **发光层:** 对象后面的独立光效层

## 文字设置
- **颜色选择:** 选择与深紫色背景形成高对比度的纯色
  - 主标题推荐: 纯白色 (#FFFFFF)
  - 强调文本推荐: 霓虹粉 (#FF1B8D) 或霓虹青色 (#00D4FF)
  - 备选色: 活力洋红色 (#8B4A9E)
- **填充方式:** 纯色填充，单一颜色，无渐变
- **禁止效果:** 
  - 无外发光/内发光
  - 无霓虹光效
  - 无描边（文字本身无描边，区别于插画元素的粗黑描边）
  - 无阴影/投影
  - 无渐变填充
  - 无色差效果
  - 无任何文字特效
- **整体原则:** 文字保持干净、简洁、高可读性，与背景的霓虹光效和插画元素形成对比，通过字体选择和排版建立复古感而非依赖特效
