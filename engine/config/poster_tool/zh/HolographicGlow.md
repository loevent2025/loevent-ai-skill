# HOLOGRAPHIC 全息幻彩风格

## 适用场景
NFT发售、DAO新提案、产品发布、艺术家合作、潮流社区活动、未来主义品牌推广、限量版发布、元宇宙激活

## 配色方案
- **主色调:** 虹彩光谱，珠光变幻色调
- **背景:** 深黑色、深紫色或渐变全息表面
- **饱和度:** 可变——根据视角从柔和色调变为霓虹色调
- **色彩表现:** 色差、彩虹渐变、多方向色彩偏移

### 常用配色组合
- 经典全息: 青色 (#00FFFF) → 洋红色 (#FF00FF) → 黄色 (#FFFF00) 渐变，带绿色 (#00FF00) 点缀
- 珠光: 柔和粉色 (#FFB3E6) → 淡紫色 (#E0BBE4) → 薄荷绿 (#B4F8C8) → 蜜桃色 (#FFA07A)
- 霓虹虹彩: 电光蓝 (#0D7EFF) → 热粉色 (#FF006E) → 柠檬绿 (#CCFF00) → 紫色 (#9D4EDD)
- 镀铬梦幻: 银白色 (#F0F0F0) → 浅蓝色 (#C4E7FF) → 玫瑰金 (#F4C2C2) → 丁香紫 (#D4BBFF)

## 视觉元素
- 彩虹渐变网格叠加
- 棱镜光线折射效果
- 全息箔片纹理模拟
- 液态镀铬表面
- 珠光闪烁高光
- 多色边缘发光
- 虹彩透明图层
- 边缘色差 (RGB分离)
- 金属反射表面
- 带有彩虹光泽的气泡状球形元素

## 效果处理
- 多方向颜色渐变
- 棱镜光线散射
- 高斯模糊用于发光效果
- 叠加混合模式用于颜色偏移
- 金属镜面高光
- 柔和的虹彩过渡
- 边缘色度分离 (RGB通道偏移)
- 带颜色折射的透明度
- 光泽湿表面反射
- 极光状彩色波纹

## 构图策略
- 中心焦点，放射状全息效果
- 分层透明元素营造深度
- 从深色到发光的渐变背景
- 带有全息处理的文字或徽标
- 漂浮的虹彩表面几何形状
- 梦幻般的大气散景背景
- 光源效果产生彩虹折射
- 对称或居中构图以实现平衡
- 哑光黑色与光泽全息元素之间的对比

## 文字设置
- **颜色选择:** 选择与深色背景形成高对比度的纯色
  - 深黑/深紫背景推荐: 纯白色 (#FFFFFF)、青色 (#00FFFF)、热粉色 (#FF00FF)
  - 备选纯色: 柠檬绿 (#CCFF00)、电光蓝 (#0D7EFF)
- **填充方式:** 纯色填充，单一颜色，无渐变
- **禁止效果:** 
  - 无金属镀铬饰面
  - 无液态镀铬效果
  - 无彩虹渐变
  - 无棱镜效果
  - 无色差/RGB分离
  - 无边缘发光
  - 无虹彩偏移
  - 无任何文字特效
- **整体原则:** 文字保持干净、锐利、高可读性，通过字体选择和尺寸建立视觉冲击力，与背景的全息效果形成清晰对比
