# Web3黑客松场景知识库 - 投资类

## 场景概述
投资类Web3黑客松聚焦创业项目孵化和融资对接，参与者不仅要完成技术开发，更要展示商业价值和市场潜力。这类活动是Web3创业者获取融资、验证商业模式、对接投资人和生态基金的重要渠道，也是投资机构发现早期优质链上项目的高效途径。Web3独特的Token经济和社区驱动模式，为投资类黑客松增添了更多融资可能性。

---

## 场景特征
- **规模**：数十至数百人，精品型为主
- **时长**：通常48-72小时开发 + 路演评审 + 后续孵化
- **形式**：组队开发 + 商业辅导 + 投资人路演 + 孵化对接
- **参与者**：Web3创业者、技术开发者、产品经理、投资人（VC/生态基金）、孵化器
- **特点**：商业导向强、资源对接丰富、后续机会多

## 参赛攻略建议

### 商业准备
- 想清楚解决什么链上/链下问题、为谁解决
- 了解目标市场的规模和链上数据
- 研究竞品协议和差异化优势
- 设计好Tokenomics和代币分配方案
- 思考获客、TVL增长和社区冷启动策略

### 路演准备
- 准备3分钟和5分钟两个版本
- 痛点→方案→Tokenomics→市场→团队→融资需求
- 用链上数据说话，避免空泛描述
- 准备好常见问题的回答（如何防止代币抛压等）
- 练习到流畅自然

### 投资人沟通
- 了解参与VC/生态基金的投资偏好和赛道
- 准备好不同详细程度的Deck和白皮书
- 对自己的估值和Token Warrant有基本判断
- 坦诚回答问题，不要夸大TVL或用户数据
- 主动跟进，约定下一步

### 团队组建
- 合约开发+前端+商业/运营组合很重要
- 有赛道行业背景会加分
- 路演表达能力要强
- 执行力比想法更重要
- 有社区运营经验者优先
