# AI Developer Meetup – Education Event Knowledge Base

## 📌 Module Activation Criteria

Apply the rules in this module when the event theme involves the following keywords:
- AI beginner meetups, newcomer gatherings, learner community events
- University AI developer meetups, student AI gatherings
- AI career transition meetups, job-seeker sharing sessions

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Experience sharing + Q&A session + open discussion + networking break |
| Duration | 2–3 hours |
| Participants | AI learners, students, career changers, senior developers (as speakers) |
| Core Objective | Learning exchange, gaining practical insights, career guidance, peer connections |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Interests |
|----------|---------------|---------------|
| P0 | AI Learners | Study methods, learning path guidance, getting questions answered |
| P1 | Students | Career direction, internship opportunities, skill preparation |
| P2 | Career Changers | Transition experience, transferable skills, job search advice |
| P3 | Senior Developers | Giving back to the community, talent discovery, mentorship |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Beginner Meetup | Energetic & Engaging (B) | Encouraging atmosphere, casual exchange |
| Student-Focused Event | Energetic & Engaging (B) | Career inspiration, learning enthusiasm |
| Career Transition Sharing | Suspense-driven (C) | Navigating uncertainty, building confidence |
| Job Search Meetup | Benefits-oriented (D) | Job search resources, referral opportunities |

---

## 📝 Required Content Elements

### 1. Learning Value (at least 2 instances)
- "Senior developers share their AI learning journey"
- "Common beginner questions answered live"

### 2. Growth Support (at least 1 instance)
- "Learning path guidance and recommendations"
- "Meet peers at the same stage of their journey"

### 3. Welcoming Atmosphere (at least 1 instance)
- "Relaxed and friendly — no such thing as a 'dumb question'"
- "A meetup designed specifically for newcomers"

### 4. Disclosure Note (recommended)
- "💡 A learning community event focused on AI skill development and experience sharing"

## ⚠️ Prohibited Actions

- ❌ Technical elitism or belittling beginners
- ❌ Aggressively promoting paid courses
- ❌ Promising that learning AI guarantees a job
- ❌ Creating learning anxiety
