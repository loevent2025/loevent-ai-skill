# Web3 Demo Day - 通用类活动知识库

## 📌 模块激活条件

当活动为综合性 Demo Day，涵盖多种类型时，应用本模块规则：
- 综合性项目展示日、多赛道成果展示
- 年度/季度项目汇报、生态全景展示
- 多主办方联合 Demo Day

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 多赛道展示 + 专业评审 + 社区互动 + 颁奖典礼 |
| 时长 | 半天至一天 |
| 参与方 | 项目团队、投资人、技术专家、社区成员 |
| 核心目标 | 成果展示、多方对接、品牌曝光、生态建设 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 展示项目方 | 曝光机会、多维反馈、资源对接 |
| P1 | 投资人 | 项目发现、投资机会、行业洞察 |
| P2 | 技术/行业观众 | 趋势了解、学习借鉴、人脉拓展 |
| P3 | 媒体/KOL | 报道素材、内容创作、行业动态 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 大型综合 Demo Day | 正式专业型 (A) | 规模影响、多元价值 |
| 生态项目展示日 | 热情活力型 (B) | 生态氛围、广泛参与 |
| 年度成果汇报 | 正式专业型 (A) | 成果回顾、里程碑 |
| 开放式展示日 | 福利导向型 (D) | 广泛参与、互动体验 |

---

## 📝 内容必含元素

### 1. 展示规模（至少2处）
- "[数量] 个项目集中展示"
- "覆盖 [赛道列表] 等多个领域"

### 2. 评审/观摩阵容（至少1处）
- "[数量]+ 位评审专家/投资人"
- "技术专家 + 投资人 + 行业导师"

### 3. 互动价值（至少1处）
- "现场体验项目 Demo"
- "与项目团队直接交流"

### 4. 合规提示（必须包含）
- "⚠️ 项目展示不构成投资建议（NFA）"

## ⚠️ 禁止事项

- ❌ 将项目展示包装成投资推荐

## 📋 场景四（Demo Day）总览表

| 活动类型 | 首选风格 | 核心受众 | 关键元素 |
|----------|----------|----------|----------|
| 技术类 | 正式专业型 (A) | 技术评审/开发者 | 技术验证、Demo 演示、专家点评 |
| 投资类 | 正式专业型 (A) | 投资机构 | 融资对接、投资人评审、1v1 对接 |
| 教育类 | 热情活力型 (B) | 学员/企业 HR | 成果展示、就业对接、证书颁发 |
| 社区类 | 热情活力型 (B) | 社区成员 | 社区投票、POAP、派对庆祝 |
| 通用类 | 正式专业型 (A) | 多元观众 | 多赛道、综合评审、广泛曝光 |
