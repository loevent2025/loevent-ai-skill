# Web3 Technical Workshop — Community Event Knowledge Base

## 📌 Module Activation Criteria

Apply the rules in this module when the event theme involves the following keywords:
- Community technical sharing, Builder workshop, open-source contribution training
- DAO tooling training, community developer skill exchange
- Ecosystem tooling tutorial, community Coding Session

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Skill sharing + hands-on practice + peer learning + community exchange |
| Duration | 2–4 hours, relaxed atmosphere |
| Participants | Community Builders, tech enthusiasts, ecosystem developers |
| Core Objective | Skill sharing, community bonding, mutual growth, ecosystem contribution |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Focus Areas |
|----------|---------------|-------------|
| P0 | Community Builders | Skill development, contribution opportunities, peer connections |
| P1 | Tech Enthusiasts | Learning exchange, hands-on practice, community belonging |
| P2 | Ecosystem Developers | Tooling mastery, ecosystem contribution, collaboration opportunities |
| P3 | New Builders | Getting started, joining the community, finding direction |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Community Technical Sharing | Energetic & Engaging (B) | Relaxed atmosphere, peer learning |
| Ecosystem Tooling Training | Energetic & Engaging (B) | Practical skills, ecosystem contribution |
| Builder Coding Session | Curiosity-Driven (C) | Technical challenges, solving together |
| Open-Source Contribution Training | Benefits-Oriented (D) | Contribution opportunities, community recognition |

---

## 📝 Required Content Elements

### 1. Community Atmosphere (at least 2 instances)
- "Learn alongside community Builders"
- "Peer learning, growing together"

### 2. Practical Skills (at least 1 instance)
- "Master practical tips for [tool / skill]"
- "Learn to contribute to the ecosystem"

### 3. Connection Value (at least 1 instance)
- "Meet like-minded Builders"
- "Find your coding partner"

### 4. Community Spirit (recommended)
- "BUIDL together, grow together" / "WAGMI"
