# AI Hackathon — Technical Events Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- AI Agent Development Hackathon, Intelligent Agent Programming Marathon
- LLM Application Development Competition, RAG / Fine-Tuning Technical Challenge
- Prompt Engineering Competition, AI Infra Innovation Challenge
- Multimodal Application Hackathon, AI Tool Development Competition

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Pre-event training + Team-based development + Mentor guidance + Demo showcase + Judging & awards |
| Duration | 24–72 hours of intensive development, or 1–4 week online competition |
| Participants | AI developers, model vendor technical teams, technical mentors, judging experts |
| Core Objective | Technical innovation validation, prototype development, talent discovery, ecosystem building |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | AI Application Developers | Technical challenge, prizes / resources, project incubation |
| P1 | AI Architects / ML Engineers | Technical exploration, innovation validation, best practices |
| P2 | Technical Founders | MVP development, team formation, funding opportunities |
| P3 | Students / Career Switchers | Hands-on practice, resume building, industry entry point |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Model Vendor Official Hackathon | Formal & Professional (A) | Technical depth, official endorsement, ecosystem value |
| Community Developer Competition | Energetic & Engaging (B) | Participation atmosphere, learning and growth, community connections |
| Technical Challenge | Suspense-Driven (C) | Technical problems, innovation directions, breakthrough opportunities |
| High-Prize / Incubation Competition | Value-Driven (D) | Prize pool, API credits, incubation resources |
