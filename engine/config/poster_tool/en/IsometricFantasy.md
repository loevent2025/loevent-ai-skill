# ISOMETRIC FANTASY — Design Style Guide

---

## Intended Use Cases

Community events, metaverse launches, NFT collections, gaming guilds, ecosystem visualizations, conference announcements, virtual world reveals.

---

## Color Palette

| Role | Color | Hex |
|------|-------|-----|
| **Primary** | Soft macaron tones — sky blue, ivory/cream, lavender | — |
| **Background** | Light gradient: sky blue to lavender | `#B8D4F1` → `#E8D5F2` |
| **Accent 1** | Blush pink | `#FFD1DC` |
| **Accent 2** | Mint green | `#C8E6C9` |
| **Accent 3** | Warm terracotta | `#D4A373` |
| **Primary Text** | Deep navy (on light backgrounds) | `#1A2332` |
| **Light Text** | Pure white (on dark areas) | `#FFFFFF` |
| **Secondary Text** | Soft gray | `#6B7280` |

All text uses solid color fill only — no text effects of any kind.

**Color Intensity:**
- Macaron tones dominate: 70–80% of visual weight
- Soft contrast throughout; no harsh blacks

**Signature Combinations:** Cream/white 3D elements on a blue-to-lavender gradient background, accented with macaron-tone details.

**Avoid:** Neon colors, pure black, high-contrast pairings, realistic textures, dark or gritty aesthetics.

## Layout Grid

```
[ Brand / Logo          15% ]
[ Isometric 3D World    60% ]
[ Event Details / UI    25% ]
```

## Visual Elements

### Typography

- Clean sans-serif; medium weight (500–700)
- Mixed case or playful title case
- Small to medium scale (15–25% of canvas)
- Positioned around the 3D focal point — supportive, not dominant
- Solid color fill only; no text effects whatsoever

### Layout Structure

- Isometric 3D world as the primary focal point (centered or offset)
- Light macaron gradient background
- UI elements anchored to corners (button top-right; info card at bottom)
- Asymmetric, playful composition
- Information integrated within the scene — not isolated in separate zones

### 3D Isometric Elements

- Low-poly modeling (visible facets, simplified geometry)
- Fixed isometric perspective (no vanishing points)
- Fantasy architecture: castles, temples, floating structures
- Crystal and diamond motifs: pyramids, gemstones, geometric forms
- Floating islands and impossible structures
- Airships, hot air balloons, whimsical vehicles
- Cross-section cutaway views
- Miniature model aesthetic
- Steampunk + fantasy fusion details

### Scene Composition

- Central hero structure (large focal building or crystal formation)
- Satellite elements (smaller buildings, characters, props)
- Layered depth: foreground / midground / background
- Atmospheric falloff for distant elements
- Narrative details (miniature characters engaged in activities)

### UI Elements

- Rounded-rectangle cards (countdown timers, statistics)
- Social media icon badges (top-right corner)
- Clean numeric displays
- Subtle shadows
- Minimal line work

---

## Effects & Treatment

- Soft ambient lighting (no harsh shadows)
- Diffuse shading; matte surfaces
- Ambient occlusion for added depth
- Soft gradient backgrounds
- Atmospheric haze on distant elements
- No glow effects; no neon colors
- Clean rendering optimized for social media

---

## Composition Strategy

- Isometric 3D world dominates (50–70% of canvas)
- Text and UI elements support the visual — no competition
- Asymmetric dynamic balance
- Clear focal hierarchy: Hero structure → Detail elements → UI
- Generous sky space at the top (30–40% of upper canvas)
- Foreground elements frame the scene
- Miniature world perspective invites exploration
- Fantasy storytelling delivered through environmental detail

---

## Typography Rules

**Color:** Choose a solid color with high contrast against the macaron gradient background:
- On light gradient backgrounds → Deep navy (`#1A2332`)
- On dark overlay areas → Pure white (`#FFFFFF`)
- Alternative → Soft gray (`#6B7280`)

**Fill Method:** Solid color fill only — single color, no gradients.

**Strictly Prohibited Effects:**
- Outer glow or inner glow
- Drop shadow or text shadow
- Stroke or outline
- Gradient fill on text
- Opacity effects on text
- Any other text special effects

**Guiding Principle:** Typography must remain clean, concise, and highly legible — a supportive element that never overshadows the scene; fully consistent with the soft, macaron-toned aesthetic.
