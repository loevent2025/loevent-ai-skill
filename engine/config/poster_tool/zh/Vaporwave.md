# VAPORWAVE / 蒸汽波风格

## 适用场景
Web3 会议主视觉、黑客马拉松、NFT 发售、加密艺术节、讽刺品牌宣传、怀旧社区活动、反企业运动

## 配色方案
- **主色调：** 柔和的粉色、天蓝色、柔和的紫色
- **背景：** 米白色、柔和的米色或浅色渐变背景
- **点缀色：** 薄荷绿、薰衣草色、桃色、水蓝色
- **饱和度：** 低到中等饱和度，带有褪色、怀旧的质感
- **色彩表现：** 柔和的色调、褪色的美学、复古色彩分级

### 常用配色组合
- 经典蒸汽波：柔和的粉色 + 天蓝色 + 柔和的紫色（在米色背景上）
- 怀旧褪色：薰衣草色 + 薄荷绿 + 桃色（带有复古水洗效果）
- 数字之梦：水蓝色 + 淡粉色（在米白色上，带有复古滤镜）

## 视觉元素
- 复古电脑图形和像素艺术元素
- 古希腊/罗马雕像和半身像
- 早期互联网和 Windows 95/98 UI 元素
- 棕榈树和热带图像
- 棋盘格地板和网格
- 海豚、埃及金字塔和超现实的并置
- 复古日文文本和字符
- 讽刺性地操纵企业标志
- 故障低分辨率图像和抖动
- 复古广告和消费产品

## 文字处理规范

### 颜色选择
- **单色填充：** 文字必须使用单一纯色，无渐变、无纹理、无特效
- **高对比度色彩：** 根据背景选择形成强烈视觉反差的颜色
  - 浅色/米色背景：纯黑 (#000000)、深紫 (#2D1B4E)、深蓝 (#0A1628)
  - 深色/饱和区域：纯白 (#FFFFFF)、荧光粉 (#FF10F0)、电蓝 (#00D4FF)、薄荷绿 (#00FFAA)
- **禁止效果：** 无描边、无阴影、无发光、无3D效果、无渐变填充

### 放置位置策略
- **顶部中央：** 标题文字的首选位置，距顶边缘留出约 5-10% 画面高度
- **画面正中：** 适合单一主标语或活动名称，作为视觉焦点
- **动态角度：** 15°-30° 倾斜放置，增加复古动感
- **对称位置：** 
  - 左上/右上对角呼应（适合主副标题）
  - 左中/右中水平对称（适合日期与地点信息）
  - 四角分布（适合装饰性短语或标签）

## 效果处理
- VHS 跟踪失真和伪影
- 低分辨率像素化和抖动
- 柔和的高斯模糊，营造梦幻效果
- 色彩通道分离和故障
- 复古照片颗粒和褪色
- 降低不透明度的扫描线叠加
- 双色调和三色调处理
- 屏门效应和隔行扫描
- 带有暖色调的柔和晕影
- JPEG 压缩伪影

## 构图策略
- 拼贴式分层，带有不协调的元素
- 居中对称构图
- 复古电脑窗口框架和 UI 边框
- 古典雕塑与数字元素的并置
- 重复图案和镶嵌
- 带有复古电脑字体的文本叠加
- 带有超现实放置的非对称平衡
-  reminiscent of early web design 的网格布局
- 带有极简复古美学的留白

## 情绪氛围
讽刺的怀旧、对消费主义的批判、梦幻般的超脱。整体氛围是忧郁而又俏皮的——唤起了早期互联网的乐观主义与后现代的讽刺，通过数字图像的超现实、美丽衰败，歌颂过时技术并质疑资本主义美学。

## 示例Prompt
将图形渲染为平面图像、表面图形设计、图案设计。画面铺满至边缘，，无外部边框，无画布/纸张纹理/模型。内容充满整个画布，边缘到边缘，边缘绝对没有边距或空白。图像中不得出现任何可见的文本、字母、数字或颜色代码。采用蒸汽波美学，营造怀旧讽刺的氛围。背景充满整个画面，柔和的米色到浅米色渐变，并叠加细微的复古照片颗粒，延伸至所有边缘。大型古典希腊大理石半身像占据中心，以柔和的粉色渲染，带有梦幻般的高斯模糊，占据大部分垂直空间。半身像后面，天蓝色的几何网格地板，带有棋盘格图案，从底部边缘延伸到淡紫色薄雾中的地平线，覆盖整个宽度。柔和薰衣草色的复古电脑窗口框架元素放置在边缘，与构图重叠，带有透明填充。薄荷绿色的棕榈树剪影从底部角落向上延伸，边缘柔和，饱和度低，伸向顶部。散布着水蓝色和桃色的像素艺术元素——海豚和几何形状——分布在整个构图中，填充负空间。VHS 跟踪失真在整个宽度上水平扫过，色彩通道分离产生微妙的 RGB 偏移。复古扫描线叠加覆盖整个画面，不透明度适中。下部出现波浪形失真效果，跨越整个宽度，暗示模拟视频降级。渐变过渡处的抖动效果增加了复古数字质感。所有元素都出血到边缘，没有空白边框或边距——完全覆盖边缘到边缘。整体氛围是讽刺的怀旧和梦幻——纯粹的蒸汽波美学通过美丽的数字衰败批判消费主义，非常适合 Web3 活动，以超现实的优雅庆祝互联网怀旧和反企业亚文化。关键：仅限纯视觉图形，生成的图像中绝不能出现任何文本或字母数字字符。
