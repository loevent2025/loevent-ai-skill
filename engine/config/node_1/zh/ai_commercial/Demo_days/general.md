# 商业化AI 路演日知识库

## 一、场景定义与特点

### 场景概述
路演日（Demo Day）是商业化AI行业中项目展示与资本对接的核心场景，为AI创业公司提供展示机会，为投资人提供项目发现渠道，同时也是加速器、孵化器和企业创新部门的重要活动形式。

### 核心特征
| 特征 | 描述 |
|------|------|
| 规模 | 50-500人，以投资人、企业和项目方为主 |
| 周期 | 半天至1天，紧凑高效 |
| 形式 | 项目Pitch+评委点评+一对一对接+交流酒会 |
| 定位 | 融资对接、项目曝光、投资发现、创新展示 |

### 典型活动形式
- AI加速器毕业路演（Y Combinator AI Day、a16z AI Demo Day）
- 企业创新Demo Day（Corporate Innovation Showcase）
- 垂直领域AI路演（AI Agent Demo Day、AI Coding Startup Pitch）
- 投资机构项目日（VC Portfolio AI Day）
- 黑客松决赛路演

## 五、关键成功要素

### 项目筛选
- **技术壁垒**：核心技术、模型能力、数据优势
- **团队背景**：技术团队、行业经验、执行能力
- **市场潜力**：市场规模、增长趋势、竞争格局
- **商业验证**：收入/用户数据、客户案例、PMF信号

### Pitch培训
- **内容结构**：问题-方案-市场-团队-愿景
- **数据呈现**：关键指标、增长曲线、对标数据
- **技术Demo**：产品演示、技术能力展示
- **时间控制**：3-8分钟Pitch、Q&A准备

### 投资人邀请
- **精准匹配**：根据项目阶段、赛道匹配投资人
- **决策层级**：合伙人、投资总监优先
- **主动基金**：有AI投资mandate的基金
- **战略投资**：相关企业CVC、战略投资者

### 对接机制
- **预约系统**：投资人-项目一对一预约
- **信息同步**：Deck提前共享、数据房准备
- **对接空间**：独立会议室、安静对话区
- **后续支持**：联系方式交换、跟进协调

## 六、注意事项

### 项目保护
- 签署保密协议、竞业信息保护
- 控制材料传播范围
- 尊重项目融资进度和信息披露边界

### 投资人体验
- 避免项目质量参差不齐
- 提供充足的一对一对接时间
- 高效的会议安排和时间管理

### 信息真实性
- 核实项目数据准确性
- 避免过度包装和夸大
- 建立反馈和信用记录

### 利益冲突
- 透明的项目筛选标准
- 关联方披露
- 评委回避机制
