- 品鉴环节（可选）

## 执行要点

### 优先级排序
1. **场地确定** - 活动前3-4周（热门场地需更早）
2. **联合主办方** - 活动前2-3周
3. **酒水餐饮** - 活动前1-2周
4. **招募启动** - 活动前10-14天
5. **传播配合** - 活动前1周

### 注意事项

| 要点 | 说明 |
|------|------|
| 氛围优先 | 合作方露出需融入氛围，避免过度商业化 |
| 体验为王 | 酒水质量>数量，场地调性>面积 |
| 轻量合作 | 酒会合作以资源置换为主，避免复杂权益 |
| 社交友好 | 任何合作环节不应打断自然社交流动 |
| 拍照传播 | 场地和布置需考虑出片效果，便于UGC |

### 常见问题预案

| 问题 | 预案 |
|-----|------|
| 场地临时变动 | 备选场地清单、室内备用方案 |
| 酒水不足 | 预留20%余量、临时采购渠道 |
| 人数超预期 | 报名上限、场地弹性区域 |
| 天气影响（户外） | 室内备用、提前通知机制 |
