# 7. 赛博朋克都市风格 / CYBERPUNK METROPOLIS

## 适用场景
电音节/DJ活动推广、科技产品发布会、游戏宣传海报、未来主题派对、周年庆典活动、夜店开业宣传、电竞赛事、科幻电影宣传、元宇宙/Web3活动、品牌科技感形象塑造

## 配色方案
- **主色调：** 深邃霓虹色系 - 赛博蓝 (#00D4FF)、电光紫 (#9B30FF)、霓虹粉 (#FF00FF)、科技青 (#00FFCC)
- **背景色：** 深蓝紫渐变，从深邃藏蓝 (#0A0A1A) 到午夜紫 (#1A1A3A) 再到深空蓝 (#0D1B2A) 的微妙过渡
- **光效色：** 青蓝辉光 (#00FFFF)、粉紫高光 (#FF66FF)、白色聚光 (#FFFFFF)，透明度中高，边缘柔和弥散
- **线条色：** 青蓝线稿 (#4A90D9)、淡紫描边 (#7B68EE)，用于建筑/机械结构勾勒
- **饱和度：** 背景低饱和 (10-20%)，光效高饱和 (80-100%)，形成强烈明暗对比
- **色彩行为：** 暗色背景与霓虹光效形成戏剧性反差，光源集中在画面焦点区域

### 常用配色组合
- 午夜都市：深藏蓝 (#0A0A1A) → 午夜紫 (#1A1A3A) + 赛博蓝光 (#00D4FF)
- 霓虹峡谷：深空蓝 (#0D1B2A) → 暗紫 (#2D1B4E) + 粉紫辉光 (#FF00FF)
- 数字黎明：深青蓝 (#0A1628) → 靛蓝 (#1E3A5F) + 青白光效 (#00FFCC)

## 视觉元素
- 未来都市天际线——高耸密集的赛博朋克风格建筑群
- 垂直光束/数据流——从建筑或天空倾泻而下的霓虹光柱
- 全息投影/发光Logo——悬浮在建筑上的几何图形或品牌标识
- 人物剪影——通常位于前景底部，面向城市/光源，营造孤独史诗感
- 霓虹招牌/LED屏幕——点缀在建筑侧面，增加都市细节
- 线条插画元素——复杂的机械/建筑结构线稿（蓝图风格）
- 星空/粒子点缀——增加深邃空间感
- 光晕/镜头光斑——聚焦视觉中心

## 效果处理
- 背景使用细密水平或垂直扫描线纹理叠加，透明度5-15%
- 主体光源使用大范围辉光弥散 (30-80px)，颜色为青蓝或粉紫
- 建筑/结构使用渐变填充，从暗部到亮部过渡
- 霓虹元素添加多层发光：内层亮白，外层带色
- 线条插画保持统一描边粗细 (1-3px)，颜色为青蓝色系
- 使用"叠加"或"滤色"混合模式增强光效层次
- 整体色调统一——冷青蓝调或暖粉紫调，避免混杂
- 可选添加轻微噪点纹理 (3-8%) 增加质感

## 构图策略
- 强透视纵深构图，引导视线向画面中心/远处汇聚
- 中央焦点法则——主体光源或建筑位于画面中央
- 前景剪影（人物/建筑边缘）框架画面，增加层次
- 两侧建筑群形成"峡谷"式包围感
- 文字信息分区明确：顶部品牌/标题，底部时间/详情
- 垂直构图优先，强调建筑高度和都市压迫感
- 留出"呼吸"空间——天空或光源区域保持简洁
- 所有元素服务于"未来、宏大、科技"的整体氛围

## 文字设置
- **颜色选择：**
  - 深色背景上使用亮色文字形成对比
  - 推荐：赛博蓝 (#00D4FF)、纯白 (#FFFFFF)、科技青 (#00FFCC)
  - 次要信息可用：浅蓝灰 (#A0B4C8)、淡紫 (#B8A0D0) 用于底部详情文字
- **填充方式：** 纯色填充，单一颜色，无渐变
- **禁止效果：**
  - 无渐变填充
  - 无阴影/投影
  - 无外发光/内发光
  - 无描边
  - 无模糊效果
  - 无任何文字特效
- **字体风格：** 科技感无衬线体或几何风格字体，字重从Regular到Bold，棱角分明，强调未来机械感；中文可使用黑体或思源黑体
- **整体原则：** 文字保持干净、锐利、高可读性，通过字体选择和亮色纯色与深色背景形成清晰的视觉层次，科技感与可读性并重

## 情绪氛围
未来、科技、宏大、孤独、沉浸。整体传递一种"赛博朋克都市"的史诗气质，深邃背景带来神秘感，霓虹光效注入活力与希望。适合追求科技感、未来感、沉浸式体验的品牌活动。唤起对未来世界的想象、对科技力量的敬畏、以及都市夜晚的迷幻魅力。
