# Web3 Demo Day - 教育类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- 训练营毕业展示、学员项目路演、课程成果展
- 高校项目展示日、学生作品展示、实训成果汇报
- 认证考核展示、技能验收展示

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 学员展示 + 导师点评 + 颁奖典礼 + 社群连接 |
| 时长 | 半天至一天，每项目 3-8 分钟展示 |
| 参与方 | 学员、导师、培训机构、企业 HR |
| 核心目标 | 学习成果展示、技能认证、就业对接、社群归属 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 参展学员 | 成果展示、获得认可、就业机会 |
| P1 | 企业 HR | 人才发现、技能评估、招聘对接 |
| P2 | 导师/培训方 | 教学成果、学员反馈、品牌展示 |
| P3 | 潜在学员 | 学习效果、课程价值、学习动力 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 训练营毕业典礼 | 热情活力型 (B) | 庆祝氛围、成长认可 |
| 高校项目展示 | 正式专业型 (A) | 学术价值、专业评审 |
| 就业对接展示 | 福利导向型 (D) | 就业机会、企业对接 |
| 技能认证展示 | 悬念引导型 (C) | 能力验证、成果检验 |

---

## 📝 内容必含元素

### 1. 成长价值（至少2处）
- "[数量] 位学员展示学习成果"
- "见证从零到一的蜕变"

### 2. 认可体系（至少1处）
- "获得官方结业证书"
- "优秀学员获得荣誉认证"

### 3. 对接机会（至少1处）
- "[数量]+ 家企业现场招聘"
- "优秀作品推荐就业/实习"

### 4. 说明提示（建议包含）
- "本活动为教育成果展示，旨在人才培养"
