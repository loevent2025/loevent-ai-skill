# AI Developer Meetup — General Event Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event is a comprehensive AI developer meetup covering multiple topics:
- Comprehensive AI developer meetups, City-level monthly AI gatherings
- Multi-topic AI exchange sessions, AI developer community days

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Multi-topic talks + Panel discussions + Open networking + Social sessions |
| Duration | 2–4 hours |
| Participants | Various AI developers, Product managers, Learners, Enthusiasts |
| Core Objective | Broad connections, Information exchange, Community building, Ecosystem engagement |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Interests |
|----------|---------------|---------------|
| P0 | AI Application Developers | Technical exchange, Industry updates, Network building |
| P1 | AI Product Managers | Product thinking, Technical understanding, Partnership opportunities |
| P2 | AI Learners | Learning resources, Industry awareness, Mentor connections |
| P3 | AI Founders / Investors | Deal sourcing, Talent connections, Market insights |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| City Monthly Meetup | Energetic & Engaging (B) | Broad participation, Regular connections |
| Themed Exchange Session | Formal & Professional (A) | Content depth, Professional exchange |
| Open Meetup | Energetic & Engaging (B) | Relaxed atmosphere, Open networking |
| Resource Matching Session | Value-Driven (D) | Resource connections, Partnership opportunities |

## 📝 Required Content Elements

### 1. Multi-Dimensional Value (at least 2 instances)
- "Technical talks + Industry exchange + Social networking"
- "[Number] topic sessions covering multiple focus areas"

### 2. Participation Value (at least 1 instance)

- "Stay up to date on the latest industry developments"

### 3. Atmosphere Setting (at least 1 instance)
- "Relaxed atmosphere — all backgrounds welcome"
- "Plenty of time for coffee-break conversations"

- "💡 This is a community exchange event. AI practitioners of all backgrounds are welcome."
