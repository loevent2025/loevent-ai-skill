# AI Technical Workshop – Education Event Knowledge Base

## 📌 Module Activation Criteria

Apply the rules in this module when the event theme involves the following keywords:
- AI introductory workshops, zero-experience AI development classes
- Prompt Engineering basics, AI tool usage training
- Student AI hands-on classes, AI skill discovery workshops

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Foundational instruction + guided hands-on practice + mentor support + project sharing |
| Duration | 3–4 hours |
| Participants | AI beginners, students, career changers, training mentors |
| Core Objective | Skill discovery, hands-on experience, building learning confidence, entry-level guidance |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Interests |
|----------|---------------|---------------|
| P0 | AI Beginners | First-time experience, building confidence, hands-on accomplishment |
| P1 | Students | Skill expansion, résumé boost, career direction |
| P2 | Career Changers | Skill validation, transition feasibility, learning path |
| P3 | Business-Side Professionals | AI tool adoption, productivity skills, practical techniques |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Zero-Experience Intro | Energetic & Engaging (B) | Encouraging atmosphere, sense of accomplishment |
| Student-Focused Session | Energetic & Engaging (B) | Learning enthusiasm, career inspiration |
| Prompt Engineering Intro | Benefits-oriented (D) | Immediately applicable, template resources |
| Skill Certification Course | Formal & Professional (A) | Structured learning, skill validation |

---

## 📝 Required Content Elements

### 1. Beginner-Friendly (at least 2 instances)
- "Zero experience welcome — no coding required"
- "Start from scratch and build your first AI application"

### 2. Achievement-Oriented (at least 1 instance)
- "Walk away from the workshop with a demo-ready project"
- "Build it on-site — see results in real time"

### 3. Learning Support (at least 1 instance)
- "Mentors guide you throughout — no one gets left behind"
- "Join the learning community for ongoing Q&A"

### 4. Disclosure Note (recommended)
- "💡 This is an introductory education event focused on AI skill discovery and building interest"

## ⚠️ Prohibited Actions

- ❌ Creating learning pressure for beginners
- ❌ Promising that completing the workshop leads to an AI job
- ❌ Belittling beginners' ability to learn
- ❌ Aggressively upselling paid follow-up courses
