# Web3 Developer Meetup — Community Event Knowledge Base

## 📌 Module Activation Criteria

Apply the rules in this module when the event theme involves the following keywords:
- Community developer meetup, Builder Hangout, ecosystem developer night
- DAO developer exchange, open-source contributor meetup, community Coding Night
- Builder After Party, developer social gathering

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Lightning talks + open mingling + social games + party atmosphere |
| Duration | 2–4 hours, relaxed atmosphere |
| Participants | Community Builders, ecosystem developers, tech enthusiasts |
| Core Objective | Community bonding, Builder socializing, casual exchange, fostering belonging |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Focus Areas |
|----------|---------------|-------------|
| P0 | Community Builders | Connecting with like-minded people, project chats, community belonging |
| P1 | Ecosystem Developers | Ecosystem updates, collaboration opportunities, social downtime |
| P2 | Tech Enthusiasts | Joining the community, meeting new friends, industry awareness |
| P3 | Community Newcomers | Breaking the ice, meeting OGs, finding their tribe |
