# AI Demo Day — Investment Event Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- AI incubator Demo Day, AI accelerator graduation showcase
- AI project investment pitch, AI Startup Pitch Day
- AI seed-round pitch, AI project fundraising showcase

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Project pitch + Investor judging panel + 1-on-1 meetings + Networking dinner |
| Duration | 4–6 hours |
| Participants | AI startup teams, investment firms, incubators, financial advisors (FAs) |
| Core Objective | Fundraising matchmaking, investor exposure, project screening, post-event follow-up |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Concerns |
|----------|---------------|--------------|
| P0 | AI Startup Teams | Fundraising opportunities, investor visibility, feedback |
| P1 | Investment Firms | Deal screening, early-stage discovery, sector coverage |
| P2 | Incubators / Accelerators | Portfolio showcase, brand visibility, ecosystem building |
| P3 | Industry Observers | Project updates, sector trends, investment signals |

- ❌ Overstating project capabilities or market outlook
