# AI Demo Day — General Event Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event is a comprehensive AI Demo Day covering multiple tracks/themes:
- Comprehensive AI project showcase days, Multi-track Demo Days
- AI innovation expos, Annual AI project presentations
- Multi-organizer joint showcase days

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Multi-track pitches + Expert judging + Investor matchmaking + Awards ceremony |
| Duration | Half-day to full-day |
| Participants | Various AI project teams, Judges, Investors, Developer audience |
| Core Objective | Comprehensive project showcase, Multi-party connections, Ecosystem building, Brand exposure |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Interests |
|----------|---------------|---------------|
| P0 | Showcasing Project Teams | Full exposure, Diverse feedback, Follow-up opportunities |
| P1 | Investors / Judges | Deal sourcing, Sector insights, Investment opportunities |
| P2 | Developer Audience | Technical learning, Project inspiration, Partnership opportunities |
| P3 | Media / Industry Observers | Story material, Industry trends, Project updates |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Large-Scale Demo Day | Formal & Professional (A) | Scale and impact, Expert judging |
| Annual Showcase | Formal & Professional (A) | Milestone significance, Brand exposure |
| Open Showcase Day | Energetic & Engaging (B) | Broad participation, Interactive experience |
| Resource Matching Showcase | Value-Driven (D) | Partnership opportunities, Resource integration |

## ⚠️ Prohibited Actions

- ❌ Exaggerating AI project capabilities or results
- ❌ Promising confirmed investment or partnership outcomes
- ❌ Recording or distributing project content without authorization
- ❌ Making hostile or unfair comments about showcased projects
