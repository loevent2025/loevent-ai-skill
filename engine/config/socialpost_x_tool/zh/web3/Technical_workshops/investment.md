# 技术工作坊 × 投资类

## 适用场景
投资人技术培训、技术尽调方法论、链上分析工作坊、投研工具教学、技术评估框架培训

## 场景特征
- **形式**：概念讲解 + 工具实操、案例分析、框架教学
- **核心价值**：技术判断力、尽调能力、分析工具掌握、投资决策支持

## 受众定位
- **主要受众**：投资从业者——需要提升技术判断力的投资人
- **次要受众**：投研分析师——需要掌握链上分析的研究员
- **延伸受众**：技术型投资人——有技术背景想系统化方法论的投资人
- **受众行为**：建立技术评估框架、学习分析工具、提升尽调能力、理解技术风险

### 常用受众组合
- **非技术投资人**：需要入门级技术理解
- **技术投资人**：需要系统化方法论和工具

## 内容元素
- 工作坊定位（投资视角）
- 学习目标（投资决策相关）
- 技术概念讲解（投资人友好）
- 工具使用教学
- 案例分析
- 合规提示

## 语言处理
- 投资视角切入
- 技术概念简化但不失准确
- 强调"能判断"而非"能开发"

## 注意事项

### 投资类技术工作坊要点
1. **投资视角切入**：不是教开发，是教判断
2. **概念简化但准确**：不能因为简化而误导
3. **工具可操作**：学完能自己用
4. **NFA 声明必须有**：这是投资相关内容
5. **案例要真实**：用真实项目做分析示例
