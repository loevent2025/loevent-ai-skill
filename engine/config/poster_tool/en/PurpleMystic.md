# SOLANA GRADIENT / NEON CYBERPUNK Style Guide

## Use Cases
Developer events, NFT ecosystem promotions, community livestreams, Web3 social events

## Signature Color System
- **Primary tones:** Solana violet (#9945FF), bright cyan-green (#14F195)
- **Background colors:** Deep purple-black (#0F0820, #1A0B2E)
- **Typography:** Pure white (#FFFFFF), cyan-green accents
- **Gradient direction:** Deep purple → Solana violet → cyan-green
- **Glow colors:** Bright violet (#BB86FC), electric cyan (#00FFF0)
- **Characteristics:** Solana brand purple, cyan-green neon, gradient light effects, cyberpunk space aesthetic

## Visual Elements

### Typography System
- **Fonts:** Extra bold condensed sans-serif (Druk Wide, Inter Black, Space Grotesk Bold)
- **Size:** Oversized headlines at 100–180pt, occupying 40–50% of the canvas as the primary anchor
- **Color:** Main headlines in pure white (#FFFFFF), emphasis text in cyan-green (#14F195), solid color fills only
- **Hierarchy:** Clear multi-level text layout, uppercase preferred
- **Alignment:** Left-aligned or centered, tight letter-spacing
- **Scale dominance:** Typography serves as the primary visual anchor element

### Neon Light Effects
- **Glow blocks:** Violet (#9945FF) glowing rectangular labels/buttons
- **Light rays:** Purple-to-cyan gradient abstract flowing light streams
- **Halos:** Soft outer glow around key elements (20–40px radius)
- **Gradient overlays:** Smooth purple-to-cyan transitions between elements
- **Circular icons:** Dark circles with question marks or symbols, violet edge glow
- **Floating accents:** Drifting across multiple transparency layers

### 3D / Background Elements
- **Cityscape:** Low-opacity 3D urban architecture (20–30% opacity), purple-toned
- **Silhouettes:** Dark-toned human silhouettes
- **Abstract shapes:** Flowing organic gradients, geometric light patterns
- **Floating geometry:** Suspended at varying spatial depths

## Visual Effects

### Neon Gradients
- **Direction:** Diagonal, radial, or vertical gradients
- **Color stops:** Deep purple → bright violet → cyan-green, 4–6 stops
- **Glow intensity:** Cyan-green (#14F195) at high luminosity, violet at medium
- **Bloom effect:** 30–50px Gaussian blur on bright elements

### Light & Shadow Treatment
- **Outer glow:** 15–30px soft glow on all neon elements to separate layers
- **Inner glow:** Subtle internal luminance on shapes and text
- **Atmospheric haze:** Purple mist/smoke effect (10–20% opacity)
- **Contrast:** Extreme contrast between dark backgrounds and bright neon colors
- **Shadows:** Used to separate layers and establish depth

### Multi-Layer Depth Architecture
- **Layer system:** 4–5 independent depth layers with clear foreground–midground–background separation
- **Opacity hierarchy:** Varying opacity levels (20–100%) to create depth tiers
- **Z-axis depth:** Systematic overlapping for spatial depth perception
- **Floating elements:** Decorative accents suspended between layers
- **Light effect layer:** Dedicated glow layer positioned behind primary elements

## Composition Strategy

### Oversized Element Approach
- **Primary element scale:** Large graphics occupying 40–50% of the canvas
- **Design philosophy:** Fewer elements, maximum visual impact
- **Bold typography:** Text as the dominant visual anchor
- **Negative space:** Generous whitespace between primary elements
- **Scale hierarchy:** Strong size contrast between primary and secondary elements

### Layering Technique
- **Foreground (100% opacity):** Primary text and main neon elements
- **Midground (50–80% opacity):** Supporting graphics, 3D elements, icons
- **Background (20–40% opacity):** Cityscape, abstract shapes, atmospheric effects
- **Light layer:** Glow effects placed behind thematic elements
- **Depth separation:** Opacity, shadows, and outer glow used to distinguish layers

### Layout Structure
- **Typography-dominant:** 40–50% allocated to bold type as the primary element
- **Visual elements:** 30–40% for 3D scenes, icons, or abstract graphics
- **Information zone:** 10–20% for event details and dates
- **Asymmetric balance:** Dynamic diagonal or dispersed layouts
- **Breathing room:** Elements maintain comfortable spacing without touching edges

### Scale Hierarchy System
- **Primary element:** Dominant scale (40–50% canvas), main focal point
- **Secondary element:** Medium scale (15–25% canvas), supporting role
- **Tertiary element:** Fine detail (5–10% canvas), text and contextual info
- **Strong contrast:** Scale as the primary tool for visual hierarchy
- **Anchoring:** Large elements anchor the composition; small elements provide context

### Spatial Composition
- **Whitespace:** Elements do not touch edges — consistent comfortable margins
- **Systematic overlap:** Overlapping creates natural depth perception
- **Primary anchor:** Large elements anchor the overall composition
- **Contextual elements:** Small elements provide information without distraction
- **Z-axis layering:** Elements positioned at different spatial depths
- **Spatial depth architecture:** 4–5 distinct depth planes working in concert

## Typography Settings
- **Color selection:** Solid colors with high contrast against the deep purple-black background
  - Main headlines: Pure white (#FFFFFF)
  - Emphasis text: Cyan-green (#14F195)
  - Alternatives: Bright violet (#BB86FC), electric cyan (#00FFF0)
- **Fill method:** Solid color fills only — single color per element, no gradients
- **Prohibited effects:**
  - No outer glow / inner glow
  - No strokes or outlines
  - No drop shadows
  - No neon light effects on text
  - No gradient fills
  - No text effects of any kind
- **General principle:** Typography must remain clean, simple, and highly legible. Visual impact is achieved through oversized scale and heavy font weight — contrasting with the neon glow effects in the background, not replicating them.

## Mood & Atmosphere
Futuristic spatial atmosphere, mysterious cyberpunk energy, electrifying community presence, technological innovation, dynamic motion, neon nightlife aesthetics, Web3 cultural identity, immersive social experience, layered dimensional depth

## Artistic References
Cyberpunk 2077 UI, Blade Runner neon aesthetics, electronic music festival visuals, synthwave art style, holographic interfaces, space-themed game graphics, layered depth design

## Technical Specifications
- **Resolution:** 4K (3840×2160) or 1080×1920 for social media, 300 DPI
- **Color space:** RGB, wide gamut for vivid neon reproduction
- **Effects:** Layer blending modes (Screen, Add for glow), multi-layer stacking
- **Software:** Photoshop, After Effects, Blender for 3D elements
- **Format:** PNG for transparency, MP4 for animated versions
- **Layers:** Minimum 4–5 independent depth layers
