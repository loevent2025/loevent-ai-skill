# Web3 Developer Meetup – Education Event Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- Beginner developer meetup, introductory exchange session, learner Meetup
- University developer meetup, student tech exchange, campus Web3 night
- Career-switching developer exchange, career development meetup, job search sharing session

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Experience sharing + Learning exchange + Q&A interaction + Social connection |
| Duration | 2–3 hours, relaxed and friendly |
| Participants | Beginner developers, students, career switchers, senior developer mentors |
| Core Objective | Learning exchange, Q&A support, peer connection, growth inspiration |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Concerns |
|----------|---------------|--------------|
| P0 | Beginner Developers | Learning path, getting-started advice, peer exchange |
| P1 | University Students | Industry orientation, internship opportunities, career planning |
| P2 | Career-Switching Developers | Transition experience, study methods, job search advice |
| P3 | Senior Developers | Giving back to the community, talent discovery, mentor role |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Beginner-Friendly Meetup | Energetic & Engaging (B) | Relaxed atmosphere, encourage participation |
| University Developer Night | Energetic & Engaging (B) | Campus atmosphere, youthful energy |
| Career Development Exchange | Suspense-Driven (C) | Career questions, experience sharing |
| Learning Resource Sharing | Benefits-Oriented (D) | Learning resources, job opportunities |

---

## 📝 Required Content Elements

### 1. Friendly Atmosphere (at least 2 instances)
- "Beginner-friendly, zero-pressure exchange"
- "There are no stupid questions"

### 2. Learning Value (at least 1 instance)
- "Senior developers sharing their learning experience"
- "Get answers to your getting-started questions"

### 3. Peer Connection (at least 1 instance)
- "Meet peers at the same stage of their journey"
- "Find study buddies to learn alongside"

### 4. Disclaimer Note (recommended)
- "This is a learning exchange event — developers of all levels are welcome"

## ⚠️ Prohibited Actions

- ❌ Showing impatience toward beginner questions
- ❌ Showing off or creating knowledge gaps
- ❌ Over-promoting courses or training programs
- ❌ Undermining the beginner-friendly atmosphere
