# Web3 商务会议 - 教育类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- 企业培训研讨、内训方案会议、人才战略会
- 高校合作洽谈、产学研对接会、课程研发会
- 认证体系研讨、培训机构合作、教育资源对接

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 闭门研讨 + 方案评审 + 合作洽谈 + 资源对接 |
| 内容深度 | 培训体系设计、课程开发、认证标准、人才战略 |
| 参与方 | 企业 HR/培训负责人、高校代表、教育机构、项目方 |
| 核心目标 | 培训合作达成、课程体系落地、人才通道建立 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 企业培训负责人 | 团队技能提升、内训方案、成本效益 |
| P1 | 高校/研究机构 | 课程合作、实习基地、产学研项目 |
| P2 | 教育培训机构 | 课程资源、认证合作、招生渠道 |
| P3 | 项目方 HR | 人才招聘、雇主品牌、校招合作 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 企业培训研讨会 | 正式专业型 (A) | 方案评估、效果论证 |
| 高校合作洽谈 | 正式专业型 (A) | 学术合作、资源互换 |
| 人才战略会议 | 悬念引导型 (C) | 招聘痛点、解决方案 |
| 培训资源对接 | 福利导向型 (D) | 合作机会、资源获取 |

## 📝 内容必含元素

### 1. 专业定位（至少1处）
- "面向企业培训决策者的闭门研讨"
- "高校与产业界的深度对话"

### 2. 议题价值（至少2处）
- "探讨 Web3 人才培养体系建设"
- "分享企业内训最佳实践"

### 3. 合作价值（至少1处）
- "促成培训/课程合作意向"
- "建立人才输送通道"

### 4. 说明提示（建议包含）
- "本活动为教育合作交流，旨在推动人才培养"

## ❓ 常见问答模板

| 问题 | 回答 |
|------|------|
| 参与门槛是什么？ | 企业培训/HR 负责人或同等决策级别 |
| 培训机构可以参加吗？ | 本场为企业方专场，机构有单独对接会 |
| 会后有资料吗？ | 会分享会议纪要和方案模板 |
| 可以带团队成员吗？ | 原则上每公司限1人 |
