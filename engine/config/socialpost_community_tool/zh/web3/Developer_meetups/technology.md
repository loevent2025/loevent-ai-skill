# Web3 开发者聚会 - 技术类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- 技术 Meetup、开发者技术分享会、协议技术交流
- 代码分享会、技术沙龙、架构讨论会
- Bug Bash、Code Review 聚会、技术问答夜

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 技术分享 + 现场讨论 + 自由交流 + 小食社交 |
| 时长 | 2-3 小时，轻松氛围 |
| 参与方 | 开发者、技术爱好者、项目方技术团队 |
| 核心目标 | 技术交流、经验分享、开发者连接、问题解决 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 活跃开发者 | 技术交流、同行连接、经验获取 |
| P1 | 项目方开发者 | 技术推广、人才发现、社区建设 |
| P2 | 技术学习者 | 学习机会、行业了解、人脉拓展 |
| P3 | 技术管理者 | 团队招聘、技术趋势、合作机会 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 协议官方技术 Meetup | 正式专业型 (A) | 技术深度、官方权威 |
| 社区开发者聚会 | 热情活力型 (B) | 轻松交流、社区氛围 |
| 技术问题讨论会 | 悬念引导型 (C) | 问题解决、技术探讨 |
| 开发者福利夜 | 福利导向型 (D) | 免费参与、礼品抽奖 |

---

## 📝 内容必含元素

### 1. 分享价值（至少2处）
- "[数量] 位开发者分享实战经验"
- "深入讨论 [技术话题]"

### 2. 交流机会（至少1处）
- "现场自由交流环节"
- "认识同城 Web3 开发者"

### 3. 轻松氛围（至少1处）
- "小食饮品供应"
- "轻松聊技术，不卷不累"

### 4. 合规提示（必须包含）
- "⚠️ 技术交流活动，不涉及投资建议（NFA）"
