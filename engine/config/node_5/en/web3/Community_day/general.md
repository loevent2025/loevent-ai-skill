# Web3 Industry — Builder Coffee Co-Creation Meetup: Partnership Reference Guide

> Applicable to: On-Chain Builder Coffee Day, Ecosystem Developer Café Meetup, Web3 Co-Working Day, and similar events

---

## Event Profile

| Attribute | Description |
|-----------|-------------|
| Scale | 80-120 attendees |
| Duration | 8-10 hours (full-day format with multiple time blocks) |
| Format | Private café booking + open building + project demos + casual networking |
| Core Value | Ecosystem engagement, builder connections, project incubation, community belonging |
| Budget Level | Low to medium budget, primarily allocated to venue and beverages |

## Execution Priorities

### Timeline
1. **Ecosystem Partner Confirmed** — 3-4 weeks before (confirm host chain/L2)
2. **Venue Secured** — 2-3 weeks before
3. **Ecosystem Partnerships Finalized** — 2 weeks before
4. **Community Promotion Launch** — 10-14 days before

### Key Considerations
- Full-day events require venue support for extended bookings; confirm operating hours in advance
- Ecosystem partners can be scheduled across different time slots for Office Hours/Q&A
- On-chain elements (POAPs, etc.) add a sense of ceremony but should not be mandatory
- Timing around ecosystem milestones (pre/post hackathon, mainnet launches) maximizes impact
- Builder community building is the long-term value — continue operations after the event
