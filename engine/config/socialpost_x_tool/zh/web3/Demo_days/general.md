# DEMO DAYS × GENERAL / 路演日 × 通用类

## 适用场景
项目路演日, 宣讲活动, 展示日, 加速器路演日, 生态项目展示, 批次毕业活动, 投资者展示会

## 场景特征
- **形式**：项目路演、评委点评、Q&A、Networking
- **核心价值**：项目曝光、获得反馈、投资人对接、行业认可

## 受众定位
- **主要受众:** 项目方 / Project Teams - 展示项目、获取反馈、寻求机会
- **次要受众:** 投资人 / Investors - 发现项目、评估机会
- **第三受众:** 行业从业者 / Industry Practitioners - 了解新项目、寻找合作
- **扩展受众:** 媒体与分析师 / Media & Analysts - 报道新项目
- **受众行为:** 展示成果、获取反馈、建立连接、发现机会

### 常用受众组合
- **投资者导向:** 项目方 + 投资人，适合融资导向 Demo Day
- **生态导向:** 项目方 + 从业者，适合生态展示型

## 内容元素
- Demo Day 主题与定位
- 路演项目介绍/数量
- 评委/嘉宾阵容
- 路演形式说明（时长、Q&A）
- Networking 环节
- 后续对接机制

## 语言处理
- 项目导向，突出展示机会
- 评委阵容是重要卖点
- 强调反馈价值
- 对接机会说清
- 投资相关需 NFA 声明

## 结构策略
- 开场信息块占比 15%：名称 + 时间地点 + 规模
- 项目说明块占比 30%：路演项目 + 赛道分布
- 评委阵容块占比 25%：评委背景 + 点评形式
- 参与价值块占比 20%：适合谁 + 能获得什么
- 行动号召块占比 10%
- Emoji 密度控制：总量 8-12 个

## 情绪氛围
聚焦亮点，机会驱动。“展示你的成果”的活力。支持建设者，支持反馈。

## 风格变体

### 变体A：综合展示型 / 综合展示
- **适用：** 多赛道综合 Demo Day
- **开场格式：** 🎯 {Demo Day名} | {数量}个项目集中亮相
- **语气：** 全面、机会导向

### 变体B：孵化毕业型 / 批次毕业
- **适用：** 孵化器/加速器 Demo Day
- **开场格式：** 🎓 {孵化器名} Demo Day | 第{X}期项目毕业路演
- **语气：** 成果展示、阶段总结

### 变体C：生态展示型 / 生态展示
- **适用：** 公链/协议生态项目展示
- **开场格式：** ⚡ {生态名} Project Showcase | 生态新星集中亮相
- **语气：** 生态导向、发现新项目

## 注意事项

### Demo Day 文案要点
1. **项目是核心**：说清有多少项目、什么赛道
2. **评委阵容重要**：决定反馈质量和对接价值
3. **路演形式说清**：时长、Q&A、点评方式
4. **双边价值**：对项目方和观众都要有吸引力
5. **Networking 环节**：Demo Day 的隐性价值

### 真实性原则
- 路演项目需确认后再列
- 评委阵容需核实
- 投资相关需 NFA 声明
