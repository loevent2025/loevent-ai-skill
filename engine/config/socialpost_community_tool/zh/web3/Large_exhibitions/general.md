# Web3 大型展会/博览会 - 通用类活动知识库

## 📌 模块激活条件

当活动为综合性展会，涵盖多个领域时，应用本模块规则：
- 综合性 Web3 峰会、区块链博览会
- 多板块并行的大型活动（技术+投资+社区）
- 年度/季度行业大会、地区性 Web3 Week

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 主论坛 + 多个分论坛 + 展区 + Side Events |
| 内容覆盖 | 技术、投资、生态、社区、政策全维度 |
| 参与方 | 项目方、投资机构、开发者、媒体、政府 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 行业从业者 | 行业全景、趋势判断、资源对接 |
| P1 | 项目方/创业者 | 品牌展示、投资对接、合作伙伴 |
| P2 | 投资人/分析师 | 赛道机会、项目发现、趋势分析 |
| P3 | 媒体/KOL | 新闻素材、采访机会、独家信息 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 大型综合峰会 | 正式专业型 (A) | 行业地位、权威阵容 |
| 地区 Web3 Week | 热情活力型 (B) | 丰富体验、多元板块 |
| 年度颁奖/盛典 | 正式专业型 (A) | 荣誉认可、品牌价值 |
| 博览会/展会 | 热情活力型 (B) | 展示体验、互动参与 |

## 📝 内容必含元素

### 1. 规模与影响力（至少2处）
- "[地区] 年度最大规模 Web3 盛会"
- "汇聚 [数量]+ 参与者 / [数量]+ 项目"

### 2. 多元板块（至少2处）
- "涵盖技术/投资/生态/社区四大板块"
- "[数量]+ 场主题论坛与分会场"

### 3. 核心价值（至少1处）
- "把握行业脉搏，洞察未来趋势"
- "一站式了解 Web3 全生态"

### 4. 合规提示（必须包含）
- "⚠️ 所有内容不构成任何投资建议（NFA）"

## ❓ 常见问答模板

| 问题 | 回答 |
|------|------|
| 不同板块需要单独报名吗？ | 标准参会证可进入所有公开板块 |
| 一张票可以参加所有三天吗？ | 是的，覆盖全部三天议程 |
