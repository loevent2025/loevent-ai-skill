# Web3 Demo Day - 社区类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- 社区项目展示、生态 Builder 展示、社区共建成果展
- DAO 提案展示、社区治理成果、开源贡献展示
- Meme/NFT 创作展示、社区创意展

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 项目展示 + 社区投票 + 颁奖派对 + 社区交流 |
| 时长 | 半天至一天，氛围轻松 |
| 参与方 | 社区 Builder、项目展示者、社区成员、生态方 |
| 核心目标 | 社区认可、成果庆祝、Builder 连接、文化传播 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 展示项目方 | 社区认可、曝光机会、反馈获取 |
| P1 | 社区成员 | 项目发现、投票参与、社区活动 |
| P2 | 生态项目方 | 合作机会、生态建设、人才发现 |
| P3 | 社区新人 | 了解生态、融入社区、学习机会 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 社区黑客松成果展 | 热情活力型 (B) | 庆祝氛围、社区认可 |
| 生态项目展示日 | 热情活力型 (B) | 生态多样、Builder 精神 |
| DAO 提案展示 | 正式专业型 (A) | 治理价值、社区决策 |
| 创意/Meme 展示 | 福利导向型 (D) | 趣味参与、奖励激励 |

---

## 📝 内容必含元素

### 1. 社区氛围（至少2处）
- "[数量] 个社区 Builder 项目集中展示"
- "社区投票选出最受欢迎项目"

### 2. 参与激励（至少1处）
- "参与投票赢取奖励"
- "展示者获得专属 POAP/NFT"

### 3. 社区价值（至少1处）
- "优秀项目获得生态官方支持"
- "展示成果纳入社区荣誉墙"

### 4. 社区精神（建议包含）
- "一起见证 Builder 的力量" / "WAGMI"
