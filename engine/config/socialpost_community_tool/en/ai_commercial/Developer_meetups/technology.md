# AI Developer Meetup — Technical Events Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- AI Tech Talk, Agent Developer Exchange, LLM Technical Salon
- RAG Tech Exchange Night, Prompt Engineering Meetup
- AI Architecture Discussion, Model Fine-Tuning Experience Sharing

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Lightning talks + Technical discussion + Open networking + Refreshment break socializing |
| Duration | 2–3 hours |
| Participants | AI developers, technical speakers, AI enthusiasts |
| Core Objective | Technical exchange, experience sharing, problem discussion, expanding connections |

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | AI Application Developers | Hands-on experience, best practices, troubleshooting |

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|

| Technical Q&A Session | Suspense-Driven (C) | Problem-driven, interactive discussion |
| Tool Hands-On Session | Value-Driven (D) | Trying new tools, accessing resources |
