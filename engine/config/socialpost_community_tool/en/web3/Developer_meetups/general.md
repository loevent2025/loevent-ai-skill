# Web3 Developer Meetup — General Event Knowledge Base

## 📌 Module Activation Conditions

Apply this module when the event is a comprehensive developer meetup covering multiple themes or audiences:
- General developer meetups, City-wide developer gatherings
- Monthly/quarterly developer exchanges, Developer community days
- Multi-topic mixed meetups

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Multi-topic talks + Panel discussions + Open networking + Social session |
| Duration | 2–4 hours |
| Participants | Developers of all types, Project teams, Learners, Investors |
| Core Objectives | Broad connections, Information exchange, Community building, Ecosystem engagement |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Active developers | Technical exchange, Project collaboration, Industry developments |
| P1 | Project teams | Talent recruitment, Developer relations, Community building |
| P2 | Learners / Newcomers | Industry awareness, Getting-started guidance, Network building |
| P3 | Investors / Observers | Talent discovery, Project developments, Industry insights |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Monthly developer meetup | Energetic & Engaging (B) | Regular exchange, Community continuity |
| City-wide developer day | Formal Professional (A) | Scale impact, City branding |
| Multi-topic mixed meetup | Energetic & Engaging (B) | Diverse content, Broad participation |
| Developer perks day | Benefits-Driven (D) | Resources and perks, Driving attendance |

---

## 📝 Required Content Elements

### 1. Diverse Value (at least 2 instances)
- "Covering [topic list] and more"
- "[Number] speakers sharing across different domains"

### 2. Networking Opportunities (at least 1 instance)
- "Open networking session"
- "Meet local developers in your city"

### 3. Open & Inclusive (at least 1 instance)
- "Developers at all stages are welcome"
- "Something for everyone — beginners and veterans alike"

### 4. Compliance Disclaimer (required)
- "⚠️ This is a technical exchange event and does not involve investment advice (NFA)"

## ⚠️ Prohibited Actions

- ❌ Turning the meetup into a product launch
- ❌ Over-using networking time for self-promotion
- ❌ Attending without engaging with others
- ❌ Collecting attendee information for marketing purposes
