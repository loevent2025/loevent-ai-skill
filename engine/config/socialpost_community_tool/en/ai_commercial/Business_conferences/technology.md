# AI B2B Meetings — Technical Events Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- CTO / Technical Executive AI Executive Session, AI Architecture Review Meeting
- Agent Technology Selection Workshop, LLM Deployment Strategy Meeting
- AI Infra Technology Strategy Session, Model Security Audit Workshop
- Enterprise AI Technology Partnership Discussion
