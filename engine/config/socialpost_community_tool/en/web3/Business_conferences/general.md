# Web3 Business Meeting — General Event Knowledge Base

## 📌 Module Activation Conditions

Apply this module when the event is a comprehensive business meeting covering multiple topics:
- Industry executive sessions, quarterly strategy meetings
- Ecosystem partner meetings, multi-party networking sessions
- Annual review meetings, industry trend discussions

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Themed discussions + Panel discussions + Breakout sessions + Business dinner |
| Content Coverage | Industry trends, Strategic planning, Partnership development, Resource alignment |
| Participants | Project founders, Executives, Investors, Industry experts |
| Core Objectives | Information exchange, Strategic alignment, Partnership advancement, Resource integration |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Founders / CEOs | Strategic direction, Industry trends, Core resources |
| P1 | Executives / VPs | Business partnerships, Market opportunities, Network building |
| P2 | Investors | Project updates, Market assessment, Investment opportunities |
| P3 | Industry Experts | Professional insights, Advisory opportunities, Influence |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Industry Executive Session | Formal Professional (A) | In-depth discussion, Strategic insights |
| Partner Meeting | Formal Professional (A) | Partnership advancement, Resource alignment |
| Trend Discussion | Suspense-Driven (C) | Industry assessment, Directional exploration |
| Resource Networking | Benefits-Driven (D) | Partnership opportunities, Resource access |

## 📝 Required Content Elements

### 1. Professional Positioning (at least 1 instance)
- "Invitation-only executive session for founders and C-level participants"
- "In-depth dialogue among key industry decision-makers"

### 2. Topic Value (at least 2 instances)
- "Exploring [year/phase] industry trends and strategic direction"
- "Sharing frontline hands-on experience and decision-making insights"

### 3. Networking Value (at least 1 instance)
- "Facilitating core business partnerships"
- "High-quality industry network"

### 4. Compliance Disclaimer (required)
- "⚠️ All content does not constitute investment advice (NFA)"
