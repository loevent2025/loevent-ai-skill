# GLASSMORPHISM GRADIENT 玻璃拟态渐变风格

## 适用场景
会议邀请、活动公告、专业演示、开发论坛、峰会资料、商务会议名片、企业通信

## 标志性色系
- **渐变A:** 橙色 (#FF6B4A) → 粉色 (#FF8FB9) → 浅蓝色 (#A8D8FF) → 蓝色 (#6BA3FF)
- **渐变B:** 粉色 (#FFB3E6) → 紫色 (#B794F6) → 蓝色 (#7B9BFF) → 深蓝色 (#4A5FFF)
- **背景:** 浅灰白色 (#F5F7FA) 带有渐变叠加
- **字体:** 主文本为黑色 (#1A1A1A)，次要文本为深灰色 (#4A4A4A)
- **玻璃效果:** 半透明白色，不透明度15-25%，背景模糊
- **特点:** 多色对角渐变、玻璃拟态效果、柔和模糊、空心字体、极简现代

## 视觉元素

### 玻璃拟态设计
- **玻璃卡片:** 半透明磨砂玻璃效果，白色不透明度15-25%
- **背景模糊:** 卡片后方背景模糊20-40px
- **边框:** 1px半透明白色边框 (30-40%不透明度)
- **阴影:** 柔和微妙的阴影 (0 8px 32px rgba(0,0,0,0.08))
- **圆角半径:** 16-24px的圆角，营造现代感
- **分层:** 多个玻璃卡片可以分层堆叠，增加深度

### 排版系统
- **标题排版:** 超粗体几何无衬线字体，纯色填充
- **字体风格:** 现代几何无衬线字体 (Inter, Montserrat, Poppins)
- **Logo文字:** 超粗体，大尺寸 (60-80pt)，纯色填充
- **正文:** 常规粗细无衬线字体 (16-20pt)
- **层级:** 清晰的尺寸对比 - 主标题 → 副标题 → 正文 → 页脚
- **对齐:** 左对齐或居中对齐，整洁的间距
- **颜色:** 纯黑色 (#1A1A1A) 或纯白色 (#FFFFFF)，单一颜色

### 背景渐变
- **方向:** 对角渐变 (45°或自定义角度)
- **色标:** 4-6个色标，从暖色到冷色过渡
- **平滑过渡:** 颜色之间平滑融合，无生硬边缘
- **叠加:** 2-5%不透明度的浅色纹理或网格图案 (可选)
- **变化:** 每张卡片可以有不同的渐变方向或调色板

### 卡片布局
- **卡片结构:** 包含活动信息的圆角玻璃卡片
- **内边距:** 24-40px的内部填充，宽裕的留白
- **内容块:** Logo → 标题 → 名片 → 日期/时间 → 地点 → 赞助商
- **嵌套卡片:** 主卡片内用于特定信息 (受邀人姓名) 的小卡片
- **间距:** 内容块之间16-24px的间距

## 效果处理

### 玻璃质感
- **材质:** 磨砂玻璃，带有微妙的透明度
- **模糊强度:** 背景上20-40px的高斯模糊
- **不透明度层:** 15-25%的白色叠加层用于玻璃效果
- **反射:** 边缘有微妙的光反射 (可选)
- **深度:** 多层卡片创建空间层次

### 渐变光效
- **柔和发光:** 弥散的光线扩散到背景中
- **颜色混合:** 平滑过渡，无条带感
- **亮度:** 渐变中心区域较亮，边缘较暗
- **氛围感:** 渐变+模糊组合营造出梦幻空灵的质感
- **对比度:** 浅色背景与深色文本，以保证易读性

### 细节装饰
- **网格叠加:** 可选的微妙网格图案 (2-3%不透明度)，营造科技感
- **图标:** 用于赞助商/Logo的极简单色图标
- **微交互:** 悬停状态时亮度增加 (适用于数字媒体)
- **纹理:** 极细微的噪点或颗粒 (2-4%不透明度)，营造高级感

## 构图策略

### 卡片式布局
- **卡片定位:** 居中或并排比较布局
- **比例:** 卡片占据画框的70-80%，留有呼吸空间
- **对齐:** 精确的基于网格的对齐方式
- **平衡:** 对称或不对称，取决于内容
- **边距:** 卡片周围10-15%的边距

### 信息层级
- **层级1:** 活动标题/Logo - 最大，超粗体纯色填充 (卡片的25-30%)
- **层级2:** 受邀人名片 - 突出嵌套卡片 (20-25%)
- **层级3:** 日期、时间、地点 - 清晰的正文文本 (15-20%)
- **层级4:** 赞助商Logo - 页脚区域 (10-15%)
- **留白:** 卡片区域的20-30%用于呼吸空间

### 视觉流程
- **阅读顺序:** 从上到下，从左到右
- **焦点:** 超粗字重首先吸引注意力
- **次要焦点:** 具有对比处理的名片
- **辅助信息:** 日期/地点易于扫描
- **页脚:** 赞助商Logo低调但可见

## 文字设置
- **颜色选择:** 选择与玻璃卡片/背景形成高对比度的纯色
  - 浅色玻璃卡片上推荐: 纯黑色 (#1A1A1A)、深灰色 (#333333)
  - 深色区域/渐变上推荐: 纯白色 (#FFFFFF)
- **填充方式:** 纯色填充，单一颜色，无渐变
- **禁止效果:** 
  - 无轮廓/描边字体
  - 无外发光/内发光
  - 无阴影/投影
  - 无渐变填充
  - 无透明度效果
  - 无任何文字特效
- **整体原则:** 文字保持干净、清晰、高可读性，通过超粗字重和尺寸层级建立视觉冲击力，与玻璃拟态的现代极简美学保持一致

- **分辨率:** 社交媒体1080x1350，印刷2480x3508 (A4)，300 DPI
- **色彩空间:** 数字RGB，印刷CMYK
- **软件:** Figma、Adobe XD用于UI设计，Photoshop用于最终渲染
- **玻璃效果:** CSS backdrop-filter 或 Photoshop模糊 + 不透明度图层
- **渐变:** 线性或径向渐变，带多个色标
- **格式:** PNG用于透明度，PDF用于可打印文件
