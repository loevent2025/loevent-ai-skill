# CYBERPUNK / 赛博朋克风格

## 适用场景
会议、黑客马拉松、NFT 发售、加密艺术节、区块链峰会、元宇宙活动、科技亚文化聚会

## 配色方案
- **Primary Colors:** 霓虹蓝 (#00F0FF), 电光紫 (#B026FF), 艳粉色 (#FF006E)
- **Background:** 深黑色 (#0A0A0A) 或 深海军蓝 (#0D1B2A)
- **Accent Colors:** 霓虹青色 (#00FFF5), 酸黄色 (#FFFF00), 毒绿色 (#39FF14)
- **Saturation:** 霓虹元素饱和度极高，营造发光效果
- **Color Behavior:** 发光光晕、色差、漏光效果

### 常用配色组合
- 经典赛博朋克: 霓虹蓝 (#00F0FF) + 艳粉色 (#FF006E) 在深黑色 (#0A0A0A) 背景上
- 银翼杀手: 紫色 (#B026FF) + 橙色 (#FF6B35) 搭配青色点缀
- 矩阵: 毒绿色 (#39FF14) + 黑色，带有数字雨效果

## 视觉元素
- 全息用户界面面板和边框
- 故障扫描线和数字伪影
- 科技符号和几何装饰元素（无文字字符）
- 电路板图案
- 线框网格和透视线
- 细节丰富的未来派天际线，具有多层建筑
- 雨线和光线反射
- 六边形和棱角分明的科技形状
- 二维码和数据可视化
- 霓虹招牌和全息广告
- 飞行器和无人机

## 效果处理
- 霓虹发光与外部光晕 (5-10像素半径)
- 边缘色差 (RGB 颜色分离)
- 光线轨迹上的运动模糊
- 颗粒/噪点叠加 (15-25% 不透明度)
- 漏光和镜头眩光
- 金属渐变反射
- 全息彩虹闪光
- 屏幕故障失真
- 带有体积光的氛围雾/霾

## 构图策略
- 垂直强调，高耸的城市景观占据画面 70-80%
- 强对角线的非对称构图
- 分层深度：远处的摩天大楼、中景建筑、前景结构
- 霓虹元素集中分布在上方三分之二区域
- 街道/地面层极少 (底部最多 10-15%)
- 透视网格汇聚到远处的灭点
- 浮动的 UI 元素散布其中
- 雨水或光线轨迹创造垂直韵律
- 高对比度区域：明亮霓虹灯与深邃阴影

## 文字设置
- **颜色选择:** 选择与深色背景形成高对比度的纯色
  - 深黑/深海军蓝背景推荐: 纯白色 (#FFFFFF)、霓虹蓝 (#00F0FF)、艳粉色 (#FF006E)
  - 备选纯色: 霓虹青色 (#00FFF5)、电光紫 (#B026FF)、毒绿色 (#39FF14)
- **填充方式:** 纯色填充，单一颜色，无渐变
- **禁止效果:** 
  - 无霓虹发光/外部光晕
  - 无色差 (RGB分离)
  - 无故障扫描线
  - 无全息彩虹闪光
  - 无漏光效果
  - 无描边
  - 无阴影/投影
  - 无任何文字特效
- **整体原则:** 文字保持干净、锐利、高可读性，作为霓虹城市景观中的清晰信息锚点，通过字体选择和高对比度纯色建立视觉冲击力，与复杂的赛博朋克背景形成"信息清晰"与"视觉混沌"的对比

## 情绪氛围
反乌托邦但充满活力，科技反叛，地下创新文化。整体能量强烈、神秘、前瞻性——捕捉高科技与低生活之间的交汇点，带有黑色电影般的氛围紧张感和数字乐观主义。
