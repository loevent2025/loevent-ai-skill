# Web3 Demo Day — Community Event Knowledge Base

## 📌 Module Activation Criteria

Apply the rules in this module when the event theme involves the following keywords:
- Community project showcase, ecosystem Builder showcase, community co-building results exhibition
- DAO proposal showcase, community governance outcomes, open-source contribution exhibition
- Meme / NFT creation showcase, community creative exhibition

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Project showcase + community voting + awards party + community exchange |
| Duration | Half-day to full day, relaxed atmosphere |
| Participants | Community Builders, project presenters, community members, ecosystem partners |
| Core Objective | Community recognition, celebrating outcomes, Builder connections, culture building |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Focus Areas |
|----------|---------------|-------------|
| P0 | Showcasing Project Teams | Community recognition, exposure, gathering feedback |
| P1 | Community Members | Discovering projects, voting participation, community events |
| P2 | Ecosystem Project Teams | Partnership opportunities, ecosystem development, talent discovery |
| P3 | Community Newcomers | Exploring the ecosystem, joining the community, learning opportunities |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Community Hackathon Results Showcase | Energetic & Engaging (B) | Celebratory atmosphere, community recognition |
| Ecosystem Project Demo Day | Energetic & Engaging (B) | Ecosystem diversity, Builder spirit |
| DAO Proposal Showcase | Formal & Professional (A) | Governance value, community decision-making |
| Creative / Meme Showcase | Benefits-Oriented (D) | Fun participation, reward incentives |
