## 📝 内容必含元素

### 1. 反差与张力营造（必须包含）

**公式**：
```
"当 [代表X世界的资本/团队] 遇见 [代表Y世界的项目/视角] 时，会发生什么？"
```

**要点**：
- 明确指出项目的跨界背景（传统经验 + Web3 能力）
- 强调投资人阵容的多元性（传统 VC + Crypto Fund）
- 突出这种组合的稀缺性和投资价值

---

### 2. 化学反应描述（必须包含）

**动态动词**：
- ✅ 使用："碰撞"、"融合"、"催化"、"激发"、"重塑"
- ❌ 避免："合作"、"结合"、"联合"

**融合公式**：
```
[传统优势] × [Web3 能力] = [独特投资价值]
```

**要点**：
- 描述跨界团队能创造什么传统团队或纯 Web3 团队无法创造的价值
- 说明为什么这种特定组合值得投资

---

### 3. 参与者碰撞描述（必须包含）

**投资人视角碰撞**：
```
"这里，传统 VC 想寻找 Web3 入口，Crypto Fund 想发现有落地能力的团队"
```

**项目方背景碰撞**：
```
"这里，Web2 老炮带着产业资源进入 Web3，Crypto Native 带着链上经验寻求破圈"
```

---

### 4. 场景与氛围构建（至少选2项）

**a) 具象化场景**
- 不要写："投资人与项目方对接"
- 要写：具体描述不同背景投资人评审同一项目时的视角差异
- 体现两种投资逻辑在同一空间碰撞的张力

**b) 时间流动感**
- 描述路演过程中认知碰撞的时刻
- 体现从陌生到理解的投资人认知变化

**c) 动态互动**
- 不要写："路演后有对接环节"
- 要写：具体描述不同背景投资人争相对接同一项目的场景

**d) 情绪能量**
- Demo Day：发现、惊喜、认可、竞争
- 对接环节：紧张、兴奋、突破、连接
- 社交晚宴：融合、流动、跨圈、延续

---

### 5. 独特投资价值想象（必须包含）

**描述只有跨界项目才能产生的投资价值**：
- 列出只有这种背景组合才能实现的商业场景
- 说明为什么传统资本和 Crypto 资本都应该关注
- 强调跨界团队的不可替代性

---

### 6. 蜕变弧线（推荐包含）

**投资人认知旅程**：
- **开始**：带着 [对另一个圈层的好奇/怀疑] 来
- **过程**：看到 [什么项目]，听到 [什么观点]，刷新 [什么认知]
- **结束**：带着 [新的投资视角/Deal Flow/跨圈人脉] 离开

---

### 7. 合规提示（必须包含）

```
"⚠️ 项目展示不构成投资建议，投资需独立判断"
```

## ✅ 跨界投资文案质量检查清单

- [ ] **反差与张力**：明确陈述了传统 vs Web3 的背景对比（项目方和投资人双维度）
- [ ] **化学反应**：使用了"碰撞""融合""催化"等动态动词描述投资价值
- [ ] **参与者碰撞**：描述了不同背景投资人和项目方的相遇场景
- [ ] **场景氛围**：至少应用了 2 种氛围构建技巧
- [ ] **独特价值**：列出了只有跨界组合才能产生的投资价值
- [ ] **投资人旅程**：有认知碰撞和视角刷新的叙事
- [ ] **情感驱动**：包含发现、惊喜、跨圈连接的兴奋感
- [ ] **合规提示**：包含投资风险提示
- [ ] **余韵**：暗示了活动之外的持续对接机制

## ⚠️ 禁止事项

- ❌ 贬低任何一方投资视角（"传统 VC 不懂 Web3""Crypto Fund 只看叙事"）
- ❌ 将 Demo Day 包装成确定的融资机会
- ❌ 承诺项目融资成功率或跨界溢价
- ❌ 泄露项目未公开融资信息
- ❌ 夸大项目估值或跨界背景价值
- ❌ 忽视跨界融合价值，写成普通投资路演文案
- ❌ 使用"合作""结合"等被动词汇替代碰撞感
- ❌ 过度渲染跨界概念，缺乏实质项目价值支撑
