# Web3 技术工作坊 - 教育类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- 入门编程工作坊、零基础开发培训
- 学生技术工作坊、高校实践课程
- 技能认证培训、职业技能工作坊

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 基础讲解 + 跟练实操 + 助教辅导 + 作品产出 |
| 时长 | 2-6 小时，循序渐进 |
| 参与方 | 学习者、学生、导师、助教 |
| 核心目标 | 入门学习、技能启蒙、兴趣培养、作品积累 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 零基础学习者 | 入门路径、学习门槛、上手难度 |
| P1 | 在校学生 | 实践机会、简历加分、行业认知 |
| P2 | 转型者 | 技能迁移、学习效率、职业方向 |
| P3 | 兴趣爱好者 | 体验尝鲜、了解技术、社交连接 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 零基础入门坊 | 热情活力型 (B) | 轻松氛围、鼓励尝试 |
| 高校合作工作坊 | 正式专业型 (A) | 学术价值、系统学习 |
| 体验式工作坊 | 福利导向型 (D) | 低门槛、有产出 |
| 技能提升工作坊 | 悬念引导型 (C) | 解决困惑、能力提升 |

---

## 📝 内容必含元素

### 1. 低门槛承诺（至少2处）
- "零基础友好，从第一行代码开始"
- "助教全程辅导，不怕跟不上"

### 2. 产出保证（至少1处）
- "完成你的第一个 [项目类型]"
- "带走一个可展示的作品"

### 3. 支持体系（至少1处）
- "[数量] 位助教现场支持"
- "小班教学，每 [数量] 人配 1 位助教"

### 4. 说明提示（建议包含）
- "本活动为教育培训性质，旨在技能启蒙"
