# Web3 Large-Scale Exhibition / Expo — Community Event Knowledge Base

## 📌 Module Activation Criteria

Apply the rules in this module when the event theme involves the following keywords:
- Web3 carnival, community conference, ecosystem festival
- Builder Fest, Creator Day, community anniversary celebration
- DAO conference, NFT art festival, community Meetup

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Main stage + ecosystem exhibition zone + interactive experiences + social party |
| Atmosphere | Relaxed, open, inclusive, WAGMI spirit |
| Participants | Community members, Builders, creators, ecosystem projects |

---

| Priority | Audience Type | Focus Areas |
|----------|---------------|-------------|
| P0 | Active Community Members | Community updates, exclusive perks, expanding connections |

| Event Type | Recommended Style | Core Focus |

## ❓ FAQ Templates

| Question | Answer |
|----------|--------|
| Can I attend without a technical background? | Of course! The carnival is open to all Web3 enthusiasts |
| How do I join the After Party? | A valid event pass grants access to all public parties |

## ⚠️ Prohibited Actions

- ❌ Promising airdrop value or financial returns
- ❌ Using language like "get rich quick" or "alpha leak" as marketing hooks
- ❌ Creating excessive FOMO
- ❌ Fabricating participating project or guest information
