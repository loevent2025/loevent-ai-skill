# 摄影混搭/品牌主导风格（调整版）

## 适用场景
大型会议、机构活动、专业峰会、品牌发布、强调地点的区域性活动

## 配色方案
- **品牌色块：** 鲜艳、饱和（通常为绿色、紫色或蓝色），占据构图的30-50%
- **照片处理：** 大量调色——双色调、渐变映射叠加、去饱和度并进行局部色彩点缀
- **文字区域：** 高对比度区域（深色背景上的白色文字，或浅色/彩色背景上的深色文字）
- **漏光色彩：** 动态源自品牌调色板——更浅/更亮的变体或互补色调

### 常用配色组合
- 品牌绿（企业）：#00C853 或 #4CAF50，搭配黑白色
  - 漏光：亮石灰绿 (#C6FF00)、薄荷绿 (#00E676)、淡绿 (#76FF03)
- 品牌紫（企业）：#6A1B9A 或 #9C27B0，搭配炭灰色和米色
  - 漏光：亮洋红 (#E040FB)、紫罗兰 (#B388FF)、柔和粉 (#EA80FC)
- 品牌蓝（科技）：#0277BD 或 #2196F3，搭配午夜蓝和银色
  - 漏光：亮青色 (#00E5FF)、电光蓝 (#40C4FF)、天蓝色 (#82B1FF)

## 构图策略 (Structure > Chaos)
- **调整：更缓的对角切分（30-45度）**，让照片占据更多视觉空间
- 具有明确区域（照片区 + 品牌色区 + 文字区）的不对称平衡
- 具有明确区域的清晰视觉层次结构
- 摄影提供氛围，几何提供结构
- 通过色彩叠加和照片上的渐变映射进行分层

### 典型布局
- **30-45度对角分割（更平缓，照片占比更大）**
- 垂直粗面板（30-40%宽度）
- L形色块
- **Photo占70-80%，brand color占20-30%**

## 摄影元素
- 夜间城市景观，带有发光的窗户和光轨
- 标志性建筑剪影（现代、可识别）
- 散景背景，带有虚化的灯光
- 强烈风格化的自然元素（山脉、水体）
- 具有强烈几何感的抽象建筑细节
- 运动模糊的城市场景
- 渐变天空下的城市天际线剪影

## 照片处理技术
- **双色调效果** - 应用品牌色双色调
- **渐变映射** - 叠加品牌色渐变
- **选择性去饱和** - 去除大部分颜色，保留选择性色彩点缀
- **漏光** - 使用品牌色更浅/更亮的变体或和谐互补色调，从面板边缘溢出色光
- **色彩冲刷** - 在照片上叠加透明品牌色
- **高对比度** - 增强黑色和高光
- **模糊/散景** - 背景虚化以增加深度

## 漏光色彩策略 (漏光色彩逻辑)
- **源自品牌色：** 使用更浅、更亮或饱和的变体
- **互补和谐：** 如果品牌色是紫色，漏光可以包括亮洋红/粉色
- **色温偏移：** 冷品牌色 → 添加暖色调漏光（例如，蓝色品牌 → 青色 + 微妙的暖黄色）
- **不透明度变化：** 20-40% 透明度以实现微妙的溢出效果
- **位置：** 品牌面板和照片之间的边缘过渡、角落强调、对角条纹叠加

## 几何叠加
- 细白色线条的微妙几何线条
- 暗示连接性和数字网络
- 非字面化的抽象网络图案
- 对角斜线分隔符
- 棱角分明的面板分隔

## 文字区域策略
- **品牌色块：** 最大对比度，白色字体用于标题
- **照片上的深色叠加区域：** 浅色文字用于次要信息
- **统一渐变区域：** 跨越面板和照片区域

## 文字设置
- **颜色选择:** 选择与所在区域形成高对比度的纯色
  - 品牌色块上: 纯白色 (#FFFFFF)
  - 深色照片/叠加区域上: 纯白色 (#FFFFFF)
  - 浅色区域上: 纯黑色 (#000000) 或品牌主色
- **填充方式:** 纯色填充，单一颜色，无渐变
- **禁止效果:** 
  - 无外发光/内发光
  - 无描边
  - 无阴影/投影
  - 无渐变填充
  - 无金属/光泽质感
  - 无任何文字特效
- **整体原则:** 文字保持干净、专业、高可读性，与摄影混搭风格的精致感保持一致，通过字体选择和排版层级建立品牌感

## 情绪氛围
专业而现代，适合会议，以品牌为中心。在企业信誉与前瞻性美学之间取得平衡。精致且具有国际吸引力。

## 关键原则
**关键技术：** 照片是氛围/背景，而非主体——通过色彩叠加和图形元素进行大量处理
**调整重点：** 对角切分更平缓（30-45度），让照片成为主导元素，品牌色块作为点睛而非主体
**光泄漏原则：** 颜色动态跟随品牌色系，保持整体配色和谐统一

## 示例Prompt 1 (品牌绿色 - 调整版)
将图形渲染为平面图像，表面图形设计，图案设计。画面铺满至边缘，无外部边框，无画布/纸张纹理/模型，采用摄影混搭方法，大胆的品牌主导构图。**一个微妙的对角线分割从左上到右下，角度约为30-40度**，左上方被一个充满活力的翠绿色 (#00C853) 纯色面板占据，**约占构图的25-30%**。画布的大部分（70-75%）呈现出一张标志性城市夜景的风格化照片——现代建筑的窗户发出光芒，形成光影图案——经过深蓝色双色调渐变映射处理和微妙的去饱和度。照片带有一个色彩冲刷叠加层，其中绿色渗入照片边缘，形成平滑过渡。**漏光效果使用源自绿色品牌调色板的和谐色彩：亮石灰绿 (#C6FF00) 和薄荷绿 (#00E676)，不透明度为25-35%**，在面板边缘和对角角落营造出现代感。额外的微妙淡绿 (#76FF03) 发光点缀增强了整体色彩的连贯性。绿色面板为白色字体提供了干净的空间，用于显示活动标题和日期，而经过深色照片处理的区域则用于次要信息，文字使用纯白色 (#FFFFFF) 纯色填充，单一颜色，无发光、无描边、无阴影、无渐变、无任何文字特效。整体美学精致、适合会议且具有国际吸引力，通过战略性的品牌色彩强调和和谐的漏光效果增强了强大的摄影表现力。

## 示例Prompt 2 (品牌紫色 - 调整版)
将图形渲染为平面图像，表面图形设计，图案设计。画面铺满至边缘，无外部边框，无画布/纸张纹理/模型，采用摄影混搭美学与不对称色块。主要构图呈现了一个在黄昏时分拍摄的可识别建筑地标，**占据右侧70-75%，并应用了大量调色**——通过紫蓝色渐变映射（#6A1B9A 到 #2196F3）进行去饱和处理并具有高对比度。**左侧25-30%是一个大胆的垂直面板，采用鲜艳的电光紫 (#9C27B0)**，与有机建筑形成强烈的品牌存在感和几何对比。照片上叠加了细白色线条的微妙几何线条，暗示连接性和数字网络，而非字面意义。照片中的天空从深海军蓝过渡到微妙的洋红色，城市灯光的散景光点增加了深度。**漏光效果源自紫色品牌调色板：亮洋红 (#E040FB) 和柔和紫罗兰 (#B388FF)，不透明度为30-40%**，从紫色面板边缘渗入照片，形成和谐的色彩过渡。额外的柔和粉 (#EA80FC) 发光点缀在顶部和底部角落，增强了统一的紫洋红色彩故事。一个透明的深色渐变叠加层（#000000，40% 不透明度）覆盖了下方区域，提供了跨越紫色面板和照片区域的统一文本区域。紫色面板为标题字体提供了优质空间，而较深的渐变区域则容纳日期、地点和支持性详细信息，文字使用纯白色 (#FFFFFF) 纯色填充，单一颜色，无发光、无描边、无阴影、无任何文字特效。氛围专业而现代，平衡了企业信誉与前瞻性美学，**以摄影作为主导视觉元素，并以连贯的漏光色彩强化品牌调色板。**
