# AI Technical Workshop – Community Events Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- AI Builder Workshop, Community Co-Building Session
- Open-Source Project Contribution Workshop, Community Coding Night
- AI Maker Hands-On Workshop, Builder Jam Session

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Themed talks + Hands-on practice + Peer collaboration + Project showcase |
| Duration | 3–4 hours |
| Participants | Community Builders, AI enthusiasts, open-source contributors |
| Core Objective | Community co-building, skill sharing, Builder connections, open-source contributions |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Community Builders | Skill development, project collaboration, community contributions |
| P1 | Open-source contributors | Open-source projects, technical contributions, community recognition |
| P2 | AI enthusiasts | Hands-on experience, technical learning, community integration |
| P3 | Community newcomers | Getting started, making friends, finding belonging |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Builder Co-Building Workshop | Energetic & Engaging (B) | Collaborative atmosphere, community spirit |
| Open-Source Contribution Night | Energetic & Engaging (B) | Open-source culture, contribution incentives |
| Technical Sharing & Practice | Curiosity-Driven (C) | Technical topics, hands-on validation |
| Beginner-Friendly Workshop | Value-Driven (D) | Resource access, community connections |

---

## 📝 Required Content Elements

### 1. Community Atmosphere (at least 2 instances)
- "Get hands-on with [number]+ Builders"
- "[Community Name] technical sharing and practice session"

### 2. Collaboration Value (at least 1 instance)
- "Form teams on-site and build a mini project together"
- "Learn from each other, review each other's code"

### 3. Community Spirit (at least 1 instance)
- "Share your technical insights"
- "Contribute code to open-source projects"

### 4. Community Expression (recommended)
- "Let's Build together!" / "Happy Coding!"

## ⚠️ Prohibited Actions

- ❌ Turning a community event into a paid training session
- ❌ Excessive promotion of personal commercial projects
- ❌ Technical gatekeeping or looking down on beginners
- ❌ Undermining the friendly, collaborative community atmosphere
