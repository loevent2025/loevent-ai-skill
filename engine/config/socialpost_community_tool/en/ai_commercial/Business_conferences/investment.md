# AI Business Conference — Investment Event Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- AI investor executive session, LLM sector LP meeting
- AI project due diligence session, post-investment management meeting, exit strategy workshop
- AI asset allocation workshop, AI startup-investor matchmaking session
- AI fund annual meeting, AI sector investment strategy session

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Executive session + Panel discussion + 1-on-1 meetings + Networking dinner |
| Content Depth | Investment decision-making, sector analysis, due diligence methodology, exit planning |
| Participants | GPs/LPs, AI fund managers, AI founders, financial advisors (FAs) |
| Core Objective | Advance investment decisions, facilitate deal flow, exchange market intelligence |

| Priority | Audience Type | Key Concerns |
