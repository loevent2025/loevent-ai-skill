# Web3大型展会场景知识库 - 投资类

## 场景概述
投资类Web3大型展会聚焦Crypto领域的投融资、项目路演和资本对接，汇聚Web3创业者、Crypto VC、产业资本、家族办公室和FA机构。这类活动是发现优质Web3项目、了解投资周期趋势、建立资本关系的核心场景，也是创业者获取融资、品牌曝光的重要舞台。

---

## 场景特征
- **规模**：数百至数千人参与
- **时长**：通常1-2天
- **形式**：项目路演 + 投资论坛 + 1v1对接 + 闭门交流 + 颁奖典礼
- **参与者**：Web3创业者、Crypto VC/Fund、家族办公室、FA/OTC、交易所投资部、孵化器、政府引导基金
- **特点**：项目集中、资本密集、对接高效、周期敏感

## 参会攻略建议

### 对于投资人
- 提前研究参会项目列表、Deck和链上数据
- 明确本次重点关注的赛道和阶段
- 准备好标准的DD问题清单（技术/Token/安全/团队）
- 预约感兴趣项目的1v1 Office Hour
- 准备好Fund介绍和投资偏好说明

### 对于创业者
- 准备精炼的8分钟Pitch + Live Demo版本
- 预判投资人可能的DD问题（Token/安全/竞争）
- 准备好不同详细程度的Deck + One-pager
- 研究参会Fund的Portfolio和投资偏好
- 准备好链上数据Dashboard和合约审计报告

### 对接技巧
- 开场简洁，30秒内讲清核心价值和Traction
- 链上数据说话，避免空泛描述Narrative
- 坦诚回答Token模型和安全问题，不要回避
- 主动跟进，24小时内发TG/邮件约定下一步
- 做好记录，及时整理每次对接的要点和反馈

## 注意事项
- **合规声明必须有**：NFA / DYOR / 不构成投资建议 / 对接≠承诺
- **不涉及Token推荐、价格预测、收益承诺**
- **投资人阵容要有决策权**：GP/Managing Director级别
- **对接机制说清楚**：1v1预约流程、Office Hour规则
- **双边价值**：对投资人和创业者都要有吸引力
- **数据和判断要有干货**：链上数据、赛道洞察，不是空谈Narrative
