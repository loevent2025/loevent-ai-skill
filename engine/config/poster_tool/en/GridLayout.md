# GRID COLLAGE — Style Guide

## Applicable Use Cases

Campus recruitment, design workshops, creative competitions, brand launch events, art exhibitions, designer recruitment, creative school open days, design software promotion

---

## Color Palette

- **Primary tones:** Bright blue, pure white, pure black
- **Backgrounds:** White or light gray grid pattern
- **Supporting colors:** Layered grays (light to dark), chromatic accents
- **Saturation:** High saturation for blue; low saturation for gray tones
- **Color behavior:** On a clean, bright base, blue and black establish visual hierarchy

---

## Standard Color Combinations

| Theme | Colors |
|---|---|
| **Design Software** | White grid background + Bright Blue + Black text blocks |
| **Campus Recruitment** | Light blue background + White + Dark gray elements |
| **Creative Collage** | White base + Multi-layered grays + Blue accent |

---

## Visual Elements

- Checkerboard / grid background patterns
- Collage-style rectangular text blocks
- Selection marquee / crop frame UI elements (with corner handles)
- Pixelated gradient color blocks (grayscale transitions)
- Blue label and button elements
- Emoji icons or mascot characters
- Layered rectangles and squares
- Subtle design software interface references

---

## Treatment & Effects

- Clean, hard-edged graphic forms
- Flat design with no three-dimensional effects
- Precise grid alignment
- Solid color fills for all color blocks
- Dashed or solid borders on selection frames
- Occlusion relationships between layered elements
- Stepped pixel gradient transitions
- Overall UI design sensibility

---

## Composition Strategies

- Grid as the foundational ordering structure
- Text blocks arranged in a staggered collage layout
- Selection frame elements create an "actively editing" feel
- Pixel gradient blocks positioned at corners or the bottom edge
- Mascot or icon elements as accent details
- Blue elements serve as visual guides
- Clear information hierarchy (primary headline, subheadline, detail)

---

## Typography Guidelines

- **Color selection:** High-contrast solid color combinations
  - Recommended primary: Pure Black `#000000`, Pure White `#FFFFFF`
  - Supporting: Bright Blue `#0099FF`, Mid Gray `#666666`
- **Fill method:** Solid color fill only — single color, no gradients
- **Prohibited effects:**
  - No gradient fills
  - No shadows / drop shadows
  - No outer glow / inner glow
  - No strokes
  - No blur effects
  - No text effects of any kind
- **Typeface style:** Geometric sans-serif or bold gothic; mixed weights (Light to Bold)
- **Overall principle:** Typography must remain clean, sharp, and highly legible — typeface selection and high-contrast solid color establish clear visual hierarchy against the grid background

---

## Mood & Atmosphere

Creative energy, the beauty of order, design thinking. The overall tone is young and professional, conveying a love of design tools and the creative process — evoking the creative satisfaction of working within design software, with a dual influence of Bauhaus design and modern UI aesthetics.

Render the graphic as a flat image, surface graphic design, pattern design. Full bleed, no external borders, no canvas/paper texture/mockups. Content fills the entire canvas to the edges with absolutely no margins or white space at corners. No visible text, letters, numbers, or color codes in the image. Grid collage aesthetic with design software interface vibes. White or light gray background with subtle grid pattern — evenly spaced lines creating a graph paper effect. Multiple rectangular shapes layered in collage style — some filled with solid black, others with bright blue (`#0099FF`), others with various grays. Selection marquee elements (dashed rectangle outlines with corner handles) suggesting active editing state. Pixel gradient blocks in grayscale showing stepped transitions from white to black, positioned in corners or bottom edge. A cute mascot or emoji-style character element (simple geometric face with dots for eyes) in blue adds personality. Design software UI elements subtly referenced — toolbars, selection handles, layer indicators. Clean flat design with no gradients, shadows, or 3D effects. The composition uses the grid as an organizing principle with elements aligned to create order within creative chaos. Lower third reserved for information with cleaner layout.
