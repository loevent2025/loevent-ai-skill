# SYNTHWAVE / 合成波风格（复古未来）

## 适用场景
Web3大会主视觉、黑客马拉松、NFT发行、加密艺术节、元宇宙发布、复古游戏活动、电子音乐节

## 配色方案
- **主色调:** 深紫色 (#7B2CBF), 炽热洋红色 (#FF006E), 电光蓝 (#00B4D8)
- **背景:** 从深紫色到黑色的渐变 (#1A0B2E 到 #000000)
- **强调色:** 霓虹粉 (#FF10F0), 青色 (#00FFFF), 落日橙 (#FF6D00)
- **饱和度:** 高饱和度，带有发光的霓虹效果
- **色彩表现:** 渐变融合、霓虹光晕、氛围色渗

### 常用配色组合
- 经典合成波: 紫色 (#7B2CBF) + 洋红色 (#FF006E) + 青色 (#00FFFF) 在深色渐变背景上
- 落日驾驶: 橙色 (#FF6D00) + 粉色 (#FF10F0) + 紫色，带地平线光晕
- 霓虹之夜: 电光蓝 (#00B4D8) + 炽热粉色 在深紫色基底上

## 视觉元素
- 带有透视消失点的网格地平线
- 复古几何太阳/月亮（通常是半圆形）
- 棕榈树剪影
- 线框山脉或金字塔
- 霓虹城市天际线剪影
- 带斜角边缘的镀铬/金属文字
- 激光束和光轨
- 星场或点状天空图案
- 复古电脑图形美学
- 扫描线叠加

## 效果处理
- 霓虹外发光（10-15像素半径带颜色）
- 带有平滑色彩过渡的渐变网格
- 关键元素的色差
- VHS跟踪线和噪声
- 太阳/光源上的径向模糊
- 带有紫色/粉色调的氛围薄雾
- 带有彩虹渐变的金属镀铬反射
- 光晕和镜头光斑
- 水平扫描线（1-2像素间距，20%不透明度）

## 构图策略
- 强烈的透视与中心消失点
- 网格地面延伸至地平线（通常在下三分之一处）
- 太阳/月亮位于地平线中心
- 带有中心焦点的对称构图
- 分层深度：前景网格、中景元素、背景天空
- 针对发光背景的剪影元素
- 文字放置在上三分之一处或居中
- 垂直光束或柱子构成构图框架
- 复古长宽比（4:3或16:9信箱格式）

## 文字设置
- **颜色选择:** 选择与深色背景形成高对比度的纯色
  - 推荐色: 青色 (#00FFFF), 纯白 (#FFFFFF), 霓虹粉 (#FF10F0), 炽热洋红 (#FF006E)
- **填充方式:** 纯色填充,单一颜色,无渐变
- **禁止效果:** 
  - 无外发光/内发光
  - 无描边
  - 无阴影/投影
  - 无金属/镀铬质感
  - 无彩虹渐变反射
  - 无斜角/浮雕
  - 无任何文字特效
- **整体原则:** 保持文字干净、简洁、高可读性,与背景的复杂视觉效果形成对比

## 情绪氛围
怀旧而又未来主义，1980年代科幻愿景的梦幻乐观主义。整体能量是充满活力、复古酷炫、令人愉悦的——捕捉了合成波音乐的精神，伴随着落日驾驶、霓虹之夜和无尽的数字地平线，唤起了怀旧和乌托邦式的科技梦想。
