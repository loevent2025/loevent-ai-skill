# AI Social Mixer — Technical Events Knowledge Base

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Casual socializing + Informal tech chat + Open-floor mingling |
| Duration | 3–4 hours |
| Participants | AI developers, tech enthusiasts, the curious |
| Core Objective | Low-key socializing, casual tech talk, developer connections |

---

## 🎨 Style Override Rules

**When this module is activated, it overrides the standard style:**

✅ Relaxed, conversational, youthful
✅ Direct, inviting, rhythmic
✅ Short sentences and bite-sized copy
✅ Purely positive framing — showcase the vibe

❌ No contrast hooks ("Tired of X?")
❌ No negativity or criticism of other events
❌ Avoid utilitarian language like "takeaways," "value," or "actionable insights"

## 📝 Required Content Elements

### 1. Scene-First Opening (mandatory)
- What makes the venue unique + sensory details + mood words
- Example: "Industrial loft, warm lighting — talk code, then talk about everything else"

### 2. Five-Senses Immersion (at least 3)
- Sight: Ambient lighting, open space, code scrolling on screens (decorative)
- Taste: Craft beer, cocktails, bites
- Sound: Lo-fi beats, keyboard-clack samples (tongue-in-cheek)
- Mood: Laid-back, slightly tipsy, geeky but never serious

### 3. Social Snapshot (mandatory)
- Describe "what's happening right now"
- Example: "Someone's at the bar debating Agent architecture, someone's on the couch ranting about APIs"

### 4. Zero-Pressure Participation (mandatory)
- "No prep needed — you don't have to write code to show up"
- "No presentations, no talks — just drinks and conversation"

## ⚠️ Prohibited Actions

- ❌ Using words like "takeaways," "actionable insights," or "learnings"
- ❌ Turning the mixer into a tech talk session
- ❌ Over-emphasizing "networking," "connections," or "resources"
- ❌ Creating FOMO ("Don't miss out!")
