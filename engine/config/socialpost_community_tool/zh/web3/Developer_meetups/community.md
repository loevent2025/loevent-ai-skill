# Web3 开发者聚会 - 社区类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- 社区开发者聚会、Builder Hangout、生态开发者夜
- DAO 开发者交流、开源贡献者聚会、社区 Coding Night
- Builder After Party、开发者社交聚会

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 轻分享 + 自由交流 + 社交游戏 + 派对氛围 |
| 时长 | 2-4 小时，氛围轻松 |
| 参与方 | 社区 Builder、生态开发者、技术爱好者 |
| 核心目标 | 社区连接、Builder 社交、轻松交流、归属感 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 社区 Builder | 同好连接、项目交流、社区归属 |
| P1 | 生态开发者 | 生态动态、合作机会、社交放松 |
| P2 | 技术爱好者 | 融入社区、认识朋友、行业了解 |
| P3 | 社区新人 | 破冰融入、认识 OG、找到组织 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| Builder 社交夜 | 热情活力型 (B) | 轻松社交、派对氛围 |
| 生态开发者聚会 | 热情活力型 (B) | 生态连接、项目交流 |
| 社区 Coding Night | 悬念引导型 (C) | 技术挑战、共同解决 |
| Builder 福利夜 | 福利导向型 (D) | 社区福利、抽奖派对 |

---

## 📝 内容必含元素

### 1. 社区氛围（至少2处）
- "和 [数量]+ Builder 一起聚"
- "社区老朋友新朋友都来"

### 2. 轻松氛围（至少1处）
- "不聊工作，纯放松"
- "小食啤酒管够"

### 3. 连接价值（至少1处）
- "认识更多社区 Builder"
- "找到你的下一个合作伙伴"

### 4. 社区精神（建议包含）
- "GM frens!" / "WAGMI" / "LFG"
