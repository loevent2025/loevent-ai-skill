# BLACK & GOLD CORPORATE LUXURY — Design Style Guide

---

## ⚠️ Core Non-Negotiables

> **Only three colors are permitted: Pure Black (#000000) + Bright Gold (#F3BA2F / #F0B90B) + Pure White (#FFFFFF).**
> **Gold is the sole accent color. Gray, blue, red, silver, and all other colors are strictly prohibited.**
> **The background must be pure black. Deep gray, gradient blacks, or any tinted dark substitute are not allowed.**

---

## Intended Use Cases

Executive summits, financial conferences, blockchain events, product launches, corporate keynotes, premium business forums, institutional presentations.

---

## Color Palette (Strictly Three Colors)

| Role | Color | Hex | Notes |
|------|-------|-----|-------|
| Background | Pure Black | `#000000` | Absolute black — `#111111`, `#1A1A1A`, and all dark-gray variants are banned |
| Accent | Bright Gold | `#F3BA2F` or `#F0B90B` | The only permitted accent; saturation 90–100% for maximum luxury impact |
| Text | Pure White | `#FFFFFF` | Used for headings and body copy |

**Ratio:** 70% Black / 20% Gold / 10% White

**Absolute restrictions:**
- No gradients, shadows, midtones, or gray values
- No fourth color of any kind
- No tinted black or off-white variants
- **Gold** — all visual elements (borders, icons, dividers, decorative particles, emphasis text, logo outlines)
- **White** — text content only (headings, body, data labels)

---

## Visual Element Types

| Category | Description |
|----------|-------------|
| **Geometric** | Wireframe cubes, pyramids, hexagons; overlapping rectangles; diamond patterns; circular rings; angular chevrons — all in gold stroke |
| **Icons** | Large-scale line icons (charts, globes, trophies, shields, crowns); abstract data visualizations; currency symbols; network diagrams — all in gold outline |
| **Abstract** | Gold light beams, particle fields, wave patterns, circuit traces, radiating lines, upward-trending graphics — all in gold lines |
| **Symbolic** | Product silhouettes, architectural skylines, mountain peaks, shield/badge outlines, crowns/laurel wreaths — all in gold stroke |

---

## Typography

**Font Family:** Geometric sans-serif — Montserrat, Futura, Avenir, or Roboto (no serif, no decorative fonts)

**Weights:** 900 (headings) · 700 (emphasis) · 400–500 (body)

**Color Rules:**
- Headings → White (`#FFFFFF`)
- Body copy → White (`#FFFFFF`)
- Emphasized words / key figures / brand name → Gold (`#F3BA2F`)
- Gray, light gold, and any other text color are prohibited

**Case:** Primary headings in ALL CAPS; subheadings in Title Case or Sentence case

**Spacing:** Wide letter-spacing for headings (100–200); line-height 1.2–1.4

**Hierarchy:** Maximum-to-minimum font size ratio of 8:1

**Effects:** None — fully flat; no shadows, outlines, or glows

---

## Layout Patterns

| Pattern | Description |
|---------|-------------|
| **Split (40/60)** | Text zone on one side; visual element on the other |
| **Center Focus** | Large visual element (40–60% of canvas); text arranged around it |
| **Horizontal Bands** | Layered zones separated by gold divider bars |
| **Asymmetric Dynamic** | Visual element anchored to one corner; text flows diagonally opposite |
| **Grid** | 8–16px baseline grid; 5–8% margins; pixel-perfect alignment; 40–50% negative space |

---

## Visual Element Specifications

| Property | Specification |
|----------|--------------|
| **Style** | Outline / wireframe — no solid fills |
| **Color** | Gold (`#F3BA2F`) is the only element color; white used sparingly as a secondary |
| **Stroke** | 2–4px uniform weight; gold throughout |
| **Size** | Hero element occupies 30–60% of the visual area |
| **Position** | Off-center to create visual momentum |
| **Opacity** | 60–100% (layering permitted, but underlying color must remain pure black) |
| **Quality** | Sharp vector graphics; crisp edges; minimal complexity |
| **Fill** | No solid fills — all elements are outlines (exception: occasional narrow gold accent bars) |

---

## Effects & Decoration

**Permitted:** Flat design only; subtle opacity variation (80–100%)

**Prohibited:**
- Gradients (including gold-to-black)
- Shadows (drop shadow, inner shadow, text shadow — all types)
- 3D rendering (wireframe excepted)
- Textures, blur, glows, rounded speech bubbles, halo effects

| Element | Specification |
|---------|--------------|
| **Icons** | 2–4px gold stroke; minimal geometric forms; sharp or minimal rounding (max 4px radius) |
| **Dividers** | Thin gold lines (1–2px) or gold accent bars (8–16px height) |
| **Contrast** | Minimum 21:1 for accessibility compliance |

---

## Composition Strategy

- Mathematical precision in spacing and alignment
- Horizontal and vertical structure dominates
- Clear visual hierarchy: Primary → Secondary → Supporting
- Minimum 40–50% negative space (pure black breathing room)
- Clearly defined information zones
- Directed visual flow: Focal point → Supporting element → Detail
- Balance and stability take precedence over dynamic tension
- Rule-of-thirds or golden-ratio positioning
