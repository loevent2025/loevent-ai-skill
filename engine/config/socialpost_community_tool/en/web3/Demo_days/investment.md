# Web3 Demo Day - Investment Event Knowledge Base

## 📌 Module Activation Conditions

Apply this module when the event theme involves the following keywords:
- Accelerator Demo Days, cohort graduation pitch events, investor-only showcases
- Fundraising pitch days, Pitch Days, project showcase events
- VC-exclusive showcases, seed-stage project days

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Project pitches + investor commentary + 1-on-1 meetings + networking dinner |
| Duration | Half-day to full-day; 5-10 minutes per project pitch |
| Participants | Founding teams, investment firms, accelerators, FAs |
| Core Objective | Fundraising matching, investor exposure, project screening, follow-up facilitation |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Investment Firms | Project quality, team background, investment opportunities |
| P1 | Founding Teams | Fundraising opportunities, investor feedback, exposure |
| P2 | FAs / Investment Banks | Deal sourcing, transaction opportunities, market intelligence |
| P3 | Media / Industry Observers | Industry trends, standout projects, editorial material |
