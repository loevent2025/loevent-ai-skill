# AI 技术工作坊 - 社区类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- AI Builder 工作坊、社区 Co-building Session
- 开源项目贡献工作坊、社区 Coding Night
- AI Maker 实操坊、Builder Jam Session

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 主题分享 + 自由实操 + 互助协作 + 成果展示 |
| 时长 | 3-4 小时 |
| 参与方 | 社区 Builder、AI 爱好者、开源贡献者 |
| 核心目标 | 社区共建、技能分享、Builder 连接、开源贡献 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 社区 Builder | 技能提升、项目协作、社区贡献 |
| P1 | 开源贡献者 | 开源项目、技术贡献、社区认可 |
| P2 | AI 爱好者 | 动手体验、技术学习、社区融入 |
| P3 | 社区新人 | 入门体验、认识朋友、找到归属 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| Builder 共建坊 | 热情活力型 (B) | 协作氛围、社区精神 |
| 开源贡献夜 | 热情活力型 (B) | 开源文化、贡献激励 |
| 技术分享实操 | 悬念引导型 (C) | 技术话题、实践验证 |
| 新手友好坊 | 福利导向型 (D) | 资源获取、社区连接 |

## 📝 内容必含元素

### 1. 社区氛围（至少2处）
- "和 [数量]+ 位 Builder 一起动手"
- "[社区名] 技术分享与实操"

### 2. 协作价值（至少1处）
- "现场组队协作完成小项目"
- "互相学习，互相 Review"

### 3. 社区精神（至少1处）
- "分享你的技术心得"
- "为开源项目贡献代码"

### 4. 社区表达（建议包含）
- "Let's Build together!" / "Happy Coding!"
