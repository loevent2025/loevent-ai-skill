# RETRO PIXEL 复古像素风格

## 适用场景
节日活动宣传、文化庆典、怀旧主题派对、数字艺术展览、独立游戏发布、Y2K复古活动、社区线下聚会、复古科技展

## 配色方案
- **主色调:** 暖米色、奶油白、柔和粉棕
- **背景:** 米白色、浅灰色、暖杏色
- **饱和度:** 低饱和度，柔和复古色调
- **色彩表现:** 有限色板、像素化渐变、抖动效果

### 常用配色组合
- 经典怀旧: 米白色 (#F5F0E6) + 暖棕色 (#8B6F5C) + 灰色 (#7A7A7A) + 深棕色 (#4A3728)
- 樱花暖调: 奶油白 (#FDF8F3) + 腮红粉 (#D4A5A5) + 灰棕色 (#9C8578) + 炭黑色 (#2D2D2D)
- 复古电脑: 浅灰色 (#E8E4DC) + 窗口边框棕 (#B8906F) + 像素黑 (#1A1A1A) + 柔和粉 (#E8C4B8)
- 暖调中性: 杏色 (#F2E6D9) + 玫瑰灰 (#C4A698) + 石墨灰 (#5C5C5C) + 深咖啡 (#3D2B1F)

## 视觉元素
- 复古操作系统窗口界面 (Windows 95/98风格)
- 8-bit/16-bit像素艺术图形
- 复古鼠标光标图标
- 文件夹、磁盘等拟物化图标
- 进度条和滚动条元素
- 网格点阵背景纹理
- 窗口标题栏和关闭按钮
- 像素化半色调图案

## 效果处理
- 有限色板 (8-16色)
- 像素抖动 (Dithering) 模拟渐变
- 硬边缘像素化处理
- 1px或2px黑色描边
- 无抗锯齿的锐利边缘
- 网格对齐的精确像素
- 复古CRT扫描线效果 (可选)
- 轻微纸张/噪点纹理叠加
- 窗口阴影投射
- 等距或正交视角

## 构图策略
- 多窗口层叠排列营造桌面感
- 主信息窗口居中放大突出
- 装饰性小窗口散布四周
- 底部任务栏/状态栏信息区
- 顶部导航栏显示品牌/日期/地点
- 网格系统严格对齐
- 留白与密集区域形成节奏
- 图标元素点缀空白区域

## 文字设置
- **字体选择:** 像素字体、点阵字体、等宽字体
  - 推荐风格: Press Start 2P、VCR OSD Mono、像素宋体、点阵黑体
- **颜色选择:** 高对比度纯色
  - 浅色背景推荐: 纯黑色 (#000000)、深棕色 (#3D2B1F)、炭灰色 (#2D2D2D)
  - 深色背景推荐: 纯白色 (#FFFFFF)、米白色 (#F5F0E6)
- **填充方式:** 纯色填充，单一颜色
- **禁止效果:**
  - 无渐变填充
  - 无发光效果
  - 无阴影特效
  - 无3D立体字
  - 无模糊处理
  - 无金属质感
- **整体原则:** 文字保持像素化锐利边缘，字重偏粗以保证可读性，与复古界面风格统一协调

温暖怀旧、数字复古、亲切友好、文化融合。整体氛围是舒适的千禧年回忆与现代设计感的结合——用低保真美学传递高级感，以像素艺术的朴素魅力唤起对早期互联网时代的温情记忆，同时保持当代设计的精致与克制。
