# CITY SILHOUETTE — Visual Style Guide

## Use Cases
Urban event announcements, regional conferences, blockchain meetups, hackathon city tours, local community gatherings, multi-city roadshow posters

## Signature Color System
- **Brand-customizable:** User provides brand primary color
- **Monochrome option:** Black/gray silhouettes on white/light gray background
- **Two-tone option:** Brand color + accent color (e.g., cyan + teal, green + yellow)
- **Silhouette layering:** Multiple opacity levels of the same color (30%, 50%, 70%, 100%) for added depth
- **Background:** White (#FFFFFF), light gray (#E8E8E8), or a subtle brand color tint
- **Typography:** Black (#1A1A1A) or dark brand color for body text
- **Key features:** Brand-color-driven customization, city silhouette occupying bottom one-third, flat vector style, monochrome or two-tone system

---

## Visual Elements

### City Silhouette Composition
- **Position:** Bottom one-third of canvas (33% vertical space)
- **Style:** Flat vector silhouette with clean geometric shapes
- **Layering:** 3–5 depth layers (foreground → background)
- **Foreground (100% opacity):** Bridges, trees, waterfront structures
- **Midground (70–80% opacity):** Mid-rise buildings, iconic landmarks
- **Background (40–60% opacity):** Distant skyscrapers, far cityscape
- **Far background (20–30% opacity):** Subtle receding architecture

### Architectural Elements
- **Landmark buildings:** Iconic urban structures (towers, cultural heritage sites)
- **Modern skyscrapers:** Varied heights and shapes for a dynamic skyline
- **Traditional elements:** Temples, pagodas to reflect local cultural identity
- **Infrastructure:** Bridges and arches as foreground features
- **Vegetation:** Simplified tree silhouettes in the foreground
- **Texture details:** Horizontal line patterns and geometric motifs on buildings

### Typography System
- **Layout ratio:** Top two-thirds for text, bottom one-third for city silhouette
- **Main headline:** Extra-bold sans-serif (60–100 pt), top-left aligned or centered
- **Subheadline:** Medium weight (24–36 pt), positioned below the main headline
- **Info cards:** White/light rectangular boxes with icon + text (date, venue)
- **Card style:** Simple rectangles with a subtle border
- **Partner logos:** Small horizontal row at the top (24–32 px height)
- **Alignment:** Primarily left-aligned with clear vertical rhythm
- **Text fill:** Solid color only — no effects of any kind

### Icon Elements
- **Icon style:** Clean line icons or filled glyphs
- **Usage:** Calendar icon for date, location pin for venue
- **Size:** 24–32 px, paired with text inside info cards
- **Color:** Matches text color or brand color
- **Position:** Left side within info cards

## Styling Treatments

### Color Application
- **Brand color dominance:** 60–80% of color elements use the primary brand color
- **Monochrome depth:** Opacity variation (30–100%) for layering effect
- **Two-tone harmony:** Brand color + complementary or analogous accent
- **Gradient option:** Subtle vertical gradient on silhouettes (light to dark)
- **Background:** Keep clean and minimal — solid color or very subtle texture

### Silhouette Detail
- **Lines:** Clean vector paths with no rough edges
- **Architectural texture:** Horizontal stripe pattern (2–3 px lines, 6–8 px spacing)
- **Window pattern:** Simple geometric grid on building faces
- **Opacity:** Layered transparency to create atmospheric perspective
- **No outlines:** Silhouettes are solid fills only — no strokes

### Spatial Hierarchy
- **Depth through opacity:** Darker/more opaque = closer; lighter = farther
- **Scale variation:** Foreground elements larger, background elements smaller
- **Overlap:** Natural layering of elements to reinforce depth
- **Atmospheric haze:** Optional subtle blur on the farthest background layer

---

## Composition Strategy

### Vertical Zone Layout
- **Top two-thirds (66%):** Text zone — headline, subheadline, info cards
- **Bottom one-third (33%):** City silhouette illustration zone
- **Divider:** Optional subtle horizontal line between zones
- **Balance:** Heavy text weight at top balanced by visual detail below

### Text Zone Structure
- **Top 10%:** Partner logos or brand identity (small, horizontal row)
- **20–50%:** Primary event headline (oversized, bold, dominant)
- **50–60%:** Subheadline or brief description (medium size)
- **60–66%:** Info cards (date and venue in bordered boxes)
- **Left margin:** 8–10% padding for left-aligned content

### Silhouette Zone Structure
- **0–8% from bottom:** Foreground (bridges, trees — 100% opacity)
- **8–18%:** Midground (primary buildings — 70–80% opacity)
- **18–28%:** Background (high-rise towers — 50–60% opacity)
- **28–33%:** Far background (distant skyline — 30–40% opacity)
- **Horizontal span:** Edge-to-edge, full-width coverage

---

## Text Settings
- **Color selection:** Choose a solid color with high contrast against the background
  - White/light gray background: Use pure black (#1A1A1A) or dark brand color
  - Dark brand color background: Use pure white (#FFFFFF)
  - Text inside info cards: Pure black (#1A1A1A)
- **Fill method:** Solid color fill only — single color, no gradients
- **Prohibited effects:**
  - No drop shadow
  - No outer/inner glow
  - No stroke/outline
  - No gradient fill
  - No text effects of any kind
- **Overall principle:** Keep text clean, professional, and highly legible — consistent with the minimal flat aesthetic. Use font weight and size hierarchy to establish visual impact.

- **Layers:** Separate layers for text, silhouette depth levels, and background
- **Export:** PDF for print, PNG for digital distribution
