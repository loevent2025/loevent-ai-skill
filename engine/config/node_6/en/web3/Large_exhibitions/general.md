# Web3 — Large-Scale Conference & Expo Marketing Strategy

## Event Definition

Large-scale conferences and expos are the most influential offline format in the Web3 industry, bringing together global projects, investors, developers, and community members in one place. Events typically feature keynote presentations, exhibition booths, project pitches, side parties, and more — making them the primary arena for brand visibility and industry dialogue.

## Core Objectives

- Brand exposure and industry positioning
- Establish strategic partnerships across the industry
- Attract potential investors and collaborators
- Community growth and user acquisition
- Media coverage and thought leadership

## Core Channel Mix

| Channel | Priority | Role |
|---------|----------|------|
| X (Twitter) | Primary | Main platform — topic momentum, real-time updates |
| Telegram | Primary | Attendee community, real-time engagement |
| Media network | Primary | In-depth coverage, brand credibility |
| Discord | Important | Community retention, long-term engagement |
| KOL network | Important | Expanded reach, added credibility |

## Distribution Strategy

### Hashtag Strategy
- Primary tags: #[EventName]2025 #[EventAbbreviation]
- Secondary tags: #Web3 #Blockchain + relevant vertical tags
- Engagement tags: #At[EventName] #[EventName]CheckIn

### Content Strategy

| Phase | Content Type | Posting Frequency |
|-------|-------------|-------------------|
| Warm-up phase | Event announcements, speaker reveals, agenda drops | 1–2 posts per day |
| Build-up phase | Booth previews, partner announcements, interactive activations | 2–3 posts per day |
| Final push phase | Countdown, last reminders, highlight previews | 3–5 posts per day |
| Event phase | Live coverage, key quotes, booth check-ins | Real-time updates |
| Wrap-up phase | Event highlights, data roundup, next event preview | 1 post per day |

### X (Twitter) Spaces Planning
- Pre-event: Speaker warm-up session, discuss event highlights
- During event: Live on-site connection, real-time sharing
- Post-event: Debrief and recap, key takeaways

## KOL Partnership Strategy

### KOL Profile Types

| Type | Profile Description | Collaboration Format |
|------|---------------------|----------------------|
| Industry thought leaders | Well-known project founders, investors | On-site presence, content endorsement |
| Media-type KOLs | Industry media editors, column writers | In-depth coverage, exclusive interviews |
| Community KOLs | Active community leaders, prominent X voices | Reposts, on-site check-in content |
| Technical KOLs | Well-known developers, developer advocates | Technical content, panel participation |

### Collaboration Formats
- Pre-event: Announcement reposts, X Spaces appearances
- During event: On-site check-ins, live stream connections
- Post-event: Recap content, in-depth summaries

## Distribution Timeline

### T–4 Weeks: Launch Phase
- Publish event announcement and establish hashtags
- Create Telegram and Discord event channels
- Begin media partnership outreach
- Release first wave of key visuals

### T–3 Weeks: Warm-Up Phase
- Roll out speaker announcements in waves
- Publish agenda overview
- Activate KOL partnerships
- Open registration

### T–2 Weeks: Build-Up Phase
- Announce booth presence and partners
- Host pre-event X Spaces
- Publish in-depth media features
- Drive community engagement activity

### T–1 Week: Final Push Phase
- Daily countdown posts
- Last-call registration reminders
- Tease on-site highlights
- Concentrated KOL repost campaign

### Event Days: Live Phase
- Real-time on-site updates
- Share key quotes as they happen
- Encourage attendee check-in posts
- Live stream key sessions

### T+1 Week: Wrap-Up Phase
- Publish event highlights recap
- Release key metrics and data roundup
- Acknowledge and thank partners
- Preview next event
