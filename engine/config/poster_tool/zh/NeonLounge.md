# 霓虹热带风格 / NEON TROPICAL NIGHTLIFE

## 适用场景
夜店/酒吧活动推广、鸡尾酒派对邀请、音乐节宣传、夏日海滩派对、DJ之夜、电音活动、热带度假村营销、时尚品牌夜场活动、单身派对、跨年倒计时活动

## 配色方案
- **主色调：** 高饱和霓虹色系 - 电光紫 (#9B30FF)、霓虹粉 (#FF1493)、赛博蓝 (#00D4FF)、热带青 (#00FFC8)
- **背景色：** 深紫渐变，从深邃紫 (#1A0033) 到霓虹紫 (#6B238E) 再到玫红 (#99004C) 的垂直过渡
- **光效色：** 粉白高光 (#FFE0F0)、青蓝辉光 (#00FFFF)，透明度中等，边缘柔和发光
- **剪影色：** 纯黑 (#000000) 或深紫 (#1A0033) 的棕榈树/热带植物轮廓
- **饱和度：** 高 (70-100%)，追求大胆鲜艳的"霓虹派对"视觉冲击
- **色彩行为：** 颜色对比强烈，明暗分明，霓虹光效与暗色背景形成戏剧性反差

### 常用配色组合
- 迈阿密之夜：深紫 (#1A0033) → 电光紫 (#9B30FF) → 霓虹粉 (#FF1493)
- 赛博海滩：深蓝 (#000033) → 霓虹青 (#00FFC8) → 电光蓝 (#00D4FF)
- 热带日落：深紫红 (#330022) → 玫红 (#FF0066) → 橙粉 (#FF6B9D)

## 视觉元素
- 棕榈树剪影作为标志性热带元素，通常位于画面两侧
- 中央焦点物：马提尼杯/香槟杯/鸡尾酒杯，带有霓虹渐变填充
- 圆形光球/气泡元素，模拟迪斯科球或梦幻泡泡效果
- 垂直光线纹理，模拟雨滴或灯光投射效果
- 星芒/闪光点缀，增加璀璨派对感
- 杯中液体的渐变色彩，从粉到紫到蓝的流动过渡
- 底部热带植物剪影层叠，增加空间深度

## 效果处理
- 背景使用垂直条纹纹理叠加，透明度10-20%
- 主体元素（酒杯）边缘添加柔和霓虹辉光 (10-20px)
- 圆形光球使用玻璃质感渲染，带有高光反射点
- 星芒效果使用4-6角光芒，颜色为白色或淡粉
- 棕榈树剪影保持纯黑或深色，不添加任何光效
- 使用"叠加"或"滤色"混合模式增强霓虹发光感
- 整体色调偏暖紫粉或冷青蓝，保持统一的霓虹氛围
- 可选的轻微光晕模糊 (5-15px) 增加迷幻夜店感

## 构图策略
- 中心对称构图，主体元素（酒杯）居中
- 棕榈树剪影框架画面两侧，形成自然边框
- 圆形光球位于酒杯上方，形成视觉焦点引导
- 文字信息分层排列：主标题居中偏上，副标题底部
- 前景、中景、背景层次分明
- 底部留出信息区域，用于活动详情
- 垂直构图优先，适合社交媒体故事/海报格式
- 所有元素服务于"热闘派对"的整体氛围

## 文字设置
- **颜色选择：**
  - 深色渐变背景上使用亮色文字形成对比
  - 推荐：霓虹粉 (#FF1493)、纯白 (#FFFFFF)、亮紫 (#CC66FF)
  - 次要信息可用：浅粉 (#FFB6C1)、浅紫 (#DDA0DD) 用于底部详情文字
- **填充方式：** 纯色填充，单一颜色，无渐变
- **禁止效果：**
  - 无渐变填充
  - 无阴影/投影
  - 无外发光/内发光
  - 无描边
  - 无模糊效果
  - 无任何文字特效
- **字体风格：** 混合使用——粗体无衬线体用于冲击性关键词（如"PARTY"），优雅手写脚本体用于氛围词汇（如"Cocktail"），字重从Bold到Black，强调派对活力感
- **整体原则：** 文字保持干净、锐利、高可读性，通过字体大小对比和亮色纯色与深色背景形成强烈的视觉层次，主次分明

## 情绪氛围
热情、狂欢、性感、梦幻、活力。整体传递一种"热带夜店"的派对气质，霓虹色彩带来兴奋感，热带元素增添异域风情。适合追求时尚、年轻、狂欢氛围的夜生活品牌。唤起对夏夜派对、微醺时刻和自由放纵的向往。
