# CITY SILHOUETTE 城市剪影风格

## 适用场景
城市活动公告、区域会议、区块链聚会、黑客马拉松城市巡游、本地社区聚会、多城市路演海报

## 标志性色系
- **可根据品牌定制:** 用户提供品牌主题色
- **单色选项:** 白色/浅灰色背景上的黑色/灰色剪影
- **双色选项:** 品牌色 + 强调色（例如，青色 + 青绿色，绿色 + 黄色）
- **剪影图层:** 同一颜色的多种深浅（30%、50%、70%、100% 透明度）以增加深度
- **背景:** 白色 (#FFFFFF)、浅灰色 (#E8E8E8) 或微妙的品牌色调
- **排版:** 黑色 (#1A1A1A) 或深色品牌色用于主文本
- **特点:** 基于品牌主题色定制、城市剪影占下1/3、扁平矢量风格、单色或双色系统

## 视觉元素

### 城市剪影构成
- **位置:** 画布底部1/3（33%垂直空间）
- **风格:** 扁平矢量剪影，简洁几何形状
- **分层:** 3-5个深度图层（前景 → 背景）
- **前景（100% 透明度）:** 桥梁、树木、水边建筑
- **中景（70-80% 透明度）:** 中层建筑、标志性地标
- **背景（40-60% 透明度）:** 远处的摩天大楼、远处的城市景观
- **远背景（20-30% 透明度）:** 微妙的远处建筑

### 建筑元素
- **地标建筑:** 标志性城市建筑（塔楼、文化遗址）
- **现代摩天大楼:** 不同高度和形状，营造动态天际线
- **传统元素:** 寺庙、宝塔，体现文化特色
- **基础设施:** 桥梁、拱门作为前景元素
- **植被:** 前景中简化的树木剪影
- **纹理细节:** 建筑物上的水平线、几何图案

### 排版系统
- **布局比例:** 上部2/3文本区域，下部1/3城市剪影
- **主标题:** 超粗无衬线字体（60-100磅），左上角或居中
- **副标题:** 中等字重（24-36磅），位于主标题下方
- **信息卡片:** 白色/浅色方框，带图标+文本（日期、地点）
- **卡片样式:** 简单的矩形卡片，带有微妙的边框
- **合作伙伴标志:** 顶部水平行，小尺寸（24-32像素高）
- **对齐方式:** 主要左对齐，具有清晰的垂直节奏
- **文字填充:** 纯色填充，无任何特效

### 图标元素
- **图标风格:** 简洁的线条图标或填充字形
- **用法:** 日期用日历图标，地点用定位针图标
- **尺寸:** 24-32像素，与信息卡片中的文本配对
- **颜色:** 与文本颜色或品牌颜色匹配
- **位置:** 信息卡片左侧

## 效果处理

### 色彩应用
- **品牌色主导:** 60-80%的彩色元素使用品牌主题色
- **单色深度:** 使用透明度变化（30-100%）进行分层
- **双色和谐:** 品牌色 + 互补色或近似色作为强调色
- **渐变选项:** 剪影上微妙的垂直渐变（从浅到深）
- **背景:** 保持简洁和最小化，纯色或非常微妙的纹理

### 剪影细节
- **线条:** 干净的矢量路径，无粗糙边缘
- **建筑纹理:** 水平条纹图案（2-3像素线，6-8像素间距）
- **窗户图案:** 建筑物上的简单几何网格
- **透明度:** 图层透明度营造大气透视感
- **无轮廓:** 剪影为实心填充，无描边

### 空间层次
- **通过透明度实现深度:** 颜色更深/更不透明 = 更近，颜色更浅 = 更远
- **比例变化:** 前景元素较大，背景元素较小
- **重叠:** 元素自然重叠以创造深度
- **大气渐变:** 最远背景层可选的轻微模糊

## 构图策略

### 垂直分区布局
- **顶部2/3 (66%):** 文本区域 - 标题、副标题、信息卡片
- **底部1/3 (33%):** 城市剪影插画区域
- **分隔线:** 可选的微妙水平线分隔区域
- **平衡:** 顶部较重的文本重量与下方视觉细节相平衡

### 文字区域结构
- **顶部10%:** 合作伙伴标志或品牌标识（小尺寸，水平排列）
- **20-50%:** 主要活动标题（超大、粗体、主导）
- **50-60%:** 副标题或描述（中等尺寸）
- **60-66%:** 信息卡片（日期、地点在带边框的方框中）
- **左边距:** 左对齐内容有8-10%的内边距

### 剪影区域结构
- **底部0-8%:** 前景（桥梁、树木，100%透明度）
- **8-18%:** 中景（主要建筑，70-80%透明度）
- **18-28%:** 背景（高层建筑，50-60%透明度）
- **28-33%:** 远背景（远处天际线，30-40%透明度）
- **水平跨度:** 边缘到边缘，全宽覆盖

## 文字设置
- **颜色选择:** 选择与背景形成高对比度的纯色
  - 白色/浅灰背景推荐: 纯黑色 (#1A1A1A) 或深色品牌色
  - 深色品牌色背景推荐: 纯白色 (#FFFFFF)
  - 信息卡片内文字: 纯黑色 (#1A1A1A)
- **填充方式:** 纯色填充，单一颜色，无渐变
- **禁止效果:** 
  - 无阴影/投影
  - 无外发光/内发光
  - 无描边
  - 无渐变填充
  - 无任何文字特效
- **整体原则:** 文字保持干净、专业、高可读性，与极简扁平的城市剪影风格保持一致，通过字体粗细和尺寸层级建立视觉冲击力

## 技术规格
- **分辨率:** 社交媒体1080x1080，故事1080x1920，印刷（A4）2480x3508，300 DPI
- **矢量格式:** AI或SVG，用于剪影和文本的可伸缩性
- **色彩模式:** 数字用RGB，印刷用CMYK
- **软件:** Adobe Illustrator用于矢量工作，Figma用于布局设计
- **分层:** 文本、剪影深度、背景的独立图层
- **格式:** 印刷品用PDF，数字分发用PNG

## 示例 Prompt

### 完整版
将图形渲染为扁平图像、表面图形设计、图案设计。画面铺满至边缘，无外部边框，无画布/纸张纹理/模型。以极简扁平矢量风格呈现城市活动公告海报，底部1/3为城市剪影。

背景：白色 (#FFFFFF) 或浅灰色 (#E8E8E8) 纯色，或用户提供的品牌主题色的微妙色调（10-15% 透明度）。干净简约的背景，无纹理。

顶部区域（0-66% 垂直）：文本区域，留有充足留白。最顶部（0-10%）：小型合作伙伴标志水平排列（每个高24-30像素），黑色或品牌色，带“×”分隔符。主标题区域（15-50%）：超粗无衬线字体（Inter Black, Montserrat ExtraBold, 72-96磅），**黑色 (#1A1A1A) 纯色填充，单一颜色，无阴影、无发光、无描边、无渐变、无任何文字特效**。活动名称全大写或首字母大写。副标题（50-58%）：中等字重文本（28-36磅），描述活动。

信息卡片（58-66%）：两张白色矩形卡片，带有微妙的1像素灰色边框。左侧卡片：日历图标（28像素）+ 日期文本。右侧卡片：定位针图标（28像素）+ 地点文本。卡片使用16-20磅文本，黑色。卡片宽200-250像素，高60-80像素，与主标题左对齐。卡片之间间隔16-20像素。

底部区域（66-100% 垂直）：扁平矢量城市剪影，全宽边缘到边缘。基于用户提供的品牌主题色（例如，单色用黑色/灰色，彩色选项用青色/青绿色，双色用绿色/黄色）。

品牌色中带有不同透明度的剪影图层：
- 远背景（28-33% 高度）：远处建筑，30-40% 透明度，简单几何形状
- 背景（18-28% 高度）：高层摩天大楼，50-60% 透明度，高度各异，部分带有水平线细节
- 中景（8-18% 高度）：主要地标建筑，70-80% 透明度，标志性建筑（塔楼、文化建筑），几何窗户图案
- 前景（0-8% 高度）：桥拱或树木，100% 透明度，详细结构线条

如果提及特定地点，城市应代表该地点（例如，深圳：包括独特的现代塔楼；上海：东方明珠塔，传统宝塔与现代建筑混合）。所有剪影均为简洁矢量风格，无轮廓，仅实心填充。建筑物具有水平条纹纹理细节（2像素线，6-8像素间距），颜色稍深。树木简化为圆形三角形形状。

左对齐布局，文本内容有8-10%的左边距。元素之间有24-32像素间距，形成清晰的垂直节奏。极简现代专业美学。除功能性图标和剪影外，无其他装饰元素。

风格：扁平化设计极简主义、瑞士排版、企业活动海报、城市品牌图形、旅行海报美学、建筑剪影艺术。技术：基于Illustrator/Figma的矢量图，1080x1080或1080x1920或A4打印尺寸，300 DPI，RGB或CMYK。

情绪：专业企业、都市现代、城市认同、国际商务、简洁高效、区域联系、社区聚会、平易近人的正式、科技会议美学。
