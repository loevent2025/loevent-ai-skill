# Web3 Demo Day – Education Event Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- Bootcamp graduation showcase, cohort project pitch, course outcome exhibition
- University project showcase day, student work presentation, practicum outcome report
- Certification assessment showcase, skill validation presentation

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Cohort showcase + Mentor review + Awards ceremony + Community connection |
| Duration | Half-day to full day; 3–8 minutes per project presentation |
| Participants | Cohort members, mentors, training institutions, corporate HR |
| Core Objective | Learning outcome showcase, skill certification, employer matchmaking, community belonging |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Concerns |
|----------|---------------|--------------|
| P0 | Presenting Cohort Members | Showcasing their work, gaining recognition, career opportunities |
| P1 | Corporate HR | Talent discovery, skill evaluation, recruitment matchmaking |
| P2 | Mentors / Training Providers | Teaching outcomes, cohort feedback, brand visibility |
| P3 | Prospective Learners | Learning effectiveness, program value, motivation to enroll |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Bootcamp Graduation Ceremony | Energetic & Engaging (B) | Celebratory atmosphere, growth recognition |
| University Project Showcase | Formal & Professional (A) | Academic value, expert review |
| Employment-Oriented Showcase | Benefits-Oriented (D) | Career opportunities, employer matchmaking |
| Skill Certification Showcase | Suspense-Driven (C) | Competency validation, outcome verification |

## ⚠️ Prohibited Actions

- ❌ Over-embellishing the quality of cohort projects
- ❌ Promising guaranteed employment opportunities
- ❌ Creating excessive presentation pressure on cohort members
- ❌ Over-commercializing an education event
