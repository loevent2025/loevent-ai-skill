# 像素复古浪潮风格 / PIXEL RETROWAVE AESTHETICS

## 适用场景
复古主题派对、合成波/蒸汽波音乐活动、独立游戏宣传、怀旧主题展览、80年代复古之夜、热带度假推广、航空/旅行品牌、复古电子乐专辑封面、街机游戏厅活动、复古科技产品发布、像素艺术展览、Vaporwave文化活动

## 配色方案
- **主色调：** 复古霓虹色系 - 电光紫 (#7B68EE)、霓虹粉 (#FF6B9D)、日落橙 (#FF9F43)、热带青 (#00CED1)
- **背景色：** 紫蓝渐变天空，从顶部钴蓝 (#4169E1) 过渡到中部紫罗兰 (#8A2BE2) 再到地平线处的霓虹粉橙 (#FF7F9F)
- **剪影色：** 深紫蓝 (#2E1A47)、午夜蓝 (#191970) 用于棕榈树和建筑轮廓
- **点缀色：** 金黄日光 (#FFD700)、珊瑚粉 (#FF6F61)、薄荷青 (#98FF98) 用于花卉和细节
- **饱和度：** 高饱和 (70-100%)，追求鲜明浓郁的"霓虹复古"视觉冲击
- **色彩行为：** 天空呈现戏剧性的紫粉橙渐变，海洋反射天空色彩，整体色调温暖而梦幻

### 常用配色组合
- 热带黄昏：钴蓝 (#4169E1) → 紫罗兰 (#8A2BE2) → 霓虹粉 (#FF6B9D) → 日落橙 (#FF9F43)
- 迈阿密之夜：深紫 (#4B0082) → 电光紫 (#9370DB) → 热粉 (#FF69B4) → 珊瑚 (#FF7F50)
- 赛博海岸：午夜蓝 (#191970) → 青紫 (#6A5ACD) → 粉紫 (#DDA0DD) → 蜜桃 (#FFDAB9)

## 视觉元素

### 核心：像素化渲染
- **全画面像素化处理**——所有元素以可见的像素网格呈现，像素大小统一（建议2-4px单位）
- **有限调色板**——模拟8-bit/16-bit游戏机的色彩限制，每个色相使用3-5个明度层级
- **抖动（Dithering）效果**——颜色过渡处使用棋盘格/斜线像素抖动，模拟复古显示器的渐变

### 场景元素
- **棕榈树剪影**——标志性热带元素，深色轮廓，位于前景两侧框架画面
- **日落/双太阳**——地平线处的发光太阳（可以是双太阳/科幻风格），带有水平扫描线或光晕
- **海洋/水面**——紫蓝渐变，带有像素化的波浪纹理和光线反射
- **远景城市天际线**——剪影式的未来都市轮廓，位于地平线
- **热带植物/花卉**——前景底部的装饰性植被，色彩丰富
- **半调网点效果**——天空或过渡区域使用规则的圆点阵列，增加复古印刷感

### 装饰细节
- 像素化星芒/十字光芒点缀夜空
- 水平扫描线叠加，模拟CRT显示器效果
- 简单的几何云朵，边缘锐利的像素轮廓

## 效果处理
- 整体画面保持锐利的像素边缘，禁止抗锯齿平滑
- 天空渐变使用像素抖动（dithering）而非平滑渐变过渡
- 太阳周围添加像素化光晕，使用水平线条暗示光芒
- 海面反射使用水平像素条纹，颜色呼应天空
- 剪影元素保持纯色填充，边缘锐利
- 可叠加轻微的CRT扫描线效果（水平线，透明度5-15%）
- 整体色调偏暖（日落场景）或偏冷（夜晚场景），保持统一
- 画面边缘可添加深色边框，模拟复古显示器/海报装裱效果

## 构图策略
- **经典三段式构图**——天空（上）、海洋/地平线（中）、前景植被（下）
- 棕榈树从画面底部或两侧伸入，形成自然框架
- 视觉焦点位于地平线的太阳/日落处
- 远景城市剪影增加空间深度
- 前景花卉/植物增加层次感和色彩丰富度
- 文字区域位于天空（顶部）或底部信息栏
- 可使用深色底边作为文字信息区域
- 整体对称或近似对称构图，营造平衡稳定的复古海报感

## 文字设置
- **颜色选择：**
  - 天空区域使用霓虹粉 (#FF6B9D) 或浅粉 (#FFB6C1) 形成对比
  - 深色底边区域使用浅色文字：白色 (#FFFFFF)、浅青 (#E0FFFF)、浅粉 (#FFC0CB)
  - 强调文字可用渐变色系（粉→青），但需像素化处理
- **填充方式：** 纯色填充，单一颜色，无渐变
- **禁止效果：**
  - 无渐变填充（除非像素化处理的标题）
  - 无阴影/投影
  - 无外发光/内发光
  - 无描边（除非像素风格描边）
  - 无模糊效果
  - 无任何平滑文字特效
- **字体风格：** 
  - 主标题：像素化手写/草书体，带有霓虹灯管质感，可有水平扫描线穿过
  - 副标题/信息：像素风格粗体无衬线，棱角分明，8-bit游戏字体风格
- **整体原则：** 文字保持像素锐利感，与整体像素艺术风格统一，避免任何平滑抗锯齿效果
