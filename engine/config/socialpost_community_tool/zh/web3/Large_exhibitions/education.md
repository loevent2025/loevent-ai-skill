# Web3 大型展会/博览会 - 教育类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- Web3 教育大会、开发者培训周
- 行业认证、人才培养、技能培训
- 高校合作、职业发展、就业对接

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 主论坛 + 分方向课程 + 实战 Workshop + 项目实操 |
| 内容深度 | 从入门到进阶，系统性知识体系搭建 |
| 参与方 | 教育机构、项目方导师、高校、求职者 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | Web3 学习者 | 学习路径、知识体系、入门门槛 |
| P1 | 转型开发者 | 技术差异、转型路径、实战项目 |
| P2 | 高校师生 | 学术研究、实习机会、课程资源 |
| P3 | 企业培训负责人 | 团队培训、人才招聘、技能认证 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 综合教育大会 | 热情活力型 (B) | 学习氛围、成长路径 |
| 技术培训周 | 悬念引导型 (C) | 学习痛点、解决方案 |
| 职业发展/招聘会 | 福利导向型 (D) | 就业机会、职业通道 |
| 高校合作/学术活动 | 正式专业型 (A) | 学术价值、产学对接 |

## 📝 内容必含元素

### 1. 学习价值（至少2处）
- "系统掌握 Web3 [领域] 核心知识体系"
- "从零到一的完整学习路径规划"

### 2. 导师资源（至少1处）
- "资深导师一对一职业规划指导"
- "[数量]+ 位行业资深从业者亲授"

### 3. 成长支持（至少2处）
- "头部项目实习/就业直通机会"
- "加入 [数量]+ 人的学习者社区"

### 4. 说明提示（建议包含）
- "本活动为教育培训性质，旨在知识传播与人才培养"
