# AI Demo Day – Community Events Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- AI Community Project Showcase, Builder Show & Tell
- AI Open-Source Project Demo Day, Community Maker Showcase
- AI Creator Exhibition, Community Voting Showcase Day

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Project showcase + Community voting + Interactive demos + Celebration party |
| Duration | 3–4 hours |
| Participants | Community Builders, AI enthusiasts, project presenters |
| Core Objective | Community recognition, project visibility, Builder connections, culture building |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Presenting Builders | Community recognition, project visibility, gathering feedback |
| P1 | Community members | Trying out projects, voting, getting inspired |
| P2 | AI enthusiasts | Novel experiences, community integration, learning opportunities |
| P3 | Community newcomers | Getting to know the community, meeting Builders, finding belonging |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Builder Show & Tell | Energetic & Engaging (B) | Casual sharing, community atmosphere |
| Community Voting Showcase | Energetic & Engaging (B) | Interactive participation, democratic selection |
| Open-Source Project Showcase | Formal & Professional (A) | Technical contributions, ecosystem value |
| AI Creator Exhibition | Value-Driven (D) | Creative showcase, prizes and incentives |
