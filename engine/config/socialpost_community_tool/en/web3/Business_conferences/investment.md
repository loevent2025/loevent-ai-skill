# Web3 B2B Conference - Investment Event Knowledge Base

## 📌 Module Activation Conditions

Apply this module when the event theme involves the following keywords:
- Investor closed-door sessions, LP meetings, fund annual meetings
- Project due diligence meetings, post-investment management meetings, exit strategy workshops
- Asset allocation workshops, compliance strategy sessions, institutional networking events

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Invitation-only sessions + panel discussions + 1-on-1 meetings + dinner networking |
| Content Depth | Investment decision-making, due diligence analysis, allocation strategy, exit planning |
| Participants | GPs/LPs, fund managers, project founders, financial advisors (FAs) |
| Core Objective | Advance investment decisions, facilitate resource matching, enable information exchange |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Fund Partners / GPs | Deal flow, investment opportunities, exit channels |
| P1 | LPs / Capital Allocators | Fund performance, allocation strategy, risk management |
| P2 | Project Founders | Fundraising opportunities, investor relations, strategic resources |
| P3 | FAs / Investment Banks | Deal sourcing, transaction facilitation, market intelligence |

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Emphasis |
|------------|-------------------|---------------|
| GP/LP Annual Meeting | Formal & Professional (A) | Performance reporting, strategic communication |
| Investor Closed-Door Workshop | Formal & Professional (A) | Market outlook, strategy discussion |
| Project Matching Session | Benefits-Driven (D) | Exclusive deal flow, priority access |
| Market Trend Workshop | Suspense-Driven (C) | Cycle analysis, allocation direction |

## 📝 Required Content Elements

### 1. Professional Positioning (at least 1 instance)
- "An invitation-only session exclusively for institutional investors"
- "GP/fund manager-level in-depth dialogue"

### 2. Topic Value (at least 2 instances)
- "Exploring [year/cycle] allocation strategy and risk management"
- "Sharing frontline investment insights and decision-making frameworks"

### 3. Networking Value (at least 1 instance)
- "Priority access to curated deal flow"
- "High-quality investor network expansion"

### 4. Compliance Disclaimer (mandatory)
- "⚠️ All content is for informational purposes only and does not constitute investment advice (NFA)"

## ⚠️ Prohibited Actions

- ❌ Publicly disclosing the names of attending institutions
- ❌ Sharing investment views discussed in closed-door sessions
- ❌ Promising investment returns or performance projections
- ❌ Positioning the session as a project pitch event
