# Web3 B2B Conference — Community Event Knowledge Base

## 📌 Module Activation Criteria

Apply the rules in this module when the event theme involves the following keywords:
- Community leads executive session, KOL networking meeting, community operations seminar
- DAO governance meeting, community partnership discussions, ecosystem BD meeting
- Brand partnership meeting, community resource matchmaking, growth strategy seminar

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Executive session + panel discussion + resource matchmaking + networking dinner |
| Content Depth | Community growth strategy, operations methodology, partnership models, resource exchange |
| Participants | Community leads, KOLs, project BD managers, marketing directors |
| Core Objective | Establish community partnerships, facilitate resource matchmaking, exchange growth strategies |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Focus Areas |
|----------|---------------|-------------|
| P0 | Community / Operations Leads | Growth strategy, user engagement, event planning |
| P1 | Project BD / Marketing | Community partnerships, KOL networking, brand exposure |
| P2 | KOLs / Content Creators | Business collaboration, content direction, monetization models |
| P3 | DAO Governors | Governance mechanisms, community incentives, proposal strategy |

---

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Community Leads Executive Session | Formal & Professional (A) | Strategy exchange, experience sharing |
| KOL Networking Meeting | Benefits-Oriented (D) | Partnership opportunities, resource matchmaking |
| DAO Governance Seminar | Curiosity-Driven (C) | Governance challenges, solutions |
| Community Operations Meetup | Energetic & Engaging (B) | Experience sharing, peer connections |

## 📝 Required Content Elements

### 1. Professional Positioning (at least 1 instance)
- "An invitation-only seminar exclusively for community leads"
- "In-depth dialogue among top project community operators"

### 2. Topic Value (at least 2 instances)
- "Sharing hands-on community growth methodology"
- "Exploring solutions to [community operations challenge]"

### 3. Networking Value (at least 1 instance)
- "Facilitating community partnerships and resource exchange"
- "A high-quality network of community operators"

- "Co-building the Web3 community ecosystem"
