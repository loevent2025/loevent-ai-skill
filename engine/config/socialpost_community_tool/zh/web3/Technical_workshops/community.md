# Web3 技术工作坊 - 社区类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- 社区技术分享、Builder 工作坊、开源贡献培训
- DAO 工具使用培训、社区开发者技能交流
- 生态工具教学、社区 Coding Session

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 技能分享 + 共同实操 + 互助学习 + 社区交流 |
| 时长 | 2-4 小时，氛围轻松 |
| 参与方 | 社区 Builder、技术爱好者、生态开发者 |
| 核心目标 | 技能共享、社区连接、互助成长、生态贡献 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | 社区 Builder | 技能提升、贡献机会、同行连接 |
| P1 | 技术爱好者 | 学习交流、实践机会、社区归属 |
| P2 | 生态开发者 | 工具掌握、生态贡献、合作机会 |
| P3 | 新人 Builder | 入门学习、融入社区、找到方向 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 社区技术分享 | 热情活力型 (B) | 轻松氛围、互助学习 |
| 生态工具培训 | 热情活力型 (B) | 实用技能、生态贡献 |
| Builder Coding Session | 悬念引导型 (C) | 技术挑战、共同解决 |
| 开源贡献培训 | 福利导向型 (D) | 贡献机会、社区认可 |

## 📝 内容必含元素

### 1. 社区氛围（至少2处）
- "和社区 Builder 一起学习"
- "互助学习，共同成长"

### 2. 实用技能（至少1处）
- "掌握 [工具/技能] 的实用技巧"
- "学会为生态做贡献"

### 3. 连接价值（至少1处）
- "认识志同道合的 Builder"
- "找到你的 Coding Partner"

### 4. 社区精神（建议包含）
- "一起 BUIDL，一起成长" / "WAGMI"
