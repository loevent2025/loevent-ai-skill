# AI 黑客松 - 技术类活动知识库

## 📌 模块激活条件

当活动主题涉及以下关键词时，应用本模块规则：
- AI Agent 开发黑客松、智能体编程马拉松
- LLM 应用开发赛、RAG/Fine-tuning 技术挑战赛
- Prompt Engineering 大赛、AI Infra 创新赛
- 多模态应用黑客松、AI 工具开发赛

---

## 🎯 场景特征概述

| 特征维度 | 具体描述 |
|----------|----------|
| 形式 | 赛前培训 + 组队开发 + 导师辅导 + Demo 展示 + 评审颁奖 |
| 时长 | 24-72 小时集中开发，或 1-4 周线上赛程 |
| 参与方 | AI 开发者、模型厂商技术团队、技术导师、评审专家 |
| 核心目标 | 技术创新验证、原型开发、人才发现、生态建设 |

---

## 👥 受众画像（按优先级）

| 优先级 | 受众类型 | 关注点 |
|--------|----------|--------|
| P0 | AI 应用开发者 | 技术挑战、奖金/资源、项目孵化 |
| P1 | AI 架构师/算法工程师 | 技术探索、创新验证、最佳实践 |
| P2 | 技术创业者 | MVP 开发、团队组建、融资机会 |
| P3 | 学生/转型开发者 | 学习实践、简历加分、行业入门 |

---

## 🔀 风格选择指引

| 活动类型 | 推荐风格 | 核心侧重 |
|----------|----------|----------|
| 模型厂商官方黑客松 | 正式专业型 (A) | 技术深度、官方背书、生态价值 |
| 社区开发者赛 | 热情活力型 (B) | 参与氛围、学习成长、社区连接 |
| 技术挑战赛 | 悬念引导型 (C) | 技术难题、创新方向、突破机会 |
| 高奖金/孵化赛 | 福利导向型 (D) | 奖金池、API 额度、孵化资源 |

- "丰厚奖金池 + [赛道数量] 大技术赛道"
- "优秀项目获得 [厂商/机构] 官方孵化支持"

- "[数量]+ 位技术导师全程辅导"

- "优胜项目直通 [孵化器/加速器]"

## ❓ 常见问答模板

| 问题 | 回答 |
|------|------|
| 可以一个人参赛吗？ | 可以！个人或组队（≤5人）均可 |

| 已有项目可以参赛吗？ | 需在赛事期间有实质性新开发 |
