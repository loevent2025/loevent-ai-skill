# AI Agent & 大模型行业 - Demo Day/路演日 合作方参考库

> 适用于：AI创业项目路演、投资对接会、加速器Demo Day、AI应用展示日等活动

---

## 场景特征

| 特征项 | 说明 |
|-------|------|
| 规模 | 50-200人（精准邀请制为主） |
| 时长 | 半天至1天 |
| 形式 | 项目Pitch+投资人点评+1对1对接+社交酒会 |
| 核心价值 | 融资对接、项目展示、投资人社交、行业洞察 |
| 预算级别 | 中等预算，重点在场地、嘉宾和精准邀请 |

## 权益设计参考

### 联合主办权益

**品牌权益：**
- 活动品牌并列
- 所有物料优先露出
- 官方联合发布
- 媒体报道中的主办定位

**项目权益：**
- 项目筛选参与权
- 路演项目投资优先权（ROFR或优先额度）
- 项目完整资料获取
- 1对1对接优先安排

**内容权益：**
- 开场/闭幕致辞
- 评审主持/点评
- 投资主题分享（AI赛道洞察）

**社交权益：**
- 晚宴/After Party主导
- VIP投资人定向邀请

### 战略合作权益

**品牌权益：**
- 主视觉Logo露出
- 社媒系列致谢
- 活动物料品牌页

**项目权益：**
- 评审参与
- 项目对接（次于主办方）
- 项目资料获取

**内容权益：**
- AI趋势/技术分享
- 圆桌参与

### 合作伙伴权益

**品牌权益：**
- 物料Logo露出
- 社媒致谢

**服务权益：**
- 服务展示/介绍机会
- 项目服务对接
- 精准客户触达

## Demo Day形式建议

### 路演环节设计
| 环节 | 时长建议 | 说明 |
|-----|---------|------|
| 项目Pitch | 5-8分钟/项目 | 产品演示+商业模式 |
| 投资人点评 | 3-5分钟/项目 | 2-3位评审轮流点评 |
| Q&A | 3-5分钟/项目 | 观众/评审提问 |
| 换场 | 2分钟 | 项目切换准备 |

### 项目分类建议
| 类型 | 适合展示内容 |
|-----|------------|
| AI应用层 | 产品Demo、用户数据、商业模式 |
| AI工具层 | 技术架构、性能指标、开发者生态 |
| AI基础设施 | 技术优势、成本效率、客户案例 |

### 配套环节
- **AI赛道洞察分享**：1-2个主题演讲（15-20分钟/个）
- **技术深度对话**：技术专家与项目方深度交流
- **1对1对接**：预约制，10-15分钟/场
- **社交茶歇/午餐**：非正式交流时间
- **After Party**：晚间深度社交

## 执行要点

### 优先级排序
1. **联合主办方** - 活动策划阶段确定（前2-3个月）
2. **投资人邀请** - 活动前1.5-2个月启动
3. **项目筛选** - 活动前1-1.5个月完成
4. **战略合作方** - 活动前1个月确定
5. **服务机构** - 活动前2-3周落实

### 注意事项
- 项目筛选需与联合主办方深度对齐标准
- 投资人邀请需精准，避免同质化
- 路演顺序安排需考虑项目赛道多样性
- 1对1对接需提前收集双方意向
- 项目资料保密性需提前约定
- 投资意向追踪机制需提前建立
- 媒体报道范围需与项目方确认
- Demo环境需提前测试（网络、设备、API）
- After Party是重要社交场景，需精心安排
