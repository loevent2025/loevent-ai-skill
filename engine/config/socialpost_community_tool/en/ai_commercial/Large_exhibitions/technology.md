Apply this module when the event theme involves the following keywords:

## ⚠️ Prohibited Actions

- ❌ Overstating AI capabilities, e.g., "replaces humans" or "100% accurate"
- ❌ Promising specific performance improvement numbers (unless backed by verified case studies)
- ❌ Using hype language such as "AI can do everything" or "disrupts everything"
- ❌ Fabricating model capabilities or API features
