# AI Expo / Large-Scale Exhibition – Community Events Knowledge Base

## 📌 Module Activation Criteria

Apply this module when the event theme involves the following keywords:
- AI Builder Fest, Developer Community Conference
- AI Hackathon Fest, AI Creator Festival
- AI Open-Source Community Conference, AI Product Maker Day
- AI User Community Conference, AI App Experience Festival

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Main stage + Product showcase area + Interactive demos + Social party |
| Atmosphere | Relaxed, open, innovative, Builder spirit |
| Participants | AI Builders, Product Makers, creators, AI enthusiasts |
| Core Objective | Community bonding, project showcase, sparking inspiration, culture building |

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
