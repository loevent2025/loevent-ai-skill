# Web3 Large-Scale Summit / Expo — General Event Knowledge Base

## 📌 Module Activation Conditions

Apply this module when the event is a comprehensive expo spanning multiple domains:
- Comprehensive Web3 summits, Blockchain expos
- Large-scale events with parallel tracks (Technology + Investment + Community)
- Annual/quarterly industry conferences, Regional Web3 Weeks

---

## 🎯 Scenario Overview

| Dimension | Description |
|-----------|-------------|
| Format | Main stage + Multiple breakout forums + Exhibition area + Side events |
| Content Coverage | Technology, Investment, Ecosystem, Community, Policy — full spectrum |
| Participants | Project teams, Investment firms, Developers, Media, Government |

---

## 👥 Audience Profiles (by Priority)

| Priority | Audience Type | Key Focus |
|----------|---------------|-----------|
| P0 | Industry practitioners | Industry panorama, Trend assessment, Resource access |
| P1 | Project teams / Founders | Brand showcase, Investor access, Partnership opportunities |
| P2 | Investors / Analysts | Track opportunities, Project discovery, Trend analysis |
| P3 | Media / KOLs | News material, Interview access, Exclusive insights |

## 🔀 Style Selection Guide

| Event Type | Recommended Style | Core Focus |
|------------|-------------------|------------|
| Large-scale comprehensive summit | Formal Professional (A) | Industry standing, Authoritative lineup |
| Regional Web3 Week | Energetic & Engaging (B) | Rich experiences, Diverse tracks |
| Annual awards / Gala | Formal Professional (A) | Recognition, Brand value |
| Expo / Exhibition | Energetic & Engaging (B) | Showcase experience, Interactive participation |

## ⚠️ Prohibited Actions

- ❌ Promising investment returns or profit expectations
- ❌ Using language like "get rich quick" or "wealth secrets"
- ❌ Fabricating speaker or organization names
- ❌ Overstating the summit's scale or influence
