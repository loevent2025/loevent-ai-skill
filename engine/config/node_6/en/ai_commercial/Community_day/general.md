# AI Industry — Coding Community Event (Cafe Cursor Style) Marketing Strategy

## Event Overview

A coding community event is an immersive, community-driven gathering designed for AI coding tool users and developers. It follows a "café buyout + collaborative coding" format, typically featuring open coding sessions, technical demo showcases, themed talks, and social networking. The atmosphere is relaxed and welcoming, making it a key format for building AI coding tool user communities and fostering developer collaboration.

**Event Scale:** Approximately 100 attendees  
**Event Duration:** Approximately 10 hours (split into morning and afternoon sessions)

---

## Core Objectives

- Build a local AI coding tool user community
- Facilitate developer networking and collaboration
- Share practical experience and best practices with AI coding tools
- Strengthen brand awareness within developer circles
- Reach a targeted AI developer audience

## Distribution Timeline

### T–2 Weeks: Recruitment Launch

- Publish event preview poster
- Announce event highlights and talk topics
- Open registration channels
- Distribute via WeChat groups and Jike as primary channels
- Sync partner channels simultaneously

### T–1 Week: Warm-up Phase

- Publish speaker / guest spotlights
- Release full event agenda
- Share venue atmosphere sneak peeks
- Push registration reminders
- KOL preview reposts

### T–3 Days: Final Push

- Daily countdown posters
- Last-call registration reminders
- Publish attendee guide (directions, Wi-Fi, key reminders)
- Increase community engagement activity

### Event Day

- Guide attendees through check-in and on-site content sharing
- Post real-time event highlights
- Encourage attendees to photograph and share
- Engage in real-time on Jike and Xiaohongshu
- Collect attendee UGC content

### T+1 Day: Recap Phase

- Publish event recap graphic / video
- Share presentation materials and resources
- Collect attendee feedback
- Preview next edition
- Publish thank-you acknowledgments to partners
