# Web3 Demo Day – Marketing Strategy

## Event Overview

A Demo Day is a pitch and showcase event where early-stage project teams present their products and business models to investors, industry practitioners, and the broader community. Typically held as an accelerator graduation event, hackathon finale, or standalone project showcase, Demo Day is a key opportunity for projects to gain visibility and access resources.

## Core Objectives

- Create exposure opportunities for project teams
- Facilitate investor-project matchmaking
- Showcase ecosystem project milestones and progress
- Establish project evaluation standards
- Cultivate an early-stage project ecosystem

## Distribution Strategy

### Hashtag Strategy
- Primary tags: #[EventName]DemoDay #Web3PitchShowcase
- Project-specific tags: Individual hashtags per project
- Engagement tags: #MostPromisingProject #Web3RisingStars

### Content Strategy

| Phase | Content Type | Posting Frequency |
|-------|-------------|-------------------|
| Warm-Up | Event announcements, project previews, judge reveals | 1 post/day |
| Build-Up | Individual project spotlights, evaluation criteria | 1–2 posts/day |
| Event Day | Live pitch updates, project highlights, judge commentary | Real-time updates |
| Post-Event | Project roundup, investment updates, follow-up coverage | 1 post/day |

### Project Showcase Strategy
- Standardized asset templates for all participating teams
- 30-second elevator pitch video per project
- One-page project brief
- Social media distribution kit

## KOL Partnership Strategy

### KOL Types

| Type | Profile | Engagement Format |
|------|---------|-------------------|
| Investors | Prominent VC partners, angel investors | Judge / Commentary panelist |
| Successful Founders | Founders of well-known projects | Judge / Experience sharing |
| Industry Observers | Journalists, analysts | Commentary / Media coverage |
| Technical Experts | Protocol developers, technical advisors | Technical reviewer |

### Engagement Formats
- **Judge Invitation:** Provide professional evaluation during pitches
- **Commentary Participation:** On-site or remote commentary segments
- **Investor Matchmaking:** Post-event one-on-one project introductions
- **Media Coverage:** In-depth project reporting and features

## Distribution Timeline

### T-3 Weeks: Launch Phase
- Publish Demo Day announcement
- Release list of selected projects
- Tease judge lineup
- Begin targeted investor outreach

### T-2 Weeks: Warm-Up Phase
- Introduce each participating project individually
- Announce full judge lineup
- Open audience registration
- Release project teaser videos

### T-1 Week: Build-Up Phase
- Highlight key differentiators for each project
- Publish evaluation criteria
- Announce live stream details

### Event Day: Live Coverage
- Real-time pitch updates
- Shareable project highlight quotes
